上一节从定义出发求了几个基本函数的导数，但每次都从定义出发太麻烦了。这一节给出系统的求导法则，有了它们，就能像搭积木一样求出绝大多数常见函数的导数。

一、函数的和、差、积、商的求导法则

定理1：设 u = u(x)，v = v(x) 都在点 x 可导，那么：

(1) (u ± v)' = u' ± v'——和的导数等于导数的和。
(2) (u·v)' = u'·v + u·v'——积的导数等于"左导右不导 + 左不导右导"。
(3) (u/v)' = (u'v - uv')/v²（v ≠ 0）——商的导数，分子是"上导下不导减上不导下导"，分母是原分母平方。

这些法则都可以推广到有限个函数。例如：
- (u + v - w)' = u' + v' - w'
- (uvw)' = u'vw + uv'w + uvw'
- (C·u)' = C·u'（常数因子可以提到导数外面）

【例题】

例1：y = 2x³ - 5x² + 3x - 7，求 y'。
解：y' = (2x³)' - (5x²)' + (3x)' - (7)' = 6x² - 10x + 3。

例2：f(x) = x³ + 4cos x - sin(π/2)，求 f'(x) 和 f'(π/2)。
解：f'(x) = 3x² - 4sin x，f'(π/2) = 3·(π/2)² - 4。

例3：y = e^x(sin x + cos x)，求 y'。
解：用积的法则：y' = (e^x)'(sin x + cos x) + e^x(sin x + cos x)' = e^x(sin x + cos x + cos x - sin x) = 2e^x cos x。

例4：y = tan x，求 y'。
解：tan x = sin x/cos x，用商的法则：
y' = [(sin x)'cos x - sin x(cos x)']/cos²x = (cos²x + sin²x)/cos²x = 1/cos²x = sec²x。
即 (tan x)' = sec²x。

例5：y = sec x = 1/cos x，求 y'。
解：y' = [-(cos x)']/cos²x = sin x/cos²x = sec x·tan x。

类似可得：
- (cot x)' = -csc²x
- (csc x)' = -csc x·cot x

二、反函数的求导法则

定理2：如果 x = f(y) 在区间 I_y 内单调、可导且 f'(y) ≠ 0，那么它的反函数 y = f⁻¹(x) 在对应区间 I_x 内也可导，且

[f⁻¹(x)]' = 1/f'(y)，或写成 dy/dx = 1/(dx/dy)

这个公式非常好记：反函数的导数等于直接函数导数的倒数。

【应用举例】

例：x = sin y，y ∈ (-π/2, π/2)，求反函数 y = arcsin x 的导数。
由于 (sin y)' = cos y > 0，cos y = √(1-sin²y) = √(1-x²)，
所以 (arcsin x)' = 1/(sin y)' = 1/cos y = 1/√(1-x²)。

同理可得其他反三角函数的导数：
- (arccos x)' = -1/√(1-x²)
- (arctan x)' = 1/(1+x²)
- (arccot x)' = -1/(1+x²)

三、复合函数的求导法则（链式法则）

这是求导法则中最强大的一条。

定理3：如果 u = g(x) 在点 x 可导，y = f(u) 在点 u = g(x) 可导，那么复合函数 y = f[g(x)] 在点 x 可导，且

dy/dx = (dy/du)·(du/dx)，或写成 y' = f'(u)·g'(x)

这个法则称为"链式法则"，意思就是：复合函数的导数 = 外层导数 × 内层导数，像链条一样一环扣一环。

可以推广到多个中间变量：y = f(u), u = g(v), v = h(x)，则 dy/dx = (dy/du)·(du/dv)·(dv/dx)。

【例题】

例：y = e^(x³)，可看作 y = e^u，u = x³，所以 dy/dx = e^u·3x² = 3x²·e^(x³)。

例：y = sin(2x/(1+x²))，分解为 y = sin u, u = 2x/(1+x²)。则
dy/dx = cos u · [2(1+x²) - 2x·2x]/(1+x²)² = [2(1-x²)/(1+x²)²]·cos(2x/(1+x²))。

熟练后可以不写出中间变量，直接"由外向内逐层求导"：
例：y = ln sin x，y' = (1/sin x)·(sin x)' = cos x/sin x = cot x。

例：y = ln cos(e^x)，y' = [1/cos(e^x)]·(-sin(e^x))·e^x = -e^x tan(e^x)。

四、基本求导法则与导数公式（总结表）

【导数公式】
1. (C)' = 0
2. (x^μ)' = μx^(μ-1)
3. (sin x)' = cos x
4. (cos x)' = -sin x
5. (tan x)' = sec²x
6. (cot x)' = -csc²x
7. (sec x)' = sec x·tan x
8. (csc x)' = -csc x·cot x
9. (a^x)' = a^x·ln a (a>0, a≠1)
10. (e^x)' = e^x
11. (log_a x)' = 1/(x·ln a)
12. (ln x)' = 1/x
13. (arcsin x)' = 1/√(1-x²)
14. (arccos x)' = -1/√(1-x²)
15. (arctan x)' = 1/(1+x²)
16. (arccot x)' = -1/(1+x²)

【求导法则】
1. (u ± v)' = u' ± v'
2. (Cu)' = Cu'（C为常数）
3. (uv)' = u'v + uv'
4. (u/v)' = (u'v - uv')/v²（v ≠ 0）
5. 反函数求导：[f⁻¹(x)]' = 1/f'(y)
6. 复合函数求导（链式法则）：{f[g(x)]}' = f'[g(x)]·g'(x)

此外，双曲函数的导数公式：
- (sh x)' = ch x
- (ch x)' = sh x
- (th x)' = 1/ch²x

反双曲函数：
- (arsh x)' = 1/√(1+x²)
- (arch x)' = 1/√(x²-1)
- (arth x)' = 1/(1-x²)

记住这个表格，加上链式法则的灵活运用，初等函数求导基本没有障碍了。
