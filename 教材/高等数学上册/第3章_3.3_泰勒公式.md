在微分应用中，我们学过用一次多项式（切线）来近似函数：f(x) ≈ f(x₀) + f'(x₀)(x - x₀)。但精度有限。泰勒公式把这个思想推广到高次多项式——用 n 次多项式来逼近函数，并给出误差的精确表达式。

一、问题的提出

设 f(x) 在 x₀ 处具有 n 阶导数。我们希望找到一个 n 次多项式

P_n(x) = a₀ + a₁(x-x₀) + a₂(x-x₀)² + … + a_n(x-x₀)^n

使得 P_n 在 x₀ 处的函数值及各阶导数值与 f(x) 的对应值相等：
P_n(x₀) = f(x₀), P_n'(x₀) = f'(x₀), P_n''(x₀) = f''(x₀), …, P_n⁽ⁿ⁾(x₀) = f⁽ⁿ⁾(x₀)

求解这些系数得到：a_k = f⁽ᵏ⁾(x₀)/k!。于是：

P_n(x) = f(x₀) + f'(x₀)(x-x₀) + [f''(x₀)/2!](x-x₀)² + … + [f⁽ⁿ⁾(x₀)/n!](x-x₀)^n

这就是 f(x) 在 x₀ 处的 n 次泰勒多项式。

二、两种余项形式

用 P_n(x) 近似 f(x) 会有误差 R_n(x) = f(x) - P_n(x)。泰勒公式给出了这个误差的具体形式。

【佩亚诺余项（定性描述）】

泰勒定理1：如果 f(x) 在 x₀ 处具有 n 阶导数，则
f(x) = P_n(x) + o((x-x₀)^n)  （当 x → x₀）

余项 R_n(x) = o((x-x₀)^n) 表示误差是比 (x-x₀)^n 更高阶的无穷小。这告诉我们"误差趋于零的速度"，但不能给出具体数值。

【拉格朗日余项（定量描述）】

泰勒定理2：如果 f(x) 在 x₀ 的某邻域内具有 (n+1) 阶导数，则
f(x) = P_n(x) + [f⁽ⁿ⁺¹⁾(ξ)/(n+1)!]·(x-x₀)^(n+1)

其中 ξ 介于 x₀ 和 x 之间。这给出了误差的精确表达式。

注意：当 n = 0 时，泰勒公式退化为拉格朗日中值定理 f(x) = f(x₀) + f'(ξ)(x-x₀)。所以泰勒公式是拉格朗日中值定理的高阶推广。

三、麦克劳林公式（x₀ = 0 的特例）

当 x₀ = 0 时，泰勒公式变成麦克劳林公式，形式更简洁：

f(x) = f(0) + f'(0)x + [f''(0)/2!]x² + … + [f⁽ⁿ⁾(0)/n!]x^n + R_n(x)

【几个重要函数的麦克劳林展开】

1. e^x = 1 + x + x²/2! + x³/3! + … + x^n/n! + e^(θx)·x^(n+1)/(n+1)!（0<θ<1）

2. sin x = x - x³/3! + x⁵/5! - … + (-1)^(m-1)·x^(2m-1)/(2m-1)! + R_{2m}
   其中 R_{2m} = (-1)^m·cos(θx)·x^(2m+1)/(2m+1)!

3. cos x = 1 - x²/2! + x⁴/4! - … + (-1)^m·x^(2m)/(2m)! + R_{2m+1}

4. ln(1+x) = x - x²/2 + x³/3 - … + (-1)^(n-1)·x^n/n + R_n

5. (1+x)^α = 1 + αx + [α(α-1)/2!]x² + … + [α(α-1)·…·(α-n+1)/n!]x^n + R_n

这些展开式在近似计算和极限求解中非常有用。比如计算 e 的近似值：取 x = 1, n = 10，e ≈ 1 + 1 + 1/2! + … + 1/10!，误差不超过 3/11! < 10⁻⁶。

四、应用

泰勒公式的用途广泛：
- 近似计算：用多项式代替复杂函数求值
- 求极限：将函数展开后比较阶数
- 证明不等式：利用拉格朗日余项的符号

例：求 lim_{x→0} (sin x - x)/x³。
解：sin x = x - x³/6 + o(x³)，所以 (sin x - x)/x³ = -1/6 + o(1) → -1/6。
