前面学过，两个无穷小相加、相减、相乘仍然是无穷小。但两个无穷小相除呢？情况就复杂了。

比如当 x → 0 时，3x²、x²、sin x 都是无穷小，但它们的比值却有不同结果：
- lim (x²)/(3x²) = 1/3
- lim (3x²)/x² = 3
- lim (sin x)/x = 1

这些不同结果，反映的是不同的无穷小趋近于 0 的"快慢"程度。同样是往 0 走，有的走得快，有的走得慢，有的速度差不多。本节就来给这种"快慢"定量地分类。

一、无穷小比较的定义

设 α 和 β 都是在同一变化过程中的无穷小，且 α ≠ 0。比较它们的比值极限：

(1) 如果 lim β/α = 0，就说 β 是比 α 高阶的无穷小，记作 β = o(α)。

通俗理解：β 比 α 更快地"缩"到 0——两者都在变小，但 β 小得更"猛"。比如 x → 0 时，x² 比 x 高阶（x² 除以 x 等于 x，趋于 0）。

(2) 如果 lim β/α = ∞，就说 β 是比 α 低阶的无穷小。

理解为：β 虽然也趋于 0，但相比之下"慢"多了。

(3) 如果 lim β/α = c ≠ 0（常数），就说 β 与 α 是同阶无穷小。

(4) 如果 lim β/α^k = c ≠ 0（k > 0），就说 β 是关于 α 的 k 阶无穷小。

比如，1 - cos x 当 x → 0 时，与 x² 同阶（因为 (1-cos x)/x² → 1/2），所以 1-cos x 是关于 x 的二阶无穷小。

(5) 如果 lim β/α = 1，就说 β 与 α 是等价无穷小，记作 α ~ β。

等价无穷小是同阶无穷小的特殊情况（c = 1）。当 x → 0 时，sin x ~ x，tan x ~ x，arcsin x ~ x，arctan x ~ x 等，都是等价无穷小。

二、常见的等价无穷小（当 x → 0 时）

- sin x ~ x
- tan x ~ x
- arcsin x ~ x
- arctan x ~ x
- 1 - cos x ~ x²/2
- ln(1+x) ~ x
- e^x - 1 ~ x
- (1+x)^a - 1 ~ a·x（a 为常数）
- √(1+x) - 1 ~ x/2

这些"配对"在求极限时非常有用，能大大简化计算。

三、等价无穷小的性质和定理

定理1：α 与 β 是等价无穷小的充分必要条件是 β = α + o(α)。

意思是说：如果 β 和 α 等价，那么 β 等于 α 加上一个比 α 更高阶的无穷小"尾巴"。

例如，当 x → 0 时 sin x ~ x，所以可以写作 sin x = x + o(x)；
同理，1 - cos x = x²/2 + o(x²)。

定理2（等价无穷小替换定理）：如果 α ~ α'，β ~ β'，且 lim β'/α' 存在，则
lim β/α = lim β'/α'

这条定理告诉我们：在求两个无穷小之比的极限时，分子和分母都可以用和它们等价的无穷小来替换！选得好的话，计算会变得极其简单。

四、应用举例

例1：求 lim_{x→0} tan 2x / sin 5x
解：x → 0 时，tan 2x ~ 2x，sin 5x ~ 5x。所以
原式 = lim_{x→0} 2x/5x = 2/5

例2：求 lim_{x→0} sin x / (x³ + 3x)
解：x → 0 时，sin x ~ x，且 x³ + 3x ~ 3x（x³ 是比 x 高阶的无穷小，可以忽略）。所以
原式 = lim_{x→0} x/(3x) = 1/3

例3：求 lim_{x→0} [(1+x²)^(1/3) - 1] / (cos x - 1)
解：x → 0 时，(1+x²)^(1/3) - 1 ~ (1/3)·x²，cos x - 1 ~ -x²/2。所以
原式 = lim_{x→0} (x²/3) / (-x²/2) = -2/3

总结一下：等价无穷小替换是做极限题的"利器"——遇到 0/0 型时，把那些熟悉的无穷小换成更简单的等价形式，往往几步就算出来了。特别要熟记 sin x ~ x, tan x ~ x, 1-cos x ~ x²/2 这几个最常用的。
