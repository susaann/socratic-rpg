导数的应用必须建立在坚实的理论基础之上。本章介绍的三个"中值定理"就是这个基础——它们揭示了函数在区间上的整体行为与某点导数值之间的深刻联系。

一、罗尔定理

【费马引理】

在讲罗尔定理之前，先介绍一个更基础的引理。费马引理说的是：如果函数 f(x) 在点 x₀ 处取得局部最大值或最小值（即在该点附近，所有函数值都不大于或不小于 f(x₀)），且 f(x) 在 x₀ 处可导，那么 f'(x₀) = 0。

通俗理解：在"山顶"或"谷底"，切线一定是水平的（导数为零）。这几乎是直觉就能接受的结论。

【罗尔定理】

如果函数 f(x) 满足：
(1) 在闭区间 [a, b] 上连续；
(2) 在开区间 (a, b) 内可导；
(3) f(a) = f(b)（两端点函数值相等），
那么在 (a, b) 内至少存在一点 ξ，使得 f'(ξ) = 0。

几何理解：一条连续光滑的曲线，如果两端在同一高度上，那么中间至少有一个位置的切线是水平的。就像你从一个山谷出发，走了一圈回到同样高度——中间必然经过了至少一个"山顶"或"谷底"。

证明思路：连续函数在闭区间上必有最大值 M 和最小值 m。如果 M = m，则函数是常数，导数处处为零。如果 M > m，由于 f(a) = f(b)，M 和 m 至少有一个不在端点取到，设 M 在 ξ 处取到（内部极值），由费马引理知 f'(ξ) = 0。

二、拉格朗日中值定理

罗尔定理要求 f(a) = f(b)，这个条件限制太强。拉格朗日中值定理把它推广了：

如果 f(x) 满足：
(1) 在闭区间 [a, b] 上连续；
(2) 在开区间 (a, b) 内可导，
那么在 (a, b) 内至少存在一点 ξ，使得

f(b) - f(a) = f'(ξ)(b - a)

也就是 f'(ξ) = [f(b) - f(a)]/(b - a)。

几何意义：连续光滑的曲线弧上，至少存在一点处的切线平行于连接两端点的弦。

证明思路：构造辅助函数 φ(x) = f(x) - f(a) - [(f(b)-f(a))/(b-a)](x-a)。这个 φ(x) 满足罗尔定理的三个条件（φ(a) = φ(b) = 0），对 φ 应用罗尔定理即可得到结论。

这个公式也可以写成"有限增量公式"：
f(x + Δx) - f(x) = f'(x + θΔx)·Δx（0 < θ < 1）

注意：这个等式是精确的（不是近似！），它把函数增量与区间内某点的导数精确联系起来。而之前学的微分 dy = f'(x)·Δx 只是近似。

【重要推论】

如果函数 f(x) 在区间 I 上的导数恒为零，那么 f(x) 在 I 上是一个常数。

证明：在 I 上任取两点 x₁ < x₂，由拉格朗日中值定理，f(x₂) - f(x₁) = f'(ξ)(x₂ - x₁) = 0，所以 f(x₁) = f(x₂)。

【应用示例】

证明：当 x > 0 时，x/(1+x) < ln(1+x) < x。

证：令 f(t) = ln(1+t)，在 [0, x] 上应用拉格朗日定理：
ln(1+x) - ln 1 = [1/(1+ξ)]·x（0 < ξ < x）
因为 1/(1+x) < 1/(1+ξ) < 1，所以 x/(1+x) < ln(1+x) < x。

三、柯西中值定理

拉格朗日中值定理还可以进一步推广。柯西中值定理处理的是两个函数的关系：

如果 f(x) 和 F(x) 满足：
(1) 在闭区间 [a, b] 上连续；
(2) 在开区间 (a, b) 内可导；
(3) F'(x) ≠ 0 在 (a, b) 内，
那么在 (a, b) 内至少存在一点 ξ，使得

[f(b) - f(a)]/[F(b) - F(a)] = f'(ξ)/F'(ξ)

柯西中值定理把拉格朗日定理作为特例（取 F(x) = x 即得）。这个定理在证明洛必达法则时会起到关键作用。
