7.7 常系数齐次线性微分方程

上一节讨论了线性微分方程解的结构，但并没有给出求特解的具体方法。对于常系数齐次线性微分方程（$P(x)$、$Q(x)$ 为常数），有一种非常漂亮的代数解法——特征方程法。

【二阶常系数齐次线性方程：$y'' + py' + qy = 0$】

注意到指数函数 $y = e^{rx}$ 的导数仍然是自身的倍数，我们尝试把 $y = e^{rx}$ 代入方程：
$$(r^2 + pr + q)e^{rx} = 0$$
由于 $e^{rx} \neq 0$，所以
$$r^2 + pr + q = 0$$
这个代数方程称为该微分方程的特征方程。$r$ 的值完全决定了微分方程解的形式。

根据特征方程的判别式 $\Delta = p^2 - 4q$，分三种情况：

【情况一：$\Delta > 0$——两个不相等的实根 $r_1 \neq r_2$】
通解为 $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$。

【情况二：$\Delta = 0$——两个相等的实根 $r_1 = r_2 = r$】
通解为 $y = (C_1 + C_2 x)e^{rx}$。
注意：不能只写 $Ce^{rx}$（那样只有一个任意常数，不是通解），需要在前面乘以一个一次多项式 $C_1 + C_2x$。

【情况三：$\Delta < 0$——一对共轭复根 $r = \alpha \pm i\beta$】
通解为 $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$。

例1：解 $y'' - 2y' - 3y = 0$。
解：特征方程 $r^2 - 2r - 3 = 0$，根 $r_1 = -1$，$r_2 = 3$（两不等实根）。通解：$y = C_1 e^{-x} + C_2 e^{3x}$。

例2：解 $\frac{d^2s}{dt^2} + 2\frac{ds}{dt} + s = 0$，$s|_{t=0}=4$，$s'|_{t=0}=-2$。
解：特征方程 $r^2+2r+1=0$，根 $r_1=r_2=-1$（二等根）。通解 $s = (C_1 + C_2t)e^{-t}$。用初值条件定出 $C_1=4$，$C_2=2$。特解：$s = (4+2t)e^{-t}$。

例3：解 $y'' - 2y' + 5y = 0$。
解：特征方程 $r^2-2r+5=0$，根 $r_{1,2} = 1 \pm 2i$（共轭复根）。通解：$y = e^x(C_1\cos 2x + C_2\sin 2x)$。

【应用：无阻尼自由振动】

弹簧振子无阻尼时的方程 $\frac{d^2x}{dt^2} + k^2 x = 0$。特征方程 $r^2 + k^2 = 0$，根 $r = \pm ki$。通解：
$$x = C_1\cos kt + C_2\sin kt = A\sin(kt + \varphi)$$
其中 $A = \sqrt{C_1^2+C_2^2}$，$\tan\varphi = \frac{C_1}{C_2}$。这就是简谐振动！振幅为 $A$，周期 $T = \frac{2\pi}{k}$，$k$ 叫做固有频率——由振动系统本身决定，与初始条件无关。

【应用：有阻尼自由振动的三种情形】

方程 $\frac{d^2x}{dt^2} + 2n\frac{dx}{dt} + k^2 x = 0$ 的特征方程为 $r^2 + 2nr + k^2 = 0$：

(1) 小阻尼（$n < k$）：$r = -n \pm i\omega$（$\omega = \sqrt{k^2-n^2}$），解为 $x = Ae^{-nt}\sin(\omega t + \varphi)$——振幅 $Ae^{-nt}$ 随时间衰减的振动，物体最终趋于平衡位置。

(2) 大阻尼（$n > k$）：两个不等负实根，解为两个指数衰减的线性组合 $x = C_1 e^{r_1 t} + C_2 e^{r_2 t}$。物体不发生振动，最多越过平衡位置一次，然后逐渐趋于平衡。

(3) 临界阻尼（$n = k$）：二等根 $r = -n$，解为 $x = e^{-nt}(C_1 + C_2 t)$。物体也不振动，但趋于平衡的速度在这种阻尼下是最快的（工程中常希望设计在临界阻尼附近）。

特征方程法也可以推广到 $n$ 阶常系数齐次线性方程，只要写出 $n$ 次特征方程，根据根的情况对应写出通解即可。
