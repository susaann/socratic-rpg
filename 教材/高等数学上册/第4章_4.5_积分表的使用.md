4.5 积分表的使用

通过前面的讨论可以看出，积分的计算比导数的计算灵活复杂得多。导数有一套严格的"计算法则"，函数复合求导也好、乘积求导也好，都可以按部就班地算。但积分就没有这么简单——没有万能的方法，很多时候需要观察、试探和技巧。

为了实用的方便，人们把常用的积分公式汇集成表，这种表叫做积分表。积分表是按照被积函数的类型来排列的。求积分时，可根据被积函数的类型直接在表中查得所需的结果，或者经过简单变形后再查表。

本书附录中有一个简单的积分表，以供查阅。

【直接查表】

例1：求 $\int \frac{x}{3x+4}\,dx$。

被积函数含有 $ax+b$，在积分表（一）中查得公式7：
$$\int \frac{x}{ax+b}\,dx = \frac{1}{a^2}(ax + b - b\ln|ax+b|) + C$$
这里 $a=3$，$b=4$，于是
$$\int \frac{x}{3x+4}\,dx = \frac{1}{9}(3x+4 - 4\ln|3x+4|) + C$$

例2：$\int \frac{dx}{5-4\cos x}$。

被积函数含有三角函数，在积分表（十一）中查关于 $\int \frac{dx}{a+b\cos x}$ 的公式。公式有两个，取决于 $a^2 > b^2$ 还是 $a^2 < b^2$ 决定选用哪一个。
现在 $a=5$，$b=-4$，$a^2=25 > b^2=16$，所以用公式105：
$$\int \frac{dx}{a+b\cos x} = \frac{2}{\sqrt{a^2-b^2}}\arctan\left(\sqrt{\frac{a-b}{a+b}}\tan\frac{x}{2}\right) + C \quad (a^2 > b^2)$$
代入得
$$\int \frac{dx}{5-4\cos x} = \frac{2}{3}\arctan\left(3\tan\frac{x}{2}\right) + C$$

【先代换再查表】

例3：求 $\int \frac{dx}{x\sqrt{4x^2+9}}$。

这个积分不能在表中直接查到，需要先做变量代换。令 $2x = u$，则 $\sqrt{4x^2+9} = \sqrt{u^2+3^2}$，且 $dx = \frac{1}{2}du$。于是
$$\int \frac{dx}{x\sqrt{4x^2+9}} = \int \frac{\frac{1}{2}du}{\frac{u}{2}\sqrt{u^2+9}} = \int \frac{du}{u\sqrt{u^2+9}}$$

被积函数中含有 $\sqrt{u^2+3^2}$，在积分表（六）中查到公式37：
$$\int \frac{dx}{x\sqrt{x^2+a^2}} = \frac{1}{a}\ln\left|\frac{\sqrt{x^2+a^2}-a}{x}\right| + C$$
令 $a=3$，$x$ 换成 $u$，得
$$\int \frac{du}{u\sqrt{u^2+9}} = \frac{1}{3}\ln\left|\frac{\sqrt{u^2+9}-3}{u}\right| + C$$

再把 $u = 2x$ 代回去，最终得
$$\int \frac{dx}{x\sqrt{4x^2+9}} = \frac{1}{3}\ln\left|\frac{\sqrt{4x^2+9}-3}{2x}\right| + C$$

【利用递推公式】

例4：求 $\int \sin^4 x\,dx$。

在积分表（十一）中查到公式95：
$$\int \sin^n x\,dx = -\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x\,dx$$
这是一个递推公式：每用一次，正弦的幂次减少2。重复使用，直到减到1次或0次为止。

现在 $n=4$：
$$\int \sin^4 x\,dx = -\frac{1}{4}\sin^3 x\cos x + \frac{3}{4}\int \sin^2 x\,dx$$

对 $\int \sin^2 x\,dx$ 用公式93：$\int \sin^2 x\,dx = \frac{x}{2} - \frac{1}{4}\sin 2x + C$。于是
$$\int \sin^4 x\,dx = -\frac{1}{4}\sin^3 x\cos x + \frac{3}{4}\left(\frac{x}{2} - \frac{\sin 2x}{4}\right) + C$$

递推公式的好处是：可以把一个高次积分通过有限步递推转化为低次积分（甚至零次积分），从而避免每次都重新推导。

【使用积分表的注意事项】

一般说来，查积分表可以节省计算积分的时间。但是，只有掌握了前面学过的基本积分方法（换元法、分部积分法），才能灵活地使用积分表。而且对一些比较简单的积分，应用基本积分方法来计算可能比查表更快。例如，对 $\int \sin x\cos^4 x\,dx$，用变换 $u = \sin x$ 很快就可得到结果。所以，求积分时究竟是直接计算，还是查表，或是两者结合使用，应该具体分析，不能一概而论。

【本章的一个重要说明】

在本章结束之前，必须指出一件重要的事情：对初等函数来说，在其定义区间上，它的原函数一定存在（因为初等函数在其定义区间内连续，而连续函数一定有原函数），但原函数不一定都是初等函数！

例如：
$$\int e^{-x^2}\,dx, \quad \int \frac{\sin x}{x}\,dx, \quad \int \frac{1}{\ln x}\,dx$$
这几个看起来"普普通通"的积分，它们的原函数就都不是初等函数——也就是说，你不可能用有限个初等函数（多项式、指数、对数、三角、反三角及其复合与四则运算）把它们的原函数表示出来。

这不是你"积分技术不够好"的问题，而是数学上已经证明：这些函数的原函数确实不存在于初等函数的范围内。因此，遇到这类积分时，不必徒劳地去寻找"初等表达式"，可以用数值积分或其他方式去处理。这也是为什么我们需要积分表、数值方法和特殊函数（如误差函数 $\text{erf}(x) = \frac{2}{\sqrt{\pi}}\int_0^x e^{-t^2}\,dt$ 就是专门用来表达 $\int e^{-x^2}\,dx$ 的）。
