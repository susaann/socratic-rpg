最后一节来学习闭区间上连续函数的几个重要性质。这些性质在理论上很重要，而且在解方程、判断函数值范围等实际问题中非常有用。

注意：以下定理都要求函数在"闭区间"上连续。如果是开区间或者函数有间断点，结论不一定成立。

一、有界性与最大值最小值定理

先明确概念：函数 f(x) 在区间 I 上的最大值，就是函数在 I 上能取到的最大函数值；最小值就是能取到的最小函数值。

例如：f(x) = 1 + sin x 在 [0, 2π] 上有最大值 2 和最小值 0。函数 y = x 在开区间 (a, b) 内既无最大值也无最小值（因为无论取多靠近边界的值，总能再靠近一点点）。

定理1（有界性与最值定理）：在闭区间 [a, b] 上连续的函数，一定在该区间上有界，且一定能取到它的最大值和最小值。

这个定理告诉我们：闭上区间的"墙壁"让函数值有个上限和下限，而且这两个极值一定能在区间内的某个点达到。如果不闭（开区间）或者不连续，那就不保证了。

反例说明：
- y = tan x 在开区间 (-π/2, π/2) 上连续但有界吗？没有，它趋于 ∞，也无最大最小值。
- 分段函数 f(x) = { -x+1, 0≤x<1; 1, x=1; -x+3, 1<x≤2 } 在 [0, 2] 上有间断点 x=1，虽然有界，但没有最大值也没有最小值。

二、零点定理与介值定理

【零点】如果 x₀ 使得 f(x₀) = 0，就称 x₀ 为函数 f(x) 的零点——也就是方程 f(x) = 0 的根。

定理2（零点定理）：设 f(x) 在闭区间 [a, b] 上连续，且 f(a) · f(b) < 0（即两端点函数值异号），那么在开区间 (a, b) 内至少存在一点 ξ，使得 f(ξ) = 0。

几何理解：一条连续曲线的两端在 x 轴的两侧（一上一下），那么这条曲线必定会和 x 轴相交至少一次。想想看，你不可能从 x 轴上方"连续地"走到下方而不穿过 x 轴——除非你跳过去，但"连续"意味着你不能跳。

定理3（介值定理）：设 f(x) 在 [a, b] 上连续，且 f(a) = A, f(b) = B，则对于 A 与 B 之间的任何一个数 C，在 (a, b) 内至少存在一点 ξ，使得 f(ξ) = C。

证明思路：构造 φ(x) = f(x) - C，则 φ(a) = A - C 与 φ(b) = B - C 异号，由零点定理即得。

几何意义：连续曲线 y = f(x) 与水平直线 y = C（C 介于 A 和 B 之间）至少相交于一点。函数从 A 走到 B，中间必须把每个值都"经过"一遍。

推论：在闭区间 [a, b] 上连续的函数 f(x) 的值域是一个闭区间 [m, M]，其中 m 和 M 分别是 f(x) 在 [a, b] 上的最小值和最大值。也就是说，函数值会"连成一片"，中间不会漏掉任何值。

【应用举例】

例：证明方程 x³ - 4x² + 1 = 0 在区间 (0, 1) 内至少有一个根。

证：令 f(x) = x³ - 4x² + 1，这是多项式，在 [0, 1] 上连续。计算 f(0) = 1 > 0，f(1) = 1 - 4 + 1 = -2 < 0。由于 f(0)·f(1) < 0，根据零点定理，在 (0, 1) 内至少存在一点 ξ，使得 f(ξ) = 0，即方程至少有一个根在 (0, 1) 内。

三、一致连续性（*选读内容）

一般情况下，函数 f(x) 在区间 I 上连续，定义中的 δ 既依赖于给定的 ε，也依赖于选取的点 x₀。换一个点，同样的 ε 可能需要不同的 δ。

但有些函数更好——存在一个 δ，只依赖于 ε，对区间上的所有点都适用。这种更强的连续性叫做"一致连续"。

定义：设函数 f(x) 在区间 I 上有定义。如果 ∀ε > 0，∃δ > 0，使得对于 I 上任意两点 x₁, x₂，当 |x₁ - x₂| < δ 时，有 |f(x₁) - f(x₂)| < ε，则称 f(x) 在 I 上一致连续。

简而言之：不论你在区间的哪个位置取两个足够近的点，函数值的差都能一致地小于 ε。δ"一通百通"，不依赖位置。

定理4（一致连续性定理）：如果 f(x) 在闭区间 [a, b] 上连续，那么它在该区间上一定一致连续。

这正是"闭区间"的威力——在闭区间上连续的函数，不仅自动有界、能取到最值，而且还自动一致连续。开区间就不行了，比如 f(x) = 1/x 在 (0, 1] 上连续，但不一致连续（靠近 0 时变化太剧烈，没有统一的 δ）。
