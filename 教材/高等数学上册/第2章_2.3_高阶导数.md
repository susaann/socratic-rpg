我们已经会求函数的导数了。但导函数本身还是一个函数——那么它的导数是什么呢？这就引出"高阶导数"的概念。

一、什么是高阶导数

以直线运动为例：位置函数 s = s(t) 的导数 v = s'(t) 是速度；而速度再对时间求导，得到加速度 a = v'(t) = s''(t)。加速度就是位置的二阶导数。

一般地，函数 y = f(x) 的导数 y' = f'(x) 仍然是 x 的函数。把 f'(x) 再求导，得到 f(x) 的二阶导数，记作 y'' 或 d²y/dx²。

类似地：
- 三阶导数：y''' 或 d³y/dx³
- 四阶导数：y⁽⁴⁾ 或 d⁴y/dx⁴
- ……
- n 阶导数：y⁽ⁿ⁾ 或 dⁿy/dxⁿ

二阶及以上的导数统称高阶导数。求高阶导数就是反复求导而已，关键在于找到规律的表达式。

二、常见函数的 n 阶导数

【指数函数 y = e^x】
y' = e^x, y'' = e^x, y''' = e^x, …, y⁽ⁿ⁾ = e^x。
结论：(e^x)⁽ⁿ⁾ = e^x——怎么求都是它自己。

【正弦函数 y = sin x】
y' = cos x = sin(x + π/2)
y'' = -sin x = sin(x + 2·π/2)
y''' = -cos x = sin(x + 3·π/2)
y⁽⁴⁾ = sin x = sin(x + 4·π/2)
一般地，(sin x)⁽ⁿ⁾ = sin(x + n·π/2)。

【余弦函数】
(cos x)⁽ⁿ⁾ = cos(x + n·π/2)。

发现规律了吗？sin 和 cos 每求导 4 次就回到自己，只是在相位上每次"前移"π/2。

【对数函数 y = ln(1+x)】
y' = 1/(1+x) = (1+x)^(-1)
y'' = -(1+x)^(-2)
y''' = 2(1+x)^(-3)
y⁽⁴⁾ = -6(1+x)^(-4)
一般地，y⁽ⁿ⁾ = (-1)^(n-1)·(n-1)!/(1+x)^n。当 n = 1 时约定 0! = 1 也适用。

【幂函数 y = x^μ】
y' = μx^(μ-1)
y'' = μ(μ-1)x^(μ-2)
y''' = μ(μ-1)(μ-2)x^(μ-3)
一般地，y⁽ⁿ⁾ = μ(μ-1)·…·(μ-n+1)x^(μ-n)。

特别地，当 μ = n（正整数）时，(x^n)⁽ⁿ⁾ = n!（常数），而更高阶导数 (x^n)⁽ⁿ⁺¹⁾ = 0。

三、乘积的高阶导数——莱布尼茨公式

如果 u(x) 和 v(x) 都有 n 阶导数，那么它们的和与差很容易：(u ± v)⁽ⁿ⁾ = u⁽ⁿ⁾ ± v⁽ⁿ⁾。

但乘积 uv 的 n 阶导数就没那么简单了。计算前几阶：
(uv)' = u'v + uv'
(uv)'' = u''v + 2u'v' + uv''
(uv)''' = u'''v + 3u''v' + 3u'v'' + uv'''

系数 1, 2, 1; 1, 3, 3, 1——是不是很像二项式定理的系数？没错！

莱布尼茨（Leibniz）公式：
(uv)⁽ⁿ⁾ = Σ_{k=0}^{n} C_n^k · u⁽ⁿ⁻ᵏ⁾ · v⁽ᵏ⁾

其中 C_n^k = n!/[k!(n-k)!] 是组合数。这公式和二项式定理 (u+v)^n = Σ C_n^k u^(n-k) v^k 形式完全一样，只是把"幂"换成了"导数阶数"。

例：求 y = x²e^(2x) 的 20 阶导数（用莱布尼茨公式）。
设 u = e^(2x), v = x²，则 u⁽ᵏ⁾ = 2^k e^(2x)，而 v 只有 v' = 2x, v'' = 2, v⁽ᵏ⁾ = 0 (k ≥ 3)。代入公式只需要算前三项，最终：
y⁽²⁰⁾ = 2²⁰e^(2x)(x² + 20x + 95)。

例：验证 y = √(2x-x²) 满足 y·y'' + 1 = 0。
y' = (1-x)/√(2x-x²)，继续求导并代入即可验证。这类"函数满足某关系式"的题是常见题型。
