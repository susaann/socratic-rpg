================================================================================
                     7.9  欧拉方程
================================================================================

变系数线性微分方程通常很难直接求解。但有一类特殊的变系数方程，可以通过巧妙的变量代换化为常系数方程，从而迎刃而解——这就是**欧拉方程**（Euler equation）。

【标准形式】
    x^n y^{(n)} + p_1 x^{n-1} y^{(n-1)} + ⋯ + p_{n-1} x y' + p_n y = f(x)

其中 p_1, p_2, …, p_n 均为常数。它的特点是：每一项中 x 的幂次恰好等于 y 的求导阶数——两者"匹配"，这是它能被化简的关键。

【解法：变量代换】

作变换 x = e^t （等价于 t = ln x）。注意这里默认 x>0；如果要在 x<0 范围求解，可以类似地令 x=-e^t 或 t=ln(-x)，结果形式相同。

由链式法则逐次求导：
    dy/dx = (dy/dt)·(dt/dx) = (1/x)·(dy/dt)
即  xy' = Dy    （记 D 为对 t 求导的运算符号）

再求二阶：
    d²y/dx² = (1/x²)(d²y/dt² - dy/dt)
即  x²y'' = D(D-1)y

三阶：
    x³y''' = D(D-1)(D-2)y

一般地：
    x^k y^{(k)} = D(D-1)⋯(D-k+1) y

将这些表达式代入欧拉方程，自变量就从 x 变成了 t，方程化为关于 t 的常系数线性微分方程。求出解后，再把 t 换回 ln x，即得原方程的解。

【核心技巧】记住 xy'=Dy，x²y''=D(D-1)y，x³y'''=D(D-1)(D-2)y 就够用了。D 的本质是"对 ln x 求导"的运算。

──────────────────────────────────────────────────────────────────────

【例题】求欧拉方程 x³y''' + x²y'' - 4xy' = 3x² 的通解。

解：作变换 x = e^t（t = ln x），利用 D 记号：

    x³y''' = D(D-1)(D-2)y，  x²y'' = D(D-1)y，  xy' = Dy，  x² = e^{2t}

代入原方程得：

    D(D-1)(D-2)y + D(D-1)y - 4Dy = 3e^{2t}

即  (D³-2D²-3D)y = 3e^{2t}，展开就是：

    d³y/dt³ - 2 d²y/dt² - 3 dy/dt = 3e^{2t}

对应齐次方程的特征方程为 r³-2r²-3r = 0，
即 r(r+1)(r-3) = 0，根为 r₁=0, r₂=-1, r₃=3。

齐次通解：Y = C₁ + C₂e^{-t} + C₃e^{3t}
换回 x：Y = C₁ + C₂/x + C₃x³

非齐次项 3e^{2t} 属于 e^{λt} 型（λ=2，不是特征根），设特解 y* = b e^{2t} = b x²。
代入化简求得 b = -1/2，于是 y* = -½ x²。

原方程通解：y = C₁ + C₂/x + C₃x³ - ½ x²

容易验证，这个解在 x<0 时也适用。

【习题 7-9（要点）】

求下列欧拉方程的通解：
1. x²y''+xy'-y=0
2. y''-(1/x)y'+(1/x²)y=2/x
3. x³y'''+3x²y''-2xy'+2y=0
4. x²y''-2xy'+2y=ln²x-2ln x
5. x²y''+xy'-4y=x³
6. x²y''-xy'+4y=x sin(ln x)
7. x²y''-3xy'+4y=x+x²ln x
8. x²y''+2xy'-2y=x³ln x+3x

（提示：每题都做代换 x=e^t，化为常系数方程后再求解。）
