3.6 函数图形的描绘

【为什么要学描绘函数图形】

借助一阶导数的符号，我们可以确定函数图形在哪个区间上上升、在哪个区间上下降；借助二阶导数的符号，可以确定函数图形在哪个区间上是凹的、哪个区间是凸的，以及拐点在哪里。掌握了函数图形的升降、凹凸和拐点后，也就掌握了函数的整体性态，可以把函数的图形画得比较准确。

现在，随着计算机技术的发展，借助计算机和数学软件可以很方便地画出各种函数的图形。但是，如何识别机器作图中的误差？如何掌握图形上的关键点？如何选择作图的范围从而进行人工干预？——这些仍然需要我们掌握运用微分学方法描绘函数图形的基本知识。

【描绘函数图形的五个步骤】

第一步：确定函数 $y = f(x)$ 的定义域及函数具有的某些特性（如奇偶性、周期性等），并求出函数的一阶导数 $f'(x)$ 和二阶导数 $f''(x)$。

第二步：求出一阶导数 $f'(x)$ 和二阶导数 $f''(x)$ 在函数定义域内的全部零点，并求出 $f(x)$ 的间断点及 $f'(x)$、$f''(x)$ 不存在的点。用这些点把函数的定义域划分成几个部分区间。

第三步：确定在这些部分区间内 $f'(x)$ 和 $f''(x)$ 的符号，并由此确定函数图形的升降、凹凸和拐点。

第四步：确定函数图形的水平渐近线、铅直渐近线以及其他变化趋势。

第五步：算出 $f'(x)$、$f''(x)$ 的零点以及不存在的点所对应的函数值，定出图形上相应的点。为了把图形描绘得更准确些，有时还需要补充一些点，然后结合第三、四步中得到的结果，连接这些点画出函数 $y = f(x)$ 的图形。

【例1】画出函数 $y = x^3 - x^2 - x + 1$ 的图形。

解：(1) 定义域为 $(-\infty, +\infty)$。求导：
$$f'(x) = 3x^2 - 2x - 1 = (3x + 1)(x - 1)$$
$$f''(x) = 6x - 2 = 2(3x - 1)$$

(2) $f'(x)$ 的零点为 $x = -\frac{1}{3}$ 和 $x = 1$，$f''(x)$ 的零点为 $x = \frac{1}{3}$。按从小到大排列，将定义域划分成四个部分区间：
$$(-\infty, -\tfrac{1}{3}], \quad [-\tfrac{1}{3}, \tfrac{1}{3}], \quad [\tfrac{1}{3}, 1], \quad [1, +\infty)$$

(3) 在 $(-\infty, -\frac{1}{3}]$ 上，$f'(x) > 0$，$f''(x) < 0$，所以曲线弧上升而且是凸的。
在 $[-\frac{1}{3}, \frac{1}{3}]$ 上，$f'(x) < 0$，$f''(x) < 0$，所以曲线弧下降而且是凸的。
在 $[\frac{1}{3}, 1]$ 上，$f'(x) < 0$，$f''(x) > 0$，曲线弧下降而且是凹的。
在 $[1, +\infty)$ 上，$f'(x) > 0$，$f''(x) > 0$，曲线弧上升而且是凹的。

(4) 当 $x \to +\infty$ 时，$y \to +\infty$；当 $x \to -\infty$ 时，$y \to -\infty$。

(5) 算出关键点的函数值：
$$f(-\tfrac{1}{3}) = \frac{32}{27}, \quad f(\tfrac{1}{3}) = \frac{16}{27}, \quad f(1) = 0$$

适当补充一些点：$f(-1) = 0$，$f(0) = 1$，$f(\frac{3}{2}) = \frac{5}{8}$。结合(3)、(4)得到的结果，即可画出函数 $y = x^3 - x^2 - x + 1$ 的图形（图3-24）。

【例2】描绘函数 $y = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$ 的图形。

这个函数其实就是概率论中非常重要的一条曲线——标准正态分布的概率密度函数。

解：(1) 定义域为 $(-\infty, +\infty)$。由于 $f(-x) = f(x)$，所以 $f(x)$ 是偶函数，图形关于 $y$ 轴对称。因此可以只讨论 $[0, +\infty)$ 上的图形。求导：
$$f'(x) = \frac{1}{\sqrt{2\pi}}(-x)e^{-\frac{x^2}{2}}$$
$$f''(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}(x^2 - 1)$$

(2) 在 $[0, +\infty)$ 上，$f'(x)$ 的零点为 $x = 0$；$f''(x)$ 的零点为 $x = 1$。用 $x = 1$ 把 $[0, +\infty)$ 划分成两个区间 $[0, 1]$ 和 $[1, +\infty)$。

(3) 在 $(0, 1)$ 内，$f'(x) < 0$，$f''(x) < 0$，所以在 $[0, 1]$ 上的曲线弧下降而且是凸的。
在 $(1, +\infty)$ 内，$f'(x) < 0$，$f''(x) > 0$，所以在 $[1, +\infty)$ 上的曲线弧下降而且是凹的。

(4) 由于 $\lim_{x \to \infty} f(x) = 0$，所以图形有一条水平渐近线 $y = 0$。

(5) 算出关键点处的函数值：$f(0) = \frac{1}{\sqrt{2\pi}}$，$f(1) = \frac{1}{\sqrt{2\pi e}}$。

结合(3)、(4)的讨论，画出 $y = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$ 在 $[0, +\infty)$ 上的图形。最后，利用图形的对称性，便可得到函数在 $(-\infty, 0]$ 上的图形（图3-25）。

【例3】描绘函数 $y = \frac{36(x-3)}{(x+3)^2}$ 的图形。

解：(1) 定义域为 $(-\infty, -3) \cup (-3, +\infty)$。求导：
$$f'(x) = \frac{36(3 - x)}{(x+3)^3}$$
$$f''(x) = \frac{72(x - 6)}{(x+3)^4}$$

(2) $f'(x)$ 的零点为 $x = 3$，$f''(x)$ 的零点为 $x = 6$；$x = -3$ 是函数的间断点。点 $x = -3$、$x = 3$ 和 $x = 6$ 把定义域划分成四个部分区间：
$$(-\infty, -3), \quad (-3, 3], \quad [3, 6], \quad [6, +\infty)$$

(3) 在各部分区间内 $f'(x)$ 及 $f''(x)$ 的符号、相应曲线弧的升降和凹凸情况如下：

- $(-\infty, -3)$：$f'(x) < 0$，$f''(x) < 0$，下降且凸
- $(-3, 3]$：$f'(x) > 0$，$f''(x) < 0$，上升且凸
- $[3, 6]$：$f'(x) < 0$，$f''(x) < 0$，下降且凸
- $[6, +\infty)$：$f'(x) < 0$，$f''(x) > 0$，下降且凹

(4) $\lim_{x \to \infty} f(x) = 1$，所以有一条水平渐近线 $y = 1$；$\lim_{x \to -3} f(x) = -\infty$，所以有一条铅直渐近线 $x = -3$。

(5) 算出关键点处的函数值：$f(3) = 4$，$f(6) = \frac{11}{4}$。又算出：
$f(0) = 1$，$f(-1) = -8$，$f(-9) = -8$，$f(-15) = -\frac{11}{4}$。

结合(3)、(4)得到的结果，画出函数图形（图3-26）。
