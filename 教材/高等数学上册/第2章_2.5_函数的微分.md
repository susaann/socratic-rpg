微分的概念是导数之后自然延伸出来的。如果说导数回答"变化率是多少"，那么微分回答"当变量变化一点点时，函数值大约变化多少"。

一、微分的定义

【从正方形面积说起】

一块正方形金属薄片，边长从 x₀ 变为 x₀ + Δx，面积 A = x² 改变了多少？

ΔA = (x₀ + Δx)² - x₀² = 2x₀·Δx + (Δx)²

这个增量分两部分：
- 第一部分 2x₀·Δx 是 Δx 的线性函数（与 Δx 成正比）；
- 第二部分 (Δx)² 当 Δx → 0 时是比 Δx 高阶的无穷小，可以忽略。

所以当 Δx 很小时，ΔA ≈ 2x₀·Δx。这个近似值 2x₀·Δx 就是微分的原型。

【严格定义】

设函数 y = f(x) 在某区间内有定义。如果函数的增量

Δy = f(x₀ + Δx) - f(x₀)

可以表示为 Δy = A·Δx + o(Δx)，其中 A 是不依赖于 Δx 的常数，o(Δx) 是比 Δx 高阶的无穷小，那么就称 f(x) 在 x₀ 处可微。而 A·Δx 就叫做 f(x) 在 x₀ 处相应于 Δx 的微分，记作 dy = A·Δx。

【可微与可导的关系】

关键结论：函数在某点可微的充分必要条件是在该点可导！而且 A = f'(x₀)。

证明很简单：Δy/Δx = A + o(Δx)/Δx → A（Δx → 0），所以 f'(x₀) = A。反之亦然。

因此，微分可以用导数来表达：dy = f'(x₀)·Δx。

当 f'(x₀) ≠ 0 时，Δy 与 dy 是等价无穷小（Δx → 0），也就是说 Δy ≈ dy，误差是比 Δx 更高阶的无穷小。所以微分的实质就是：用线性函数来近似替代非线性函数在小范围内的变化量。

【函数微分的一般形式】

在任意点 x 处，微分记作 dy = f'(x)·Δx。通常把自变量的增量 Δx 记作 dx（自变量的微分），于是：

dy = f'(x)·dx，即 f'(x) = dy/dx。

这就是为什么导数又记作 dy/dx——它确实是微分 dy 除以微分 dx！这个记号现在有了"商"的含义。

例1：y = x² 在 x = 1 和 x = 3 处的微分。
解：dy = (x²)'·Δx = 2x·Δx。
在 x = 1：dy = 2·Δx；在 x = 3：dy = 6·Δx。

例2：y = x³，求 x = 2, Δx = 0.02 时的微分。
解：dy = 3x²·Δx。当 x = 2, Δx = 0.02：dy = 3·4·0.02 = 0.24。

二、微分的几何意义

在函数 y = f(x) 的图形上，取点 M(x₀, y₀)。当 x 增加 Δx 后，得到曲线上另一点 N(x₀+Δx, y₀+Δy)。

过 M 做切线，切线的斜率为 f'(x₀)。在切线上，与 Δx 对应的纵坐标增量等于 f'(x₀)·Δx——这恰好就是微分 dy！

所以几何上：
- Δy = QN（曲线上纵坐标的增量）
- dy = QP（切线上纵坐标的增量）

当 |Δx| 很小时，|Δy - dy| 非常小。也就是说，在 M 附近，可以用切线近似代替曲线本身。这叫"非线性函数的局部线性化"，是微分学的核心思想之一——把弯曲的东西在局部拉直来看。

三、基本微分公式与运算法则

由于 dy = f'(x)·dx，所以求微分归根到底就是求导再乘以 dx。所有导数公式都可以转化成微分公式：

【基本微分公式】（选列常用）
- d(x^μ) = μx^(μ-1)dx
- d(sin x) = cos x dx
- d(cos x) = -sin x dx
- d(tan x) = sec²x dx
- d(e^x) = e^x dx
- d(ln x) = (1/x)dx
- d(arcsin x) = dx/√(1-x²)
- d(arctan x) = dx/(1+x²)

【运算法则】
- d(u ± v) = du ± dv
- d(Cu) = C du
- d(uv) = v du + u dv
- d(u/v) = (v du - u dv)/v²（v ≠ 0）

【复合函数的微分——微分形式不变性】

复合函数 y = f[g(x)] 的微分：dy = f'(u)·g'(x)·dx = f'(u)·du。

注意这里：不管 u 是自变量还是中间变量，微分形式 dy = f'(u)·du 始终保持不变！这叫做"微分形式不变性"，是微分运算的巨大优势——你可以"套用"公式而不用担心中间变量。

例3：y = sin(2x+1)，求 dy。
把 2x+1 看作 u，则 dy = cos u·du = cos(2x+1)·d(2x+1) = 2cos(2x+1)dx。

例4：y = ln(1+e^(x²))，求 dy。
dy = [1/(1+e^(x²))]·d(1+e^(x²)) = [1/(1+e^(x²))]·e^(x²)·d(x²) = [2x·e^(x²)/(1+e^(x²))]dx。

例5：y = e^(-3x)·cos x，求 dy。
用积的微分法则：dy = cos x·d(e^(-3x)) + e^(-3x)·d(cos x)
= cos x·e^(-3x)·(-3)dx + e^(-3x)·(-sin x)dx
= -e^(-3x)(3cos x + sin x)dx。

四、微分在近似计算中的应用

如果 |Δx| 很小，且 f'(x₀) ≠ 0，则有近似公式：

f(x₀ + Δx) ≈ f(x₀) + f'(x₀)·Δx

也就是说，想知道 x 变一点点后函数值大致变成多少，用"当前值 + 导数×变化量"就能估出来。

取 x₀ = 0，就得到常用的"小 x 近似公式"：
- (1+x)^α ≈ 1 + αx（|x| 很小时）
- sin x ≈ x（x 用弧度）
- tan x ≈ x
- e^x ≈ 1 + x
- ln(1+x) ≈ x

例6：半径为 1 cm 的球镀 0.01 cm 铜，估需铜量。
球体积 V = (4/3)πR³，dV = 4πR²·dR。
R₀ = 1, dR = 0.01：ΔV ≈ 4π·1²·0.01 ≈ 0.13 cm³。
质量 ≈ 0.13 × 8.9 ≈ 1.16 g。

例7：计算 sin 30°30' 的近似值。
30°30' = π/6 + π/360（弧度）。
取 x₀ = π/6，Δx = π/360，f(x) = sin x，f'(x) = cos x。
sin(π/6 + π/360) ≈ sin(π/6) + cos(π/6)·(π/360)
≈ 0.5 + (√3/2)×0.0087 ≈ 0.5076。

这些近似公式在工程实践中非常有用——复杂的计算用简单的线性近似代替，精度还足够。
