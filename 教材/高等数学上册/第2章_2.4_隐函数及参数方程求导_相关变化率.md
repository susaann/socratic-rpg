前面见过的函数大多是"显函数"——y 直接写成 x 的表达式，如 y = sin x。但有时候 y 和 x 的关系是一个方程，比如 x + y³ - 1 = 0，y 没有直接"显"出来，这叫隐函数。另外，曲线也经常用参数方程来表示。本节就学这两类函数的求导方法。

一、隐函数的导数

【什么是隐函数】

方程 F(x, y) = 0 在一定条件下能确定 y 是 x 的函数（满足方程），但没有写成 y = f(x) 的显式形式，这就是隐函数。比如 x + y³ - 1 = 0，虽然可以解出 y = ∛(1-x)（显化），但很多隐函数是解不出来的。

【隐函数求导法】

方法很简单：方程两边同时对 x 求导，把 y 看成 x 的函数 y(x)，遇到含 y 的项就用复合函数求导法则。

例1：求方程 e^y + xy - e = 0 确定的隐函数的导数 dy/dx。
解：两边对 x 求导，注意 y 是 x 的函数：
(e^y)·y' + (1·y + x·y') - 0 = 0
整理得：y'(e^y + x) + y = 0
所以 dy/dx = -y/(e^y + x)。

例2：求曲线 x² + y² = 1 在点 (1/2, √3/2) 处的切线方程。
解：两边对 x 求导：2x + 2y·y' = 0，得 y' = -x/y。
在给定点处，y' = -(1/2)/(√3/2) = -1/√3。
切线方程：y - √3/2 = (-1/√3)(x - 1/2)。

例3：求方程 x - y + (1/2)sin y = 0 确定的隐函数的二阶导数。
解：先求一阶导：1 - y' + (1/2)cos y·y' = 0，得 y' = 1/(2 - cos y)。
再对 x 求二阶导：y'' = d/dx [1/(2-cos y)] = [sin y·y']/(2-cos y)²。
代入 y' 得 y'' = sin y / [(2-cos y)³]。

【对数求导法】

对于形如 y = u(x)^(v(x))（幂指函数）或含有多个因子的乘除式，两边先取对数再求导非常方便。

例4：求 y = x^(sin x)（x > 0）的导数。
解：ln y = sin x·ln x。两边对 x 求导：
y'/y = cos x·ln x + sin x/x。
所以 y' = x^(sin x)·(cos x·ln x + sin x/x)。

一般地，幂指函数 y = u^v（u > 0）的导数为 y' = u^v (v'·ln u + v·u'/u)。

例5：求 y = √[(x-1)(x-2)/((x-3)(x-4))]（x > 4）的导数。
解：ln y = (1/2)[ln(x-1) + ln(x-2) - ln(x-3) - ln(x-4)]。
两边求导：y'/y = (1/2)[1/(x-1) + 1/(x-2) - 1/(x-3) - 1/(x-4)]。
所以 y' = (y/2)[1/(x-1) + 1/(x-2) - 1/(x-3) - 1/(x-4)]。

你看，如果不取对数，直接对根号里面那一大块求导该多麻烦！

二、由参数方程所确定的函数的导数

【参数方程表示的函数】

有些曲线用 x 和 y 分别表示为第三个变量 t（称为参数）的函数更方便：
{ x = φ(t)
{ y = ψ(t)

这叫做参数方程。当 φ(t) 单调且可导、φ'(t) ≠ 0 时，t 可以反解为 x 的函数，从而 y 也是 x 的函数。

【求导公式】

dy/dx = (dy/dt) / (dx/dt) = ψ'(t)/φ'(t)

简单记：y 对 x 的导数 = y 对 t 的导数 ÷ x 对 t 的导数。"上导除下导"。

二阶导数公式：
d²y/dx² = d(dy/dx)/dx = [d(dy/dx)/dt] / (dx/dt)
= [ψ''(t)φ'(t) - ψ'(t)φ''(t)] / [φ'(t)]³

注意：二阶导不是直接对一阶导再套用一次除法，而是要把一阶导看作 t 的函数，再除以 dx/dt。

【例题】

例6：椭圆参数方程 x = a cos t, y = b sin t，求在 t = π/4 处的切线斜率。
解：dy/dx = (b cos t) / (-a sin t) = -(b/a) cot t。
在 t = π/4 处，斜率为 -(b/a)。

例7：抛射体轨迹 x = v₁t, y = v₂t - (1/2)gt²，求速度方向和大小。
解：水平分速度 dx/dt = v₁，竖直分速度 dy/dt = v₂ - gt。
速度大小 v = √[v₁² + (v₂ - gt)²]。
切线方向 tan α = dy/dx = (v₂ - gt)/v₁。

例8：摆线 x = a(t - sin t), y = a(1 - cos t)，求二阶导数。
解：dy/dx = [a sin t] / [a(1 - cos t)] = sin t/(1 - cos t)。
d²y/dx² = … = -1/[a(1 - cos t)²]（t ≠ 2nπ）。

三、相关变化率

如果 x = x(t) 和 y = y(t) 都随时间 t 变化，且 x 和 y 之间存在某种方程关系，那么 dx/dt 和 dy/dt 之间也有关系。已知其中一个，就可求出另一个——这就是相关变化率问题。

例：气球从离观察员 500 m 处以 140 m/min 的速度垂直上升，求气球高度 500 m 时仰角的增加速率。
解：tan α = h/500。两边对 t 求导：sec²α·(dα/dt) = (1/500)·(dh/dt)。
当 h = 500 时，tan α = 1，sec²α = 2，dh/dt = 140。
代入：2·(dα/dt) = 140/500，得 dα/dt = 0.14 rad/min。
