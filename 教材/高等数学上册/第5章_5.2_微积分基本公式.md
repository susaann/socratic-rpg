5.2 微积分基本公式

【引言——为什么要寻找新方法】

在5.1节中，我们按定义计算了 $\int_0^1 x^2\,dx = \frac{1}{3}$，仅仅是 $x^2$ 这样的简单函数已经很不轻松。如果被积函数更复杂，直接用定义计算几乎不可行。因此，必须寻找计算定积分的新方法。

下面我们先从变速直线运动的联系中寻找灵感。

【一、变速直线运动中位置函数与速度函数之间的联系】

有一物体沿数轴做直线运动，在时刻 $t$ 的位置为 $s(t)$，速度为 $v(t)$（设 $v(t) \geq 0$）。

一方面，物体在 $[T_1, T_2]$ 内的路程可以用定积分表示为 $\int_{T_1}^{T_2} v(t)\,dt$；
另一方面，路程又可以用位置函数的增量 $s(T_2) - s(T_1)$ 来表示。

由于 $s'(t) = v(t)$，即位置函数 $s(t)$ 就是速度函数 $v(t)$ 的原函数。于是我们得到了：
$$\int_{T_1}^{T_2} v(t)\,dt = s(T_2) - s(T_1)$$

这提示了一个重要事实：定积分的值等于被积函数的原函数在积分区间上的增量！

【二、积分上限的函数及其导数】

设 $f(x)$ 在 $[a, b]$ 上连续，$x$ 为 $[a, b]$ 上的一点。考虑变上限的定积分：
$$\Phi(x) = \int_a^x f(t)\,dt \quad (a \leq x \leq b)$$

当上限 $x$ 变动时，这个积分定义了一个函数 $\Phi(x)$。

定理1（变上限积分求导公式——微积分第一基本定理）：如果 $f(x)$ 在 $[a, b]$ 上连续，那么积分上限的函数
$$\Phi(x) = \int_a^x f(t)\,dt$$
在 $[a, b]$ 上可导，且
$$\Phi'(x) = \frac{d}{dx}\int_a^x f(t)\,dt = f(x)$$

证明思路：增量 $\Delta\Phi = \Phi(x+\Delta x) - \Phi(x) = \int_x^{x+\Delta x} f(t)\,dt$，由积分中值定理 $= f(\xi)\Delta x$，其中 $\xi$ 介于 $x$ 与 $x+\Delta x$ 之间。取极限 $\Delta x \to 0$ 即得 $\Phi'(x) = f(x)$。

这个定理揭示了一个极其重要的关系：连续函数 $f(x)$ 取变上限积分再求导，结果就是 $f(x)$ 本身！从原函数的定义立即得到：

定理2（原函数存在定理）：如果 $f(x)$ 在 $[a, b]$ 上连续，那么函数
$$\Phi(x) = \int_a^x f(t)\,dt$$
就是 $f(x)$ 在 $[a, b]$ 上的一个原函数。

换句话说：连续函数一定有原函数，而且这个原函数可以用变上限积分具体构造出来！

【三、牛顿-莱布尼茨公式（微积分基本定理）】

定理3（微积分基本定理）：如果函数 $F(x)$ 是连续函数 $f(x)$ 在 $[a, b]$ 上的一个原函数，那么
$$\int_a^b f(x)\,dx = F(b) - F(a)$$

证明：由定理2，$\Phi(x) = \int_a^x f(t)\,dt$ 是 $f(x)$ 的一个原函数。又已知 $F(x)$ 也是 $f(x)$ 的原函数，所以 $F(x) - \Phi(x) = C$（常数）。令 $x=a$，得 $F(a) - 0 = C$，故 $C = F(a)$。于是 $\Phi(x) = F(x) - F(a)$。令 $x=b$，即得 $\int_a^b f(x)\,dx = F(b) - F(a)$。

为了方便，常把 $F(b) - F(a)$ 记作 $\left[F(x)\right]_a^b$，于是
$$\int_a^b f(x)\,dx = \left[F(x)\right]_a^b$$

这就是牛顿-莱布尼茨公式——微积分中最重要的公式！它把定积分的计算转化为求原函数的问题：只需找到 $f(x)$ 的一个原函数 $F(x)$，然后计算 $F(b)-F(a)$ 即可。

这个公式的意义怎么强调都不为过。有了它，第四章学的不定积分技术（换元法、分部积分法）就可以直接用来计算定积分了。微积分的两大分支——微分学和积分学——通过这个公式完美地统一在一起。

例：计算 $\int_0^1 x^2\,dx$。
解：$x^2$ 的一个原函数是 $\frac{x^3}{3}$，所以 $\int_0^1 x^2\,dx = \left[\frac{x^3}{3}\right]_0^1 = \frac{1}{3} - 0 = \frac{1}{3}$。
——比起5.1节中用定义苦算几页纸，这个计算过程简直不费吹灰之力！

定理1的进一步推广：变上限求导的链式法则。
$$\frac{d}{dx}\int_a^{\varphi(x)} f(t)\,dt = f(\varphi(x)) \cdot \varphi'(x)$$
这对于含变上限积分的极限问题（配合洛必达法则）非常有用。
