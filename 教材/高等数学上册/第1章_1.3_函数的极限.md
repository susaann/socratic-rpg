上一节讲了数列的极限——数列可以看作自变量为正整数 n 的函数 x_n = f(n)。把数列极限概念中"n → ∞"这种特殊性拿掉，推广到一般的函数，就得到了函数极限的概念。

一句话概括：在自变量的某个变化过程中，如果对应的函数值无限接近于某个确定的数，那么这个数就叫做该过程中函数的极限。

自变量的变化过程不同，极限的形式也不同。主要研究两种情形：
（1）x 趋近于某个有限值 x₀（记作 x → x₀）；
（2）x 的绝对值无限增大，即 x 趋于无穷大（记作 x → ∞）。

一、函数极限的定义

【情形一：自变量趋于有限值 x → x₀】

考虑 x 趋近于 x₀ 的过程。如果在 x → x₀ 的过程中，函数值 f(x) 无限接近于某个确定的数值 A，就说 A 是 f(x) 当 x → x₀ 时的极限。注意：函数在 x₀ 处有没有定义都没关系——我们关心的是 x 靠近 x₀ 时 f(x) 趋近于什么值，而不是 x = x₀ 这一点的值。

"f(x) 无限接近 A"怎样精确描述？就是 |f(x) - A| 可以任意小，即 ∀ε > 0（任意给定正数 ε），|f(x) - A| < ε。而 x 充分接近 x₀ 则用 0 < |x - x₀| < δ 来表示（δ 是某个正数）。0 < |x - x₀| 意味着 x ≠ x₀。从几何上看，0 < |x - x₀| < δ 就是 x₀ 的去心 δ 邻域——以 x₀ 为中心、半径为 δ 的开区间，但挖掉中心点。

正式定义（ε-δ 语言）：设函数 f(x) 在点 x₀ 的某一去心邻域内有定义。如果存在常数 A，对于任意给定的正数 ε（不论多么小），总存在正数 δ，使得当 0 < |x - x₀| < δ 时，对应的函数值 f(x) 都满足 |f(x) - A| < ε，那么常数 A 就叫做 f(x) 当 x → x₀ 时的极限，记作 lim_{x→x₀} f(x) = A 或 f(x) → A (x → x₀)。

符号简写：lim_{x→x₀} f(x) = A ⇔ ∀ε > 0, ∃δ > 0, 当 0 < |x - x₀| < δ 时，有 |f(x) - A| < ε。

几何解释：给定 ε，做两条水平线 y = A + ε 和 y = A - ε，形成一条横条区域。则存在 x₀ 的一个去心 δ 邻域，使得在这个邻域内，函数图形上的点全部落在横条区域中。换句话说：x 越靠近 x₀，f(x) 就越靠近 A。

【证明极限的几个例子】

例1：证明 lim_{x→x₀} c = c（常数的极限就是它自己）。

证明：|f(x) - A| = |c - c| = 0 < ε 永远成立，所以任取 δ > 0 都行。证毕。

例2：证明 lim_{x→x₀} x = x₀。

证明：|f(x) - A| = |x - x₀|。∀ε > 0，取 δ = ε，则当 0 < |x - x₀| < δ 时自然 |x - x₀| < ε。证毕。

例3：证明 lim_{x→1} (2x - 1) = 1。

证明：|f(x) - A| = |(2x-1) - 1| = 2|x-1|。∀ε > 0，要使 2|x-1| < ε，只需 |x-1| < ε/2。所以取 δ = ε/2 即可。

例4：证明 lim_{x→1} (x² - 1)/(x - 1) = 2。

注意：这个函数在 x = 1 处没有定义（分母为 0），但极限与它无关。因为当 x ≠ 1 时，(x² - 1)/(x - 1) = x + 1，所以 |f(x) - 2| = |(x+1) - 2| = |x - 1|。取 δ = ε 即得证。

例5：证明当 x₀ > 0 时，lim_{x→x₀} √x = √x₀。

证明：|√x - √x₀| = |x - x₀| / (√x + √x₀) ≤ |x - x₀| / √x₀。要使这个值小于 ε，只需 |x - x₀| < √x₀ · ε。同时还要保证 x ≥ 0，所以需要 |x - x₀| < x₀。取 δ = min{x₀, √x₀ · ε} 即可。

【左极限与右极限】

前面说 x → x₀ 是 x 从左右两边任意方式趋近。但有时需要单独考虑 x 只从左边趋近（x → x₀⁻）或只从右边趋近（x → x₀⁺）。

左极限：在极限定义中把 0 < |x - x₀| < δ 改为 x₀ - δ < x < x₀，得到的 A 称为左极限，记作 lim_{x→x₀⁻} f(x) = A 或 f(x₀⁻) = A。

右极限：把条件改为 x₀ < x < x₀ + δ，记作 lim_{x→x₀⁺} f(x) = A 或 f(x₀⁺) = A。

左右极限统称单侧极限。

重要结论：lim_{x→x₀} f(x) 存在的充分必要条件是左右极限都存在并且相等：f(x₀⁻) = f(x₀⁺)。

例6：函数 f(x) = { x-1, x<0; 0, x=0; x+1, x>0 }，讨论 x → 0 时的极限。

解：左极限 lim_{x→0⁻} f(x) = lim_{x→0⁻} (x-1) = -1，右极限 lim_{x→0⁺} f(x) = lim_{x→0⁺} (x+1) = 1。因为 -1 ≠ 1，所以 lim_{x→0} f(x) 不存在。

【情形二：自变量趋于无穷大 x → ∞】

如果在 |x| 无限增大的过程中，f(x) 无限接近于确定的数值 A，就称 A 为 x → ∞ 时 f(x) 的极限。

正式定义：设 f(x) 当 |x| 大于某个正数时有定义。如果存在常数 A，对于 ∀ε > 0，总 ∃X > 0，使得当 |x| > X 时，有 |f(x) - A| < ε，则称 A 为 x → ∞ 时 f(x) 的极限，记作 lim_{x→∞} f(x) = A。

符号简写：lim_{x→∞} f(x) = A ⇔ ∀ε > 0, ∃X > 0, 当 |x| > X 时，有 |f(x) - A| < ε。

类似地：
- x → +∞ 时：把 |x| > X 改为 x > X
- x → -∞ 时：把 |x| > X 改为 x < -X

几何解释：做直线 y = A - ε 和 y = A + ε，总存在正数 X，使得当 x < -X 或 x > X 时，函数图形完全位于这两条直线之间。此时直线 y = A 是函数图形的水平渐近线。

例7：证明 lim_{x→∞} 1/x = 0。

证明：∀ε > 0，要使 |1/x - 0| = 1/|x| < ε，只需 |x| > 1/ε。所以取 X = 1/ε，当 |x| > X 时结论成立。直线 y = 0 是 y = 1/x 的水平渐近线。

二、函数极限的性质

函数极限的性质与收敛数列的性质很类似。下面以 x → x₀ 的情形为代表来叙述，其他情形类似。

【定理1：函数极限的唯一性】如果 lim_{x→x₀} f(x) 存在，则极限唯一。

【定理2：函数极限的局部有界性】如果 lim_{x→x₀} f(x) = A，则存在常数 M > 0 和 δ > 0，使得当 0 < |x - x₀| < δ 时，|f(x)| ≤ M。

理解："局部有界"是说在 x₀ 附近的某个小范围内，函数值不会无穷大。证明很简单：取 ε = 1，存在 δ 使得 |f(x) - A| < 1，于是 |f(x)| ≤ |f(x) - A| + |A| < 1 + |A|，令 M = |A| + 1 即可。

【定理3：函数极限的局部保号性】如果 lim_{x→x₀} f(x) = A，且 A > 0（或 A < 0），则存在 δ > 0，使得当 0 < |x - x₀| < δ 时，f(x) > 0（或 f(x) < 0）。

直观理解：如果极限是正的，那么在离 x₀ 足够近的范围内，函数值也都是正的。就像你往一个正的方向走，靠近目标时自然在正的区域。

证明（A > 0）：取 ε = A/2 > 0，存在 δ 使得当 0 < |x - x₀| < δ 时，|f(x) - A| < A/2，从而 f(x) > A - A/2 = A/2 > 0。

更强的结论（定理3'）：如果 lim f(x) = A ≠ 0，则存在 x₀ 的某去心邻域，使得在该邻域内 |f(x)| > |A|/2。

推论：如果在 x₀ 的某去心邻域内 f(x) ≥ 0（或 f(x) ≤ 0），且 lim f(x) = A，则 A ≥ 0（或 A ≤ 0）。

【定理4：函数极限与数列极限的关系】如果 lim_{x→x₀} f(x) 存在，而 { x_n } 是 f(x) 定义域内任一收敛于 x₀ 的数列，且 x_n ≠ x₀ (n = 1, 2, …)，那么相应的函数值数列 { f(x_n) } 必定收敛，且 lim_{n→∞} f(x_n) = lim_{x→x₀} f(x)。

这个定理可以这样记：当 x 沿任何一条"数列路径"趋近 x₀ 时，f(x) 都趋于同一个值。反过来说，如果能找到两条不同的路径（数列），使得 f 沿这两条路径的极限不同，那么函数极限就不存在。这在实际判断时非常有用。
