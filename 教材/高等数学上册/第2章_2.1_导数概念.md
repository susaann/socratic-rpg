导数是微分学中最核心的概念。它到底是什么意思？简单说，导数就是"变化率"的精确数学表达——它回答"某个量变化得有多快"这个问题。本节从两个实际问题出发引出导数，然后给出严格定义和几何意义。

一、引例

【速度问题】

设一个质点沿直线运动，它在时刻 t 的位置坐标为 s = f(t)，这叫位置函数。如果是匀速运动，速度就是路程除以时间。但如果是非匀速运动呢？某一时刻的速度怎么定义？

思路是这样的：取时间间隔 [t₀, t]，在这段时间内的平均速度是 (f(t) - f(t₀))/(t - t₀)。如果让时间间隔越来越短（t → t₀），这个平均速度会趋近于某个极限值——这个极限值就是质点在 t₀ 时刻的瞬时速度。即 v = lim_{t→t₀} (f(t) - f(t₀))/(t - t₀)。

【切线问题】

圆的切线好定义——"和圆只有一个交点的直线"即可。但对于一般曲线，这个定义不行。比如抛物线 y = x² 在原点处，x 轴和 y 轴都和它只有一个交点，但显然只有 x 轴才算切线。

正确的定义是割线的极限位置：在曲线上取点 M，再取另外一点 N，做割线 MN。当 N 沿曲线无限趋近 M 时，如果割线 MN 绕 M 旋转趋于一个极限位置 MT，这条直线 MT 就是切线。

对于曲线 y = f(x)，设 M(x₀, y₀)，N(x, f(x))。割线 MN 的斜率为 (f(x) - f(x₀))/(x - x₀)。当 x → x₀ 时，如果这个比值存在极限 k，那么 k 就是切线的斜率。即 k = tan α = lim_{x→x₀} (f(x) - f(x₀))/(x - x₀)。

你看，两个完全不同的问题——速度和切线——居然归结为同一个数学形式：函数增量与自变量增量之比的极限。这就是导数的来源。

二、导数的定义

【定义】

设函数 y = f(x) 在点 x₀ 的某个邻域内有定义。当自变量 x 在 x₀ 处取得增量 Δx（x₀+Δx 仍在邻域内）时，相应的函数增量 Δy = f(x₀+Δx) - f(x₀)。如果 Δy 与 Δx 之比当 Δx → 0 时的极限存在，则称函数在 x₀ 处可导，并称这个极限为函数在 x₀ 处的导数，记作 f'(x₀)，即：

f'(x₀) = lim_{Δx→0} Δy/Δx = lim_{Δx→0} [f(x₀+Δx) - f(x₀)]/Δx

也可以等价地写成：
f'(x₀) = lim_{x→x₀} [f(x) - f(x₀)]/(x - x₀)
f'(x₀) = lim_{h→0} [f(x₀+h) - f(x₀)]/h

其他记号：y'|_{x=x₀}，dy/dx|_{x=x₀}，df(x)/dx|_{x=x₀}。

导数的本质就是"瞬时变化率"：Δy/Δx 是在区间 [x₀, x₀+Δx] 上的平均变化率，取极限 Δx → 0 得到的就是在点 x₀ 处的变化率。

如果函数在开区间 I 内每一点都可导，就称函数在 I 内可导。此时，对于 I 内每个 x 都有一个导数值与之对应，这就定义了一个新函数——导函数 f'(x)，简称导数。

【求导举例】

例1：常数函数 f(x) = C。f'(x) = lim_{h→0} (C-C)/h = 0。即常数的导数为零（常量不变化，变化率自然是零）。

例2：幂函数 f(x) = x^n（n 为正整数）。当 n = 1 时，f'(x) = 1。当 n > 1 时，利用二项式展开可得 f'(x) = n·x^(n-1)。

例3：幂函数 f(x) = x^μ（μ 为任意实数，x > 0）。同样有 f'(x) = μ·x^(μ-1)。这个公式非常通用。例如：(√x)' = (x^(1/2))' = 1/(2√x)；(1/x)' = (x^(-1))' = -1/x²。

例4：f(x) = sin x。f'(x) = lim_{h→0} [sin(x+h) - sin x]/h = lim_{h→0} cos(x+h/2)·[sin(h/2)/(h/2)] = cos x。即 (sin x)' = cos x。

例5：f(x) = cos x。类似可得 (cos x)' = -sin x。

例6：f(x) = a^x（a > 0, a ≠ 1）。f'(x) = a^x·ln a。特别地，(e^x)' = e^x——以 e 为底的指数函数，导数等于它自己，这是 e 的独特魅力。

例7：f(x) = log_a x。f'(x) = 1/(x·ln a)。特别地，(ln x)' = 1/x。

【单侧导数】

根据左右极限的概念，可以定义左右导数：
- 左导数 f'₋(x₀) = lim_{h→0⁻} [f(x₀+h) - f(x₀)]/h
- 右导数 f'₊(x₀) = lim_{h→0⁺} [f(x₀+h) - f(x₀)]/h

定理：f(x) 在 x₀ 可导的充分必要条件是左右导数都存在且相等。

例：f(x) = |x| 在 x = 0 处，左导数为 -1，右导数为 1，不相等，因此在 x = 0 不可导。图形上表现为"尖点"——在原点处折了一下，切线无法唯一确定。

三、导数的几何意义

函数 y = f(x) 在点 x₀ 处的导数 f'(x₀) 在几何上表示曲线 y = f(x) 在点 M(x₀, f(x₀)) 处切线的斜率，即 f'(x₀) = tan α，其中 α 是切线的倾角。

由此可得：
- 切线方程：y - y₀ = f'(x₀)·(x - x₀)
- 法线方程（切线垂直的直线，f'(x₀) ≠ 0 时）：y - y₀ = -(1/f'(x₀))·(x - x₀)

例：求等边双曲线 y = 1/x 在点 (1/2, 2) 处的切线和法线。
解：y' = (1/x)' = -1/x²，在 x = 1/2 处 y' = -4。
切线方程：y - 2 = -4(x - 1/2)，即 4x + y - 4 = 0。
法线斜率 k = 1/4，法线方程：y - 2 = (1/4)(x - 1/2)，即 2x - 8y + 15 = 0。

四、可导性与连续性的关系

定理：如果函数在某点可导，那么它在该点一定连续。

证明思路：可导意味着 Δy/Δx → f'(x)。则 Δy = f'(x)·Δx + α·Δx（α → 0）。当 Δx → 0 时，Δy → 0，所以连续。

但反过来不成立——连续不一定可导。两个反例：
- y = ∛x 在 x = 0 连续，但导数为无穷大（切线垂直于 x 轴），不可导。
- y = |x| 在 x = 0 连续（值=0，左右极限都是0），但尖点处左右导数不等，不可导。

总结：可导 → 连续，但连续 ↛ 可导。连续是可导的必要条件，不是充分条件。
