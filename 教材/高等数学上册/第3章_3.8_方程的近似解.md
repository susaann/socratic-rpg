3.8 方程的近似解

【引言】

在科学技术问题中，经常会遇到求解高次代数方程或其他类型的方程的问题。要求得这类方程的实根的精确值，往往比较困难，因此就需要寻求方程的近似解。

求方程 $f(x) = 0$ 的近似解，一般分两步走：

第一步是确定根的大致范围。具体来说，就是确定一个区间 $[a, b]$，使得所求的根是位于这个区间内的唯一实根（这个区间称为根的隔离区间）。这一步的工作叫做根的隔离。

由于方程 $f(x) = 0$ 的实根在几何上表示曲线 $y = f(x)$ 与 $x$ 轴交点的横坐标，因此为了确定根的隔离区间，可以先较精确地画出 $y = f(x)$ 的图形，然后从图上定出它与 $x$ 轴交点的大概位置。由于作图和读数的误差，这种做法得不到根的高精度近似值，但一般已经可以确定出根的隔离区间。

第二步是以根的隔离区间的端点作为根的初始近似值，逐步改善根的近似值的精确度，直至求得满足精确度要求的近似解。下面我们介绍三种常用的方法——二分法、切线法和割线法（弦截法）。按照这些方法编出简单的程序，就可以在计算机上求出方程足够精确的近似解。

【一、二分法】

设 $f(x)$ 在区间 $[a, b]$ 上连续，$f(a) \cdot f(b) < 0$，且方程 $f(x) = 0$ 在 $(a, b)$ 内仅有一个实根 $\xi$，则 $[a, b]$ 就是根的一个隔离区间。

二分法的基本思想非常简单：每次把区间对半分开，根据中点函数值的符号判断根在哪一半，然后舍去没有根的那一半。每做一次，区间长度减半。

具体来说：取 $[a, b]$ 的中点 $\xi_1 = \frac{a+b}{2}$，计算 $f(\xi_1)$。

- 如果 $f(\xi_1) = 0$，那么 $\xi = \xi_1$，已经找到精确解；
- 如果 $f(\xi_1)$ 与 $f(a)$ 同号，则取 $a_1 = \xi_1$，$b_1 = b$；
- 如果 $f(\xi_1)$ 与 $f(b)$ 同号，则取 $a_1 = a$，$b_1 = \xi_1$。

这样，新的隔离区间 $[a_1, b_1]$ 的长度为 $b_1 - a_1 = \frac{1}{2}(b - a)$，且 $\xi \in (a_1, b_1)$。

重复这个过程，做 $n$ 次之后，我们得到 $a_n < \xi < b_n$，且 $b_n - a_n = \frac{b - a}{2^n}$。如果以 $a_n$ 或 $b_n$ 作为 $\xi$ 的近似值，那么误差小于 $\frac{b - a}{2^n}$。

例1：用二分法求方程 $x^3 + 1.1x^2 + 0.9x - 1.4 = 0$ 的实根的近似值，使误差不超过 $10^{-3}$。

解：设 $f(x) = x^3 + 1.1x^2 + 0.9x - 1.4$，$f(x)$ 在 $(-\infty, +\infty)$ 内连续。

$f'(x) = 3x^2 + 2.2x + 0.9$，判别式 $\Delta = 2.2^2 - 4 \times 3 \times 0.9 = -5.96 < 0$，故 $f'(x) > 0$，$f(x)$ 在整个实数范围内单调增加，$f(x) = 0$ 至多有一个实根。

又 $f(0) = -1.4 < 0$，$f(1) = 1.6 > 0$，故 $f(x) = 0$ 在 $[0, 1]$ 内有一个唯一实根。取 $a = 0$，$b = 1$，$[0, 1]$ 就是一个隔离区间。

逐次二分计算（计算时取三位小数）：

$\xi_1 = 0.5$，$f(\xi_1) = -0.55 < 0$，故取 $a_1 = 0.5$，$b_1 = 1$；
$\xi_2 = 0.75$，$f(\xi_2) = 0.32 > 0$，故取 $a_2 = 0.5$，$b_2 = 0.75$；
$\xi_3 = 0.625$，$f(\xi_3) = -0.16 < 0$，故取 $a_3 = 0.625$，$b_3 = 0.75$；
$\xi_4 = 0.687$，$f(\xi_4) = 0.062 > 0$，故取 $a_4 = 0.625$，$b_4 = 0.687$；
$\xi_5 = 0.656$，$f(\xi_5) = -0.054 < 0$，故取 $a_5 = 0.656$，$b_5 = 0.687$；
$\xi_6 = 0.672$，$f(\xi_6) = 0.005 > 0$，故取 $a_6 = 0.656$，$b_6 = 0.672$；
$\xi_7 = 0.664$，$f(\xi_7) = -0.025 < 0$，故取 $a_7 = 0.664$，$b_7 = 0.672$；
$\xi_8 = 0.668$，$f(\xi_8) = -0.010 < 0$，故取 $a_8 = 0.668$，$b_8 = 0.672$；
$\xi_9 = 0.670$，$f(\xi_9) = -0.002 < 0$，故取 $a_9 = 0.670$，$b_9 = 0.672$；
$\xi_{10} = 0.671$，$f(\xi_{10}) = 0.001 > 0$，故取 $a_{10} = 0.670$，$b_{10} = 0.671$。

于是 $0.670 < \xi < 0.671$。以 $0.670$ 作为根的不足近似值，$0.671$ 作为根的过剩近似值，两者的误差都小于 $10^{-3}$。

二分法的优点是简单可靠、必定收敛；缺点是收敛速度比较慢，每步只能把区间长度减半。

【二、切线法（牛顿法）】

设 $f(x)$ 在 $[a, b]$ 上具有二阶导数，$f(a) \cdot f(b) < 0$，且 $f'(x)$ 和 $f''(x)$ 在 $[a, b]$ 上保持定号（即不改变符号）。在这些条件下，方程 $f(x) = 0$ 在 $(a, b)$ 内有唯一的实根 $\xi$，$[a, b]$ 是根的一个隔离区间。此时，$y = f(x)$ 在 $[a, b]$ 上的图形只有四种不同情形（图3-38）。

【此处原有图表，内容描述：图3-38 四种情形下f(x)图形的形状：(a)f(a)<0,f(b)>0,f''>0；(b)f(a)>0,f(b)<0,f''>0；(c)f(a)<0,f(b)>0,f''<0；(d)f(a)>0,f(b)<0,f''<0】

切线法的核心思路是：用曲线弧一端的切线来代替曲线弧，从而求出方程实根的近似值。具体做法是——在纵坐标与 $f''(x)$ 同号的那个端点处作切线，该切线与 $x$ 轴交点的横坐标会比原来的端点更接近方程的根。

以图3-38(a)（$f(a) < 0$，$f(b) > 0$，$f''(x) > 0$）的情形为例：此时因为 $f(b)$ 与 $f''(x)$ 同号，所以令 $x_0 = b$，在点 $(x_0, f(x_0))$ 处作切线（图3-39）。切线的方程为
$$y - f(x_0) = f'(x_0)(x - x_0)$$
令 $y = 0$，解出 $x$，就得到切线与 $x$ 轴交点的横坐标：
$$x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}$$

$x_1$ 比 $x_0$ 更接近方程的根 $\xi$。继续在点 $(x_1, f(x_1))$ 处作切线，可得根的近似值 $x_2$。如此继续，一般地，在点 $(x_n, f(x_n))$ 处作切线，得根的近似值：
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

这就是切线法的迭代公式，也叫牛顿迭代公式。

例2：用切线法求方程 $x^3 + 1.1x^2 + 0.9x - 1.4 = 0$ 的实根的近似值，使误差不超过 $10^{-3}$。

解：由例1知 $[0, 1]$ 是根的一个隔离区间。$f(0) < 0$，$f(1) > 0$。在 $[0, 1]$ 上，
$$f'(x) = 3x^2 + 2.2x + 0.9 > 0, \quad f''(x) = 6x + 2.2 > 0$$
这属于图3-38中情形(a)。因为 $f''(x)$ 与 $f(1)$ 同号，所以取 $x_0 = 1$。

连续应用迭代公式（计算时取三位小数）：

$x_1 = 1 - \frac{f(1)}{f'(1)} = 1 - \frac{1.6}{6.1} \approx 0.738$
$x_2 = 0.738 - \frac{f(0.738)}{f'(0.738)} \approx 0.674$
$x_3 = 0.674 - \frac{f(0.674)}{f'(0.674)} \approx 0.671$
$x_4 = 0.671 - \frac{f(0.671)}{f'(0.671)} \approx 0.671$

至此计算不能再继续（迭代结果不再改变）。注意到 $f(0.671) > 0$，而 $f(0.670) < 0$，于是有 $0.670 < \xi < 0.671$。以 $0.670$ 或 $0.671$ 作为根的近似值，误差都小于 $10^{-3}$。

可以看出，切线法比二分法收敛快得多——二分法需要10步才能达到的精度，切线法只需3-4步！但切线法需要对函数求导，且要求 $f'(x)$ 在隔离区间上不为零。

【三、割线法（弦截法）】

利用切线法需要计算函数的导数，当 $f(x)$ 比较复杂时计算 $f'(x)$ 可能有困难。这时，可以考虑用差商来近似代替导数：
$$\frac{f(x_n) - f(x_{n-1})}{x_n - x_{n-1}} \approx f'(x_n)$$

用差商替换切线法迭代公式中的 $f'(x_n)$，得到割线法的迭代公式：
$$x_{n+1} = x_n - \frac{x_n - x_{n-1}}{f(x_n) - f(x_{n-1})} \cdot f(x_n)$$
其中 $x_0$、$x_1$ 是给定的两个初始值。

【此处原有图表，内容描述：图3-40 用过两点(x0,f(x0))和(x1,f(x1))的割线来代替切线逼近根ξ】

迭代公式的几何意义是：用过点 $(x_{n-1}, f(x_{n-1}))$ 和点 $(x_n, f(x_n))$ 的割线来近似代替点 $(x_n, f(x_n))$ 处的切线，取该割线与 $x$ 轴交点的横坐标作为新的近似值（见图3-40）。因此，这个方法叫做割线法或弦截法。

例3：用割线法求前面例2中方程 $x^3 + 1.1x^2 + 0.9x - 1.4 = 0$ 的近似解。

解：取 $x_0 = 1$，$x_1 = 0.8$。连续应用迭代公式（计算时取三位小数）：

$x_2 = 0.8 - \frac{0.8 - 1}{f(0.8) - f(1)} \cdot f(0.8) \approx 0.699$
$x_3 = 0.699 - \frac{0.699 - 0.8}{f(0.699) - f(0.8)} \cdot f(0.699) \approx 0.672$
$x_4 = 0.672 - \frac{0.672 - 0.699}{f(0.672) - f(0.699)} \cdot f(0.672) \approx 0.671$
$x_5 = 0.671 - \frac{0.671 - 0.672}{f(0.671) - f(0.672)} \cdot f(0.671) \approx 0.671$

至此计算不能再继续。因 $x_4$ 与 $x_5$ 小数前三位数字相同，故以 $0.671$ 作为根的近似值，其误差小于 $10^{-3}$。

割线法的优点是不需要计算导数，收敛速度介于二分法和切线法之间。缺点是每次迭代需要用到前两次的结果，且初始值的选择对收敛有影响。

【三种方法小结】
- 二分法：简单可靠，必定收敛，但速度慢。适合对函数性质了解不多的情况。
- 切线法（牛顿法）：收敛速度快（平方收敛），但需要计算导数，且要求初始值足够好。
- 割线法（弦截法）：不需要导数，收敛速度介于两者之间，超线性收敛。
