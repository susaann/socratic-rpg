4.2 换元积分法

利用基本积分表与积分的性质，所能计算的不定积分是非常有限的。因此，有必要进一步研究不定积分的求法。本节把复合函数的求导法则反过来用于求不定积分——利用中间变量的代换，得到了一种强大的积分方法，称为换元积分法，简称换元法。换元法通常分成两类，我们先讲第一类。

【一、第一类换元法（凑微分法）】

设 $f(u)$ 具有原函数 $F(u)$，即 $F'(u) = f(u)$，$\int f(u)\,du = F(u) + C$。如果 $u = \varphi(x)$ 是中间变量，且 $\varphi(x)$ 可微，那么根据复合函数求导法则，有
$$dF[\varphi(x)] = f[\varphi(x)] \cdot \varphi'(x)\,dx$$
两边积分就得到换元公式：

定理1：设 $f(u)$ 具有原函数，$u = \varphi(x)$ 可导，则有
$$\int f[\varphi(x)]\varphi'(x)\,dx = \left[\int f(u)\,du\right]_{u=\varphi(x)}$$

这个公式怎么用呢？假设要计算 $\int g(x)\,dx$，如果能凑出 $g(x) = f[\varphi(x)]\varphi'(x)$ 的形式，那么就可以实行变量代换 $u = \varphi(x)$，把原积分化为关于 $u$ 的积分，求出结果后再把 $u$ 换回 $\varphi(x)$。

这个方法也叫"凑微分法"——关键就是想办法在被积函数中凑出一个 $\varphi'(x)\,dx = du$。

例1：$\int 2\cos 2x\,dx$。
这个积分里，$\cos 2x$ 是由 $\cos u$ 与 $u = 2x$ 复合而成，常数2恰好是 $u'$。作变换 $u = 2x$，便有
$$\int 2\cos 2x\,dx = \int \cos 2x \cdot 2\,dx = \int \cos u\,du = \sin u + C$$
再换回去：$\int 2\cos 2x\,dx = \sin 2x + C$。

例2：$\int \frac{1}{3+2x}\,dx$。
被积函数是 $\frac{1}{3+2x}$，如果能凑出 $(3+2x)' = 2$ 这个因子就好了——缺这样一个常数因子不要紧，系数可以调整：
$$\int \frac{1}{3+2x}\,dx = \frac{1}{2}\int \frac{1}{3+2x} \cdot 2\,dx = \frac{1}{2}\int \frac{1}{u}\,du = \frac{1}{2}\ln|u| + C = \frac{1}{2}\ln|3+2x| + C$$

一般地，对于 $\int f(ax+b)\,dx$（$a \neq 0$），总可作变换 $u = ax+b$ 来处理。

例3：$\int \frac{x}{(x+2)^4}\,dx$。
令 $u = x+2$，则 $x = u-2$，$dx = du$。于是
$$\int \frac{x}{(x+2)^4}\,dx = \int \frac{u-2}{u^4}\,du = \int (u^{-3} - 2u^{-4})\,du = -\frac{1}{2u^2} + \frac{2}{3u^3} + C$$
$$= \frac{2}{3(x+2)^3} - \frac{1}{2(x+2)^2} + C$$

例4：$\int 2xe^{x^2}\,dx$。
被积函数中的因子 $e^{x^2} = e^u$，而 $2x$ 恰好是 $u = x^2$ 的导数。所以
$$\int 2xe^{x^2}\,dx = \int e^{x^2} \cdot 2x\,dx = \int e^u\,du = e^u + C = e^{x^2} + C$$

例5：$\int x\sqrt{1-x^2}\,dx$。
令 $u = 1-x^2$，则 $du = -2x\,dx$，即 $-\frac{1}{2}du = x\,dx$。因此
$$\int x\sqrt{1-x^2}\,dx = \int \sqrt{u} \cdot (-\frac{1}{2}du) = -\frac{1}{2} \cdot \frac{2}{3}u^{\frac{3}{2}} + C = -\frac{1}{3}(1-x^2)^{\frac{3}{2}} + C$$

在对变量代换比较熟练以后，就不一定写出中间变量 $u$ 了。

例6：$\int \frac{1}{a^2+x^2}\,dx$（$a > 0$）。
$$\int \frac{1}{a^2+x^2}\,dx = \frac{1}{a^2}\int \frac{1}{1+(\frac{x}{a})^2}\,dx = \frac{1}{a}\int \frac{1}{1+(\frac{x}{a})^2}\,d(\frac{x}{a}) = \frac{1}{a}\arctan\frac{x}{a} + C$$

例7：$\int \frac{1}{\sqrt{a^2-x^2}}\,dx$（$a > 0$）。
$$\int \frac{1}{\sqrt{a^2-x^2}}\,dx = \int \frac{1}{a\sqrt{1-(\frac{x}{a})^2}}\,dx = \int \frac{1}{\sqrt{1-(\frac{x}{a})^2}}\,d(\frac{x}{a}) = \arcsin\frac{x}{a} + C$$

例8：$\int \frac{1}{x^2-a^2}\,dx$（$a \neq 0$）。
利用拆项技巧：$\frac{1}{x^2-a^2} = \frac{1}{2a}(\frac{1}{x-a} - \frac{1}{x+a})$。于是
$$\int \frac{1}{x^2-a^2}\,dx = \frac{1}{2a}\left(\int \frac{1}{x-a}\,dx - \int \frac{1}{x+a}\,dx\right) = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C$$

例9：$\int \frac{1}{x(1+2\ln x)}\,dx$。
注意到 $\frac{1}{x}\,dx = d(\ln x)$，所以
$$\int \frac{1}{x(1+2\ln x)}\,dx = \int \frac{1}{1+2\ln x}\,d(\ln x) = \frac{1}{2}\int \frac{1}{1+2\ln x}\,d(1+2\ln x) = \frac{1}{2}\ln|1+2\ln x| + C$$

例10：$\int \frac{e^{\frac{1}{x}}}{x^2}\,dx$。
由于 $\frac{1}{x^2}\,dx = -d(\frac{1}{x})$，因此
$$\int \frac{e^{\frac{1}{x}}}{x^2}\,dx = -\int e^{\frac{1}{x}}\,d(\frac{1}{x}) = -e^{\frac{1}{x}} + C$$

【三角函数积分的凑微分法】

处理含三角函数的积分时，往往要用到三角恒等式。

例11：$\int \sin^3 x\,dx$。
$$\int \sin^3 x\,dx = \int \sin^2 x \cdot \sin x\,dx = -\int (1-\cos^2 x)\,d(\cos x) = -\cos x + \frac{1}{3}\cos^3 x + C$$

例12：$\int \sin^2 x \cos^5 x\,dx$。
$$\int \sin^2 x \cos^5 x\,dx = \int \sin^2 x \cos^4 x \cdot \cos x\,dx = \int \sin^2 x (1-\sin^2 x)^2\,d(\sin x)$$
$$= \int (\sin^2 x - 2\sin^4 x + \sin^6 x)\,d(\sin x) = \frac{1}{3}\sin^3 x - \frac{2}{5}\sin^5 x + \frac{1}{7}\sin^7 x + C$$

一般地，对于 $\sin^{2k+1}x \cos^n x$ 或 $\sin^m x \cos^{2k+1}x$ 型的积分（$k \in \mathbb{N}$），总可凑 $u = \cos x$ 或 $u = \sin x$ 来求得结果。

例13：$\int \tan x\,dx = \int \frac{\sin x}{\cos x}\,dx = -\int \frac{1}{\cos x}\,d(\cos x) = -\ln|\cos x| + C$。
类似地，$\int \cot x\,dx = \ln|\sin x| + C$。

例14：$\int \cos^2 x\,dx = \int \frac{1+\cos 2x}{2}\,dx = \frac{1}{2}(x + \frac{1}{2}\sin 2x) + C = \frac{x}{2} + \frac{\sin 2x}{4} + C$。

例15：$\int \sin^2 x \cos^2 x\,dx$。
利用倍角公式化为 $\cos 2x$ 的多项式：
$$\int \sin^2 x \cos^2 x\,dx = \frac{1}{8}\int (1-\cos^2 2x)\,dx = \frac{1}{8}x - \frac{1}{32}\sin 4x + C$$

一般地，对于 $\sin^{2k}x \cos^{2n}x$（$k, n \in \mathbb{N}$）型的积分，可利用 $\sin^2 x = \frac{1-\cos 2x}{2}$、$\cos^2 x = \frac{1+\cos 2x}{2}$ 化为 $\cos 2x$ 的多项式，然后再求积。

例16：$\int \sec^4 x\,dx = \int (1+\tan^2 x)^2\,d(\tan x) = \tan x + \frac{2}{3}\tan^3 x + \frac{1}{5}\tan^5 x + C$。

【二、第二类换元法】

第一类换元法是通过凑微分来实现的。但有时候，被积函数不容易凑，反而直接作一个变量代换更方便——这就是第二类换元法。

定理2：设 $x = \psi(t)$ 是单调的、可导的函数，并且 $\psi'(t) \neq 0$。又设 $f[\psi(t)]\psi'(t)$ 具有原函数，则有换元公式
$$\int f(x)\,dx = \left[\int f[\psi(t)]\psi'(t)\,dt\right]_{t=\psi^{-1}(x)}$$

常用的第二类换元法有三种：

1. 三角代换（去根号 $\sqrt{a^2-x^2}$、$\sqrt{a^2+x^2}$、$\sqrt{x^2-a^2}$）

当被积函数含有 $\sqrt{a^2-x^2}$ 时，令 $x = a\sin t$（或 $x = a\cos t$）；
含有 $\sqrt{a^2+x^2}$ 时，令 $x = a\tan t$；
含有 $\sqrt{x^2-a^2}$ 时，令 $x = a\sec t$。

这样做的原理是利用三角恒等式消除根号。

2. 根式代换

当被积函数中含有 $\sqrt[n]{ax+b}$ 时，令 $t = \sqrt[n]{ax+b}$，则 $x = \frac{t^n-b}{a}$，$dx = \frac{n t^{n-1}}{a}\,dt$，根号就被消除了。

3. 倒代换

当分母的次数比分子高很多时，有时令 $x = \frac{1}{t}$（即倒代换）可以简化积分。

第二类换元法的关键在于：做完代换后要把 $t$ 换回 $x$（利用反函数关系）。

例（含有 $\sqrt{a^2-x^2}$）：$\int \sqrt{a^2-x^2}\,dx$。
令 $x = a\sin t$，$t \in [-\frac{\pi}{2}, \frac{\pi}{2}]$，则 $\sqrt{a^2-x^2} = a\cos t$，$dx = a\cos t\,dt$。于是
$$\int \sqrt{a^2-x^2}\,dx = \int a\cos t \cdot a\cos t\,dt = a^2\int \cos^2 t\,dt$$
$$= a^2\int \frac{1+\cos 2t}{2}\,dt = \frac{a^2}{2}(t + \frac{1}{2}\sin 2t) + C$$
$$= \frac{a^2}{2}(t + \sin t\cos t) + C = \frac{a^2}{2}\arcsin\frac{x}{a} + \frac{x}{2}\sqrt{a^2-x^2} + C$$

这个积分的结果非常重要，在很多后续内容中都会用到。

例（含有 $\sqrt{a^2+x^2}$）：$\int \frac{1}{\sqrt{a^2+x^2}}\,dx$。
令 $x = a\tan t$，$t \in (-\frac{\pi}{2}, \frac{\pi}{2})$，则 $\sqrt{a^2+x^2} = a\sec t$，$dx = a\sec^2 t\,dt$。于是
$$\int \frac{1}{\sqrt{a^2+x^2}}\,dx = \int \frac{a\sec^2 t}{a\sec t}\,dt = \int \sec t\,dt = \ln|\sec t + \tan t| + C$$
再换回 $x$：$\sec t = \frac{\sqrt{a^2+x^2}}{a}$，$\tan t = \frac{x}{a}$，所以
$$\int \frac{1}{\sqrt{a^2+x^2}}\,dx = \ln\left|\frac{\sqrt{a^2+x^2}}{a} + \frac{x}{a}\right| + C = \ln\left|\sqrt{a^2+x^2} + x\right| + C'$$

这个结果也可以记作 $\int \frac{1}{\sqrt{a^2+x^2}}\,dx = \ln(x + \sqrt{a^2+x^2}) + C$。

换元积分法是求不定积分最核心、最常用的方法。核心思想就一句话：通过变量代换，把不会积的式子变成会积的式子！
