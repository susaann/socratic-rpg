7.5 可降阶的高阶微分方程

从这一节起讨论二阶及二阶以上的高阶微分方程。对有些高阶方程，可以通过代换将它化成较低阶的方程来求解。下面介绍三种容易降阶的类型。

【类型一：$y^{(n)} = f(x)$ 型】

右端仅含自变量 $x$。只需接连积分 $n$ 次，每次积分出现一个任意常数，最终得到含有 $n$ 个任意常数的通解。

例1：解 $y''' = e^{2x} - \cos x$。
解：积分三次：
$$y'' = \frac{1}{2}e^{2x} - \sin x + C_1$$
$$y' = \frac{1}{4}e^{2x} + \cos x + C_1x + C_2$$
$$y = \frac{1}{8}e^{2x} + \sin x + \frac{C_1}{2}x^2 + C_2x + C_3$$

【类型二：$y'' = f(x, y')$ 型——不显含 $y$】

方程右端不显含未知函数 $y$。令 $y' = p$，则 $y'' = p'$，方程降为一阶方程 $p' = f(x, p)$。解出 $p = \varphi(x, C_1)$ 后，再积分一次得 $y = \int \varphi(x, C_1)\,dx + C_2$。

例2：解 $(1+x^2)y'' = 2xy'$，满足 $y|_{x=0}=1$，$y'|_{x=0}=3$。

解：令 $y'=p$，代入得 $(1+x^2)p' = 2xp$，分离变量 $\frac{dp}{p} = \frac{2x}{1+x^2}\,dx$，积分得 $\ln|p| = \ln(1+x^2) + \ln C$，即 $p = C(1+x^2)$。由 $y'|_{x=0}=3$ 得 $C=3$，故 $y' = 3(1+x^2)$。再积分得 $y = x^3 + 3x + C_2$。由 $y|_{x=0}=1$ 得 $C_2=1$。特解为 $y = x^3 + 3x + 1$。

【类型三：$y'' = f(y, y')$ 型——不显含 $x$】

方程右端不显含自变量 $x$。令 $y' = p$，并利用 $y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p\frac{dp}{dy}$，将方程化为以 $y$ 为自变量、$p$ 为未知函数的一阶方程 $p\frac{dp}{dy} = f(y, p)$。

【应用：悬链线问题——$y'' = f(x, y')$ 型的重要实例】

设有一条均匀、柔软的绳索，两端固定，仅受重力作用下垂。求绳索在平衡状态时的曲线方程。

解：取最低点 $A$ 为参考点，设水平张力为 $H$，绳索线密度为 $\rho$。由受力分析得微分方程
$$y'' = \frac{1}{a}\sqrt{1+y'^2} \quad (a = \frac{H}{\rho g})$$
这是 $y'' = f(x, y')$ 型。令 $y' = p$，分离变量积分，并结合初值条件，最终解得
$$y = a\,\text{ch}\frac{x}{a} = \frac{a}{2}(e^{\frac{x}{a}} + e^{-\frac{x}{a}})$$
这条曲线叫做悬链线。注意！很多人误以为悬垂的绳索是抛物线，其实它是双曲余弦曲线。悬链线在建筑（拱形设计）、桥梁工程中有重要应用。
