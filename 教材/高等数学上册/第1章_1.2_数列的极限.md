极限这个概念是怎么来的呢？它源于人们探索实际问题精确答案的过程。我国古代数学家刘徽（公元3世纪）发明的"割圆术"，就是用极限思想来算圆的面积。

一、数列极限的定义

【从割圆术说起】

设想有一个圆。先做一个内接正六边形，把它的面积记为 A₁；再做一个内接正十二边形，面积记为 A₂；然后再做内接正二十四边形，面积记为 A₃……这样一直做下去，每次边数翻倍。一般地，把内接正 6×2ⁿ⁻¹ 边形的面积记为 A_n（n = 1, 2, 3, …）。

这样我们就得到一列有次序的数：A₁, A₂, A₃, …, A_n, …。n 越大，内接正多边形就越接近圆，A_n 作为圆面积的近似值也就越精确。但是，不管 n 取多大，只要 n 是确定的值，A_n 终究只是多边形的面积，而不是真正圆的面积。

于是我们设想：如果 n 无限增大（记作 n → ∞，读作"n 趋于无穷"），也就是内接正多边形的边数无限增多，在这个"无限接近"的过程中，A_n 会无限接近某一个确定的数值——这个数值就理解为圆的面积。这个确定的数值，数学上就叫做数列 A₁, A₂, A₃, … 当 n → ∞ 时的极限。

极限方法是高等数学中最基本的方法。接下来我们把它说清楚。

【数列是什么】

如果按某种规则，对每个正整数 n，都对应一个确定的实数 x_n，这些实数按照下标 n 从小到大排列成 x₁, x₂, x₃, …, x_n, …，就叫做数列，简记为 { x_n }。数列中每个数叫项，第 n 项 x_n 叫通项（一般项）。

例如：
- 1/2, 2/3, 3/4, …, n/(n+1), …
- 2, 4, 8, …, 2^n, …
- 1, -1, 1, …, (-1)^(n+1), …

在几何上，数列 { x_n } 可以看作数轴上的一个动点，依次经过 x₁, x₂, x₃, … 各个位置。数列也可以看作自变量取正整数的函数：x_n = f(n), n ∈ N₊。

【严格定义数列极限】

我们来分析数列 x_n = (n + (-1)^(n-1)) / n（即 2, 1/2, 4/3, 3/4, …）。可以观察到：当 n 越来越大时，x_n 似乎在不断靠近 1。怎么精确描述这种"靠近"呢？

两个数 a 与 b 的接近程度，可以用它们差的绝对值 |b - a| 来衡量——绝对值越小，越接近。

对于这个数列：|x_n - 1| = 1/n。所以，如果想让 x_n 与 1 的差距小于 1/100，只要取 n > 100 就行了。想让差距小于 1/10000？那就取 n > 10000。一般来说，不管你想要多小的差距（记作 ε，读作 epsilon），总存在一个正整数 N，使得当 n > N 时，不等式 |x_n - 1| < ε 恒成立。这就是"x_n 当 n → ∞ 时无限接近于 1"的真正含义。

正式定义：设 { x_n } 为一数列，如果存在常数 a，对于任意给定的正数 ε（不论它多么小），总存在正整数 N，使得当 n > N 时，不等式 |x_n - a| < ε 都成立，那么就称常数 a 是数列 { x_n } 的极限（或者称数列收敛于 a），记作 lim_{n→∞} x_n = a 或 x_n → a (n → ∞)。如果不存在这样的常数 a，就说数列没有极限，或称数列发散。

用符号简写为：lim_{n→∞} x_n = a ⇔ ∀ε > 0, ∃正整数 N, 当 n > N 时，有 |x_n - a| < ε。

【几何解释】

把常数 a 和各 x_n 在数轴上标出来，然后在数轴上做 a 的 ε 邻域（即开区间 (a-ε, a+ε)）。因为 |x_n - a| < ε 等价于 a-ε < x_n < a+ε，所以当 n > N 时，所有的点 x_n 都落在 (a-ε, a+ε) 内，只有有限个（至多 N 个）落在外边。

换句形象的话：不管你把"瞄准圈"画得多小，数列中足够靠后的项全都会落进去。

【几个证明极限的例子】

例1：证明数列 x_n = (n + (-1)^(n-1)) / n 的极限是 1。

证明：|x_n - 1| = |((-1)^(n-1))/n| = 1/n。∀ε > 0，要使 |x_n - 1| < ε，只需 1/n < ε，即 n > 1/ε。所以取 N = [1/ε]（取整），当 n > N 时就有 |x_n - 1| < ε。证毕。

例2：设 x_n = (-1)^n / (n+1)²，证明极限是 0。

证明：|x_n - 0| = 1/(n+1)² ≤ 1/n。∀ε > 0，要使 |x_n - 0| < ε，只需 1/n < ε，即 n > 1/ε。取 N > 1/ε 的正整数即可。

例3：设 |q| < 1，证明等比数列 q^(n-1) 的极限是 0。

证明：|q^(n-1) - 0| = |q|^(n-1)。∀ε > 0（设 ε < 1），要使 |q|^(n-1) < ε，两边取自然对数得 (n-1)ln|q| < ln ε。因为 |q| < 1，所以 ln|q| < 0，因此 n > 1 + ln ε / ln|q|。取 N 为满足条件的正整数，则当 n > N 时结论成立。即 lim_{n→∞} q^(n-1) = 0。

二、收敛数列的性质

【定理1：极限的唯一性】如果数列 { x_n } 收敛，那么它的极限唯一。

证明思路（反证法）：假设 x_n → a 且 x_n → b，a ≠ b。取 ε = |b - a|/2 > 0。根据极限定义，存在 N₁ 使 n > N₁ 时 |x_n - a| < ε，存在 N₂ 使 n > N₂ 时 |x_n - b| < ε。取 N = max{N₁, N₂}，则 n > N 时两个不等式同时成立。但这意味着 a - ε < x_n < a + ε 且 b - ε < x_n < b + ε，而这两个区间不重叠（因为 ε = |b-a|/2），矛盾。所以 a = b。

这也告诉我们：如果一个数列有两个子列趋于不同的值，那它一定发散。

例4：证明数列 x_n = (-1)^(n+1)（即 1, -1, 1, -1, …）是发散的。

证：如果收敛，极限唯一。但取 ε = 1/2，因为数列只有 1 和 -1 两个值，长度为 1 的开区间不可能同时包含 1 和 -1，所以不可能收敛。

【有界性概念】

对于数列 { x_n }，如果存在正数 M，使得对一切 n 都满足 |x_n| ≤ M，则称数列有界。否则无界。例如，x_n = 1/n 有界（|1/n| ≤ 1），x_n = 2^n 无界。

【定理2：收敛数列必定有界】如果数列收敛，则它一定有界。

证明：设 lim x_n = a。取 ε = 1，存在 N，当 n > N 时 |x_n - a| < 1，从而 |x_n| ≤ |x_n - a| + |a| < 1 + |a|。令 M = max{ |x₁|, |x₂|, …, |x_N|, 1+|a| }，则对所有 n 都有 |x_n| ≤ M。

注意：反过来不成立——有界的数列不一定收敛。例如 1, -1, 1, -1, … 有界(|x_n| ≤ 1)但发散。所以"有界"是收敛的必要条件，不是充分条件。

【定理3：收敛数列的保号性】如果 lim x_n = a，且 a > 0（或 a < 0），那么存在 N，当 n > N 时，x_n > 0（或 x_n < 0）。

直观理解：极限是正的，那么足够靠后的项也必然是正的。就像一条路最终通往正数方向，走到后面自然在正数区域。

证明（a > 0的情形）：取 ε = a/2 > 0，存在 N，当 n > N 时 |x_n - a| < a/2，从而 x_n > a - a/2 = a/2 > 0。

推论：如果数列从某项起 x_n ≥ 0（或 x_n ≤ 0），且 lim x_n = a，则 a ≥ 0（或 a ≤ 0）。

【子数列与定理4】

在数列中任意抽取无限多项，并保持原来先后顺序，这样得到的新数列叫子数列（子列）。

例如，从 { x_n } 中取第 n₁ 项、第 n₂ 项、第 n₃ 项……得到 x_{n₁}, x_{n₂}, x_{n₃}, …，其中 n₁ < n₂ < n₃ < …。注意：x_{n_k} 是子列中的第 k 项，但在原数列中是第 n_k 项，且 n_k ≥ k。

【定理4：收敛数列的子列也收敛】如果数列 { x_n } 收敛于 a，那么它的任一子列也收敛于 a。

这个定理的妙用在于：如果某个数列有两个子列收敛于不同的极限，那么这个数列必定发散。比如 x_n = (-1)^(n+1)，其偶数项子列收敛于 -1，奇数项子列收敛于 1，因此必然发散。这也说明：发散数列也可能有收敛的子列。
