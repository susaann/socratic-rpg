4.3 分部积分法

前面我们在复合函数求导法则的基础上得到了换元积分法。现在我们利用两个函数乘积的求导法则，来推出另一个求积分的基本方法——分部积分法。如果说换元法是"反用链式法则"，那么分部积分法就是"反用乘法法则"。

设函数 $u = u(x)$ 及 $v = v(x)$ 具有连续导数，则两个函数乘积的导数公式为
$$(uv)' = u'v + uv'$$
移项得 $uv' = (uv)' - u'v$。两边求不定积分，得
$$\int uv'\,dx = uv - \int u'v\,dx$$

这就是分部积分公式。为了书写方便，也常写成：
$$\int u\,dv = uv - \int v\,du$$

这个公式的灵魂在于：如果 $\int v\,du$ 比 $\int u\,dv$ 容易积分，那我们就"赚"了！

【怎样选取 $u$ 和 $dv$？】

选取 $u$ 和 $dv$ 一般要考虑下面两点：
(1) $v$ 要容易由 $dv$ 求得（也就是 $dv$ 要好积）；
(2) $\int v\,du$ 要比 $\int u\,dv$ 容易积出。

如果选择不当，不仅不能简化计算，反而可能越积越复杂。

例1：求 $\int x\cos x\,dx$。

设 $u = x$，$dv = \cos x\,dx = d(\sin x)$，则 $du = dx$，$v = \sin x$。于是
$$\int x\cos x\,dx = x\sin x - \int \sin x\,dx = x\sin x + \cos x + C$$

如果反过来设 $u = \cos x$，$dv = x\,dx$，则 $du = -\sin x\,dx$，$v = \frac{x^2}{2}$，那么
$$\int x\cos x\,dx = \frac{x^2}{2}\cos x + \int \frac{x^2}{2}\sin x\,dx$$
右边的积分比原来的积分还复杂！所以选取不对，等于自找麻烦。

例2：$\int xe^x\,dx$。
设 $u = x$，$dv = e^x\,dx = d(e^x)$，则
$$\int xe^x\,dx = \int x\,d(e^x) = xe^x - \int e^x\,dx = xe^x - e^x + C = e^x(x-1) + C$$

例3：$\int x^2 e^x\,dx$。
设 $u = x^2$，$dv = e^x\,dx = d(e^x)$，则
$$\int x^2 e^x\,dx = x^2 e^x - \int e^x\,d(x^2) = x^2 e^x - 2\int x e^x\,dx$$
这里 $\int xe^x\,dx$ 比 $\int x^2 e^x\,dx$ 降低了一次幂，再用一次分部积分：
$$\int x^2 e^x\,dx = x^2 e^x - 2\left[xe^x - e^x\right] + C = e^x(x^2 - 2x + 2) + C$$

例3向我们展示了一个很有用的规律：当被积函数是 $x^n$ 乘指数函数（或三角函数）时，每次分部积分使 $x^n$ 的幂降一次，用 $n$ 次分部积分即可求得结果。这种方法叫"降幂法"。

【"去对数、去反三角"——另一种选择策略】

例4：$\int \ln x\,dx$。
这个积分看似简单，但基本积分表里没有。分部积分法恰好能解决它——令 $u = \ln x$，$dv = dx$，则
$$\int \ln x\,dx = x\ln x - \int x\,d(\ln x) = x\ln x - \int x \cdot \frac{1}{x}\,dx = x\ln x - x + C$$

例5：$\int \arccos x\,dx$。
令 $u = \arccos x$，$dv = dx$，则
$$\int \arccos x\,dx = x\arccos x - \int x\,d(\arccos x) = x\arccos x + \int \frac{x}{\sqrt{1-x^2}}\,dx$$
$$= x\arccos x - \sqrt{1-x^2} + C$$

例6：$\int \arctan x\,dx$。
$$\int \arctan x\,dx = x\arctan x - \int x\,d(\arctan x) = x\arctan x - \int \frac{x}{1+x^2}\,dx$$
$$= x\arctan x - \frac{1}{2}\ln(1+x^2) + C$$

从例4到例6可以看出一个规律：当被积函数含有对数函数或反三角函数时（这些函数求导后变成有理式或代数式），就把它们选作 $u$，求导后积分反而变简单。

【总结选取 $u$ 的策略】

有一个口诀可以参考——"反、对、幂、三、指"，越靠前的越优先选作 $u$：
- 反（反三角函数）
- 对（对数函数）
- 幂（幂函数 $x^n$）
- 三（三角函数）
- 指（指数函数 $e^x$、$a^x$）

按照这个顺序，排在后面的选作 $dv$。

【"转圈"型分部积分——循环解法】

例7：$\int e^x \sin x\,dx$。
这类积分靠一次分部积分不行——但奇妙的是，连续用两次分部积分会转回来！

第一次：$\int e^x \sin x\,dx = \int \sin x\,d(e^x) = e^x\sin x - \int e^x\cos x\,dx$
第二次：对 $\int e^x\cos x\,dx$ 再用分部积分：
$$\int e^x\cos x\,dx = \int \cos x\,d(e^x) = e^x\cos x + \int e^x\sin x\,dx$$

代入第一次的结果：
$$\int e^x\sin x\,dx = e^x\sin x - e^x\cos x - \int e^x\sin x\,dx$$

把右端的 $\int e^x\sin x\,dx$ 移到左边去：
$$2\int e^x\sin x\,dx = e^x(\sin x - \cos x) + C_1$$
$$\int e^x\sin x\,dx = \frac{1}{2}e^x(\sin x - \cos x) + C$$

这种"转圈"解法非常优雅！方程两端出现了同一个积分，移项解之即可。

例8：$\int \sec^3 x\,dx$。
$$\int \sec^3 x\,dx = \int \sec x\,d(\tan x) = \sec x\tan x - \int \sec x\tan^2 x\,dx$$
$$= \sec x\tan x - \int \sec x(\sec^2 x - 1)\,dx$$
$$= \sec x\tan x - \int \sec^3 x\,dx + \int \sec x\,dx$$

移项得：$2\int \sec^3 x\,dx = \sec x\tan x + \ln|\sec x + \tan x| + C_1$，所以
$$\int \sec^3 x\,dx = \frac{1}{2}(\sec x\tan x + \ln|\sec x + \tan x|) + C$$

在积分过程中往往需要兼用换元法与分部积分法。例如求 $\int e^{\sqrt{x}}\,dx$，先令 $\sqrt{x} = t$ 换元变成 $2\int te^t\,dt$，再用分部积分即可。

分部积分法与换元积分法是求不定积分的两大基本方法——它们构成了不定积分计算的核心技术。
