前几节学了极限的定义，但怎么求极限呢？这一节就来建立一套"运算规则"，让我们可以像做加减乘除一样求极限。主要讲两部分：极限的四则运算法则和复合函数的极限运算法则。

注意：下面所有的定理对 x → x₀ 和 x → ∞ 都成立。我们只以 x → x₀ 为代表来证明。

一、极限的四则运算法则

先看无穷小的一些性质，它们是整个运算法则的基础。

定理1：两个无穷小的和还是无穷小。

定理2：有界函数与无穷小的乘积是无穷小。

（简单理解：一个"趋于0"的东西乘以一个"不超界"的东西，结果还是"趋于0"。）

由定理2可以推出两个有用的结论：
- 推论1：常数乘以无穷小仍是无穷小。
- 推论2：有限个无穷小的乘积仍是无穷小。

现在来看最重要的极限四则运算法则。

定理3：如果 lim f(x) = A，lim g(x) = B，那么：
- (1) lim[f(x) ± g(x)] = lim f(x) ± lim g(x) = A ± B
- (2) lim[f(x) · g(x)] = lim f(x) · lim g(x) = A · B
- (3) 若 B ≠ 0，则 lim[f(x)/g(x)] = lim f(x) / lim g(x) = A / B

这个定理的意思是：如果两个函数各自有极限，那么它们的和、差、积、商的极限，就等于各自极限的和、差、积、商（商要求分母极限不为零）。一句话：极限符号可以"穿进"四则运算中。

证明思路：利用"极限 = 常数 + 无穷小"这个关系。由 lim f(x) = A 可得 f(x) = A + α（α是无穷小）；同理 g(x) = B + β。那么 f(x) + g(x) = (A+B) + (α+β)，而后面的 α+β 是无穷小（定理1），所以 f+g 的极限就是 A+B。其他运算同理可证。

【推论】
- 常数可以提出极限符号外：lim[c·f(x)] = c·lim f(x)（因为 lim c = c）
- 幂的极限等于极限的幂：lim[f(x)]^n = [lim f(x)]^n（n为正整数）

关于数列，也有完全类似的四则运算法则（定理4，略）。

定理5：如果 φ(x) ≤ ψ(x)，且 lim φ(x) = A，lim ψ(x) = B，则 A ≤ B。这叫"保不等式性"。

【直接代入法】

作为一个非常重要的应用：对于多项式 f(x) = a₀x^n + a₁x^(n-1) + … + a_n，有 lim_{x→x₀} f(x) = f(x₀)。也就是说，对于多项式，求 x → x₀ 的极限，直接把 x₀ 代入就行！

对于有理分式函数 F(x) = P(x)/Q(x)（P 和 Q 是多项式），如果分母 Q(x₀) ≠ 0，同样有 lim_{x→x₀} F(x) = P(x₀)/Q(x₀)。

【典型例题】

例1：求 lim_{x→1} (2x-1)
解：lim(2x-1) = lim 2x - lim 1 = 2·lim x - 1 = 2·1 - 1 = 1

例2：求 lim_{x→2} (x³-1)/(x²-5x+3)
解：分母代入 x=2 得 4-10+3=-3 ≠ 0，可以直接代入：
原式 = (8-1)/(4-10+3) = 7/(-3) = -7/3

【0/0 型与 ∞/∞ 型的处理技巧】

但有时直接代入会得到"0/0"这样的不定式，需要特殊处理。

例3：求 lim_{x→3} (x-3)/(x²-9)
解：代入 x=3 得 0/0，不能直接用法则。注意分子分母有公因子 (x-3)：
原式 = lim_{x→3} (x-3)/[(x-3)(x+3)] = lim_{x→3} 1/(x+3) = 1/6

例4：求 lim_{x→∞} (3x³+4x²+2)/(7x³+5x²-3)
解：x→∞ 时分子分母都→∞，用 x³ 同时除分子分母：
原式 = lim_{x→∞} (3+4/x+2/x³)/(7+5/x-3/x³) = 3/7（因为 1/x → 0）

结论（一般公式）：当 a₀ ≠ 0, b₀ ≠ 0, m 和 n 为非负整数时：
lim_{x→∞} (a₀x^n + a₁x^(n-1) + …)/(b₀x^m + b₁x^(m-1) + …) = { 0, 当 n < m; a₀/b₀, 当 n = m; ∞, 当 n > m }

简单记：看分子分母最高次——分子次数低→0，相等→系数比，分子次数高→∞。

例5：求 lim_{x→0} (sin x)/x
解：当 x→0 时，1/x → ∞，但 sin x 是有界函数(|sin x| ≤ 1)。根据定理2（有界函数×无穷小=无穷小），原式 = 0。

二、复合函数的极限运算法则

定理6（换元法）：设 y = f[g(x)] 由 u = g(x) 和 y = f(u) 复合而成，在 x₀ 的某去心邻域内有定义。如果 lim_{x→x₀} g(x) = u₀，lim_{u→u₀} f(u) = A，且在 x₀ 的某去心邻域内 g(x) ≠ u₀，则 lim_{x→x₀} f[g(x)] = lim_{u→u₀} f(u) = A。

这个定理的用处在于：当我们遇到一个复合函数求极限时，可以"换元"——令 u = g(x)，算出 u 趋于什么值，再去求 f(u) 的极限。就像做代换一样方便。

三、总结

这一节的核心就是一句话：当各部分的极限都存在（分母极限不为0）时，求极限可以"逐项进行"。如果遇到 0/0、∞/∞ 之类的"不定式"，就需要用约分、同除以最高次项、等价替换等技巧来处理。有了这些工具，大量函数的极限就可以计算了。
