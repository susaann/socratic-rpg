有些极限没法直接用四则运算来算，需要借助一些"判定定理"。本节介绍两个判定极限存在的准则，以及由此导出的两个极其重要的极限。

一、准则Ⅰ：夹逼准则

【数列版的夹逼准则】

如果数列 {x_n}、{y_n}、{z_n} 满足：
(1) 从某项起，y_n ≤ x_n ≤ z_n（x_n 被 y_n 和 z_n "夹"在中间）；
(2) lim y_n = a，lim z_n = a，
那么 lim x_n 也存在，并且 lim x_n = a。

这个准则的道理非常直观：两边都往同一个数 a 走，中间那个只能跟着也走到 a——就像三明治的两片面包越靠越近，中间的火腿也只能往同一个地方去。

【函数版的夹逼准则】

如果当 x 在 x₀ 的某去心邻域（或 |x| > M）时，g(x) ≤ f(x) ≤ h(x)，且 lim g(x) = lim h(x) = A，则 lim f(x) 存在且等于 A。

这个定理通常称为"夹逼准则"或"三明治定理"。

二、第一个重要极限：lim_{x→0} sin x / x = 1

利用夹逼准则，我们来证明这个最经典的重要极限。

考虑四分之一单位圆，设圆心角 ∠AOB = x（0 < x < π/2）。作出 BC ⊥ OA，以及 A 处的切线与 OB 的延长线交于 D。则几何上：
- sin x = CB（对边比斜边，半径=1）
- x = 弧 AB 的长度（弧度制下）
- tan x = AD

由面积比较：△AOB 的面积 < 扇形 AOB 的面积 < △AOD 的面积
即：sin x / 2 < x / 2 < tan x / 2

从而 sin x < x < tan x。不等号各边除以 sin x，得到：
1 < x / sin x < 1 / cos x，即 cos x < sin x / x < 1。

因为 lim_{x→0} cos x = 1 且 lim_{x→0} 1 = 1，由夹逼准则，即得：
lim_{x→0} sin x / x = 1

这个极限在整个微积分中都非常重要。直观理解就是：当角度特别小时，sin x 几乎就等于 x。

【应用举例】

例1：求 lim_{x→0} tan x / x
解：tan x / x = (sin x / x) · (1 / cos x)，所以
lim_{x→0} tan x / x = lim_{x→0} (sin x / x) · lim_{x→0} (1 / cos x) = 1 · 1 = 1

例2：求 lim_{x→0} (1 - cos x) / x²
解：1 - cos x = 2 sin²(x/2)，故
原式 = lim_{x→0} [2 sin²(x/2)] / x² = lim_{x→0} [sin²(x/2)] / [2(x/2)²] = 1/2 · 1² = 1/2

例3：求 lim_{x→0} arcsin x / x
解：令 t = arcsin x，则 x = sin t，当 x→0 时 t→0。由复合函数极限法则：
原式 = lim_{t→0} t / sin t = 1

三、准则Ⅱ：单调有界准则

如果数列单调且有界，那么它一定收敛（极限存在）。

具体说：
- 单调增加 + 有上界 → 收敛
- 单调减少 + 有下界 → 收敛

直观理解：一个数列只能往一个方向走（单调），又被一堵墙挡住了（有界），那它只能越来越靠近某个点——这个点就是极限。就像你单向朝一堵墙走去，永远过不去，只能无限接近它。

对于函数也有类似的准则：如果函数 f(x) 在 x₀ 的某个左邻域内单调并且有界，则左极限 f(x₀⁻) 必定存在。

四、第二个重要极限：lim_{x→∞} (1 + 1/x)^x = e

利用准则Ⅱ（单调有界），我们来导出自然常数 e。

考虑数列 x_n = (1 + 1/n)^n。用二项式定理展开：

x_n = 1 + 1 + (1/2!)(1-1/n) + (1/3!)(1-1/n)(1-2/n) + … + (1/n!)(1-1/n)(1-2/n)…(1-(n-1)/n)

类似地展开 x_{n+1} = (1 + 1/(n+1))^(n+1)，比较两项可发现 x_n < x_{n+1}，即数列单调增加。

同时，将展开式中括号内的数都用 1 代替（变大），可证 x_n < 1 + 1 + 1/2! + 1/3! + … + 1/n! < 1 + 1 + 1/2 + 1/4 + … = 3，即数列有上界。

因此根据准则Ⅱ，这个数列的极限存在，我们把这个极限记作 e：
lim_{n→∞} (1 + 1/n)^n = e

可以证明，当 x 取实数趋于 +∞ 或 -∞ 时，函数 (1 + 1/x)^x 的极限都存在且都等于 e。因此：
lim_{x→∞} (1 + 1/x)^x = e

e 是一个无理数，其值约为 2.718281828459045…。这个数在数学中极为重要，前面的指数函数 y = e^x 和自然对数 y = ln x 中的底 e 就是它。

另一种等价形式：令 t = 1/x，则当 x→∞ 时 t→0，有 lim_{t→0} (1 + t)^(1/t) = e。

【应用举例】

例4：求 lim_{x→∞} (1 - 1/x)^x
解：令 t = -x，则当 x→∞ 时 t→-∞。
原式 = lim_{t→-∞} (1 + 1/t)^(-t) = 1 / lim_{t→-∞} (1 + 1/t)^t = 1/e

五、柯西（Cauchy）极限存在准则

前面准则Ⅱ的"单调有界"是收敛的充分条件，但不是必要条件（收敛的数列不一定单调）。柯西准则给出了收敛的充分必要条件。

数列 { x_n } 收敛的充分必要条件是：对于任意给定的 ε > 0，存在正整数 N，使得当 m > N, n > N 时，有 |x_m - x_n| < ε。

几何意义：数列收敛，等价于"序号足够大的项，任意两项之间的距离可以任意小"。也就是说，数列后面的项彼此靠得越来越近。这个准则也叫"柯西审敛原理"，在理论分析中非常重要。
