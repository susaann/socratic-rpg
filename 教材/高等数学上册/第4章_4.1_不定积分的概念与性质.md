4.1 不定积分的概念与性质

【引言——微积分的另一半】

在第二章中，我们讨论的是"已知函数，求其导数"的问题。本章讨论它的反问题：已知一个函数，能不能找到一个可导函数，使得它的导函数恰好等于已知函数？这就是积分学的基本问题之一。

打个比方：如果说求导是"从路程求速度"，那么求不定积分就是"从速度反推路程"。

【一、原函数与不定积分的概念】

定义1（原函数）：如果在区间 $I$ 上，可导函数 $F(x)$ 的导函数为 $f(x)$，即对任一 $x \in I$，都有
$$F'(x) = f(x) \quad \text{或} \quad dF(x) = f(x)\,dx$$
那么函数 $F(x)$ 就称为 $f(x)$（或 $f(x)\,dx$）在区间 $I$ 上的一个原函数。

例如，因为 $(\sin x)' = \cos x$，故 $\sin x$ 是 $\cos x$ 的一个原函数。
又如当 $x \in (1, +\infty)$ 时，$[\ln(x + \sqrt{x^2-1})]' = \frac{1}{\sqrt{x^2-1}}$，故 $\ln(x + \sqrt{x^2-1})$ 是 $\frac{1}{\sqrt{x^2-1}}$ 在区间 $(1, +\infty)$ 内的一个原函数。

关于原函数的第一个重要问题是：一个函数具备什么条件时，它的原函数一定存在？这里先告诉大家一个结论（详细证明将在下一章给出）：

原函数存在定理：如果函数 $f(x)$ 在区间 $I$ 上连续，那么在区间 $I$ 上存在可导函数 $F(x)$，使得对任一 $x \in I$ 都有 $F'(x) = f(x)$。简单说就是：连续函数一定有原函数！

还有两点需要说明：
第一，如果 $f(x)$ 在区间 $I$ 上有原函数 $F(x)$，那么对任何常数 $C$，显然也有 $[F(x) + C]' = f(x)$。也就是说，如果 $f(x)$ 有一个原函数，那么它就有无限多个原函数（每个原函数之间只差一个常数）。
第二，如果在区间 $I$ 上 $F(x)$ 是 $f(x)$ 的一个原函数，$G(x)$ 是 $f(x)$ 的另一个原函数，那么 $[G(x) - F(x)]' = f(x) - f(x) = 0$。在第三章中我们知道，在一个区间上导数恒为零的函数必为常数，所以 $G(x) - F(x) = C_0$（某个常数）。这表明 $G(x)$ 与 $F(x)$ 只差一个常数。因此，当 $C$ 为任意常数时，$F(x) + C$ 就可以表示 $f(x)$ 的任意一个原函数。

基于以上两点，我们引入不定积分的定义。

定义2（不定积分）：在区间 $I$ 上，函数 $f(x)$ 的带有任意常数项的原函数称为 $f(x)$（或 $f(x)\,dx$）在区间 $I$ 上的不定积分，记作
$$\int f(x)\,dx$$
其中记号 $\int$ 称为积分号，$f(x)$ 称为被积函数，$f(x)\,dx$ 称为被积表达式，$x$ 称为积分变量。

由此定义可知，如果 $F(x)$ 是 $f(x)$ 在区间 $I$ 上的一个原函数，那么 $F(x) + C$ 就是 $f(x)$ 的不定积分，即
$$\int f(x)\,dx = F(x) + C$$
因而不定积分 $\int f(x)\,dx$ 可以表示 $f(x)$ 的任意一个原函数。

例1：求 $\int x^2\,dx$。
解：由于 $(\frac{x^3}{3})' = x^2$，所以 $\frac{x^3}{3}$ 是 $x^2$ 的一个原函数。因此 $\int x^2\,dx = \frac{x^3}{3} + C$。

例2：求 $\int \frac{1}{x}\,dx$。
解：当 $x > 0$ 时，$(\ln x)' = \frac{1}{x}$，所以 $\ln x$ 是 $\frac{1}{x}$ 在 $(0, +\infty)$ 内的一个原函数，$\int \frac{1}{x}\,dx = \ln x + C$。
当 $x < 0$ 时，$[\ln(-x)]' = \frac{1}{x}$，所以 $\ln(-x)$ 是 $\frac{1}{x}$ 在 $(-\infty, 0)$ 内的一个原函数，$\int \frac{1}{x}\,dx = \ln(-x) + C$。
把两种情况合起来，可以统一写作：
$$\int \frac{1}{x}\,dx = \ln|x| + C$$

例3（曲线方程问题）：设曲线通过点 $(1, 2)$，且其上任一点处的切线斜率等于该点横坐标的两倍，求此曲线的方程。
解：设所求的曲线方程为 $y = F(x)$，按题意 $y' = 2x$，即 $F(x)$ 是 $2x$ 的一个原函数。因为 $\int 2x\,dx = x^2 + C$，故必有某个常数 $C$ 使 $F(x) = x^2 + C$。因所求曲线通过点 $(1, 2)$，故 $2 = 1 + C$，得 $C = 1$。于是所求曲线方程为 $y = x^2 + 1$。

函数 $f(x)$ 的原函数的图形称为 $f(x)$ 的积分曲线。本例即是求函数 $2x$ 的通过点 $(1, 2)$ 的那条积分曲线。显然，这条积分曲线可以由另一条积分曲线（例如 $y = x^2$）沿 $y$ 轴方向平移而得（图4-1）。

例4（上抛运动问题）：质点以初速度 $v_0$ 铅直上抛，不计阻力，求它的运动规律。
解：取质点所在的铅直线为坐标轴，朝上为正向，轴与地面的交点作为原点。设抛出时刻为 $t = 0$，初位置 $x_0 = x(0)$。

按导数物理意义，$\frac{dx}{dt} = v(t)$ 是质点在时刻 $t$ 时的速度。加速度 $a(t) = \frac{dv}{dt} = -g$（重力加速度向下，取负号）。

于是 $v(t) = \int (-g)\,dt = -gt + C_1$。由 $v(0) = v_0$ 得 $C_1 = v_0$，所以 $v(t) = -gt + v_0$。

再求位置：$x(t) = \int v(t)\,dt = \int (-gt + v_0)\,dt = -\frac{1}{2}gt^2 + v_0t + C_2$。由 $x(0) = x_0$ 得 $C_2 = x_0$。于是所求运动规律为
$$x(t) = -\frac{1}{2}gt^2 + v_0t + x_0$$

这个公式大家在高中物理中就见过！现在我们从积分角度重新推导了一遍。

从不定积分的定义可以得到一个非常重要的关系：微分运算与积分运算是互逆的。
$$\frac{d}{dx}\left[\int f(x)\,dx\right] = f(x) \quad \text{或} \quad d\left[\int f(x)\,dx\right] = f(x)\,dx$$
$$\int F'(x)\,dx = F(x) + C \quad \text{或} \quad \int dF(x) = F(x) + C$$

由此可见，微分运算（以记号 $d$ 表示）与求不定积分的运算（以记号 $\int$ 表示）是互逆的。当 $\int$ 与 $d$ 连在一起时，要么抵消，要么抵消后差一个常数。

【二、基本积分表】

既然积分运算是微分运算的逆运算，那么很自然地可以从导数公式得到相应的积分公式。下面就是基本积分表（一定要熟练掌握！）：

(1) $\int k\,dx = kx + C$（$k$ 是常数）
(2) $\int x^\mu\,dx = \frac{x^{\mu+1}}{\mu+1} + C$（$\mu \neq -1$）
(3) $\int \frac{1}{x}\,dx = \ln|x| + C$
(4) $\int \frac{1}{1+x^2}\,dx = \arctan x + C$
(5) $\int \frac{1}{\sqrt{1-x^2}}\,dx = \arcsin x + C$
(6) $\int \cos x\,dx = \sin x + C$
(7) $\int \sin x\,dx = -\cos x + C$
(8) $\int \frac{1}{\cos^2 x}\,dx = \int \sec^2 x\,dx = \tan x + C$
(9) $\int \frac{1}{\sin^2 x}\,dx = \int \csc^2 x\,dx = -\cot x + C$
(10) $\int \sec x \tan x\,dx = \sec x + C$
(11) $\int \csc x \cot x\,dx = -\csc x + C$
(12) $\int e^x\,dx = e^x + C$
(13) $\int a^x\,dx = \frac{a^x}{\ln a} + C$

例5：求 $\int \frac{1}{x^3}\,dx$。
解：$\int \frac{1}{x^3}\,dx = \int x^{-3}\,dx = \frac{x^{-3+1}}{-3+1} + C = -\frac{1}{2x^2} + C$。

例6：求 $\int x^2\sqrt{x}\,dx$。
解：$\int x^2\sqrt{x}\,dx = \int x^{\frac{5}{2}}\,dx = \frac{x^{\frac{7}{2}}}{\frac{7}{2}} + C = \frac{2}{7}x^{\frac{7}{2}} + C$。

例7：求 $\int \frac{1}{x\sqrt[3]{x}}\,dx$。
解：$\int x^{-\frac{4}{3}}\,dx = \frac{x^{-\frac{1}{3}}}{-\frac{1}{3}} + C = -3x^{-\frac{1}{3}} + C$。

以上几例表明，有时被积函数表面上看不是幂函数，但只要化为 $x^\mu$ 的形式，就可以直接用幂函数的积分公式。

【三、不定积分的性质】

性质1（和的积分等于积分的和）：
$$\int [f(x) + g(x)]\,dx = \int f(x)\,dx + \int g(x)\,dx$$

证明：对等式右端求导，得 $\left[\int f(x)\,dx + \int g(x)\,dx\right]' = f(x) + g(x)$，这表示右端是 $f(x) + g(x)$ 的原函数。又右端有两个积分记号，形式上含两个任意常数，由于任意常数之和仍为任意常数，故实际上含一个任意常数，因此右端就是 $f(x) + g(x)$ 的不定积分。$\square$

性质2（常数因子可以提出来）：
$$\int k\,f(x)\,dx = k\int f(x)\,dx \quad (k \neq 0)$$

利用基本积分表以及不定积分的这两个性质，就可以求出很多简单函数的不定积分了。

例8：求 $\int \sqrt{x}(x^2 - 5)\,dx$。
解：$\int \sqrt{x}(x^2 - 5)\,dx = \int (x^{\frac{5}{2}} - 5x^{\frac{1}{2}})\,dx = \frac{2}{7}x^{\frac{7}{2}} - 5 \cdot \frac{2}{3}x^{\frac{3}{2}} + C$。

注意：检验积分结果是否正确，只要对结果求导，看它的导数是否等于被积函数。

例9：$\int \frac{(x-1)^3}{x^2}\,dx = \int \frac{x^3-3x^2+3x-1}{x^2}\,dx = \int (x - 3 + \frac{3}{x} - \frac{1}{x^2})\,dx = \frac{x^2}{2} - 3x + 3\ln|x| + \frac{1}{x} + C$。

例10：$\int (e^x - 3\cos x)\,dx = e^x - 3\sin x + C$。

例11：$\int 2^x e^x\,dx = \int (2e)^x\,dx = \frac{(2e)^x}{\ln(2e)} + C = \frac{2^x e^x}{1 + \ln 2} + C$。

例12：$\int \tan^2 x\,dx = \int (\sec^2 x - 1)\,dx = \tan x - x + C$。

例13：$\int \sin^2\frac{x}{2}\,dx = \int \frac{1-\cos x}{2}\,dx = \frac{1}{2}(x - \sin x) + C$。

例14：$\int \frac{1}{\sin^2\frac{x}{2}\cos^2\frac{x}{2}}\,dx = \int \frac{4}{\sin^2 x}\,dx = 4\int \csc^2 x\,dx = -4\cot x + C$。

例15：$\int \frac{2x^4+x^2+3}{x^2+1}\,dx$——先做多项式除法，化为 $\int (2x^2 - 1 + \frac{4}{x^2+1})\,dx = \frac{2}{3}x^3 - x + 4\arctan x + C$。
