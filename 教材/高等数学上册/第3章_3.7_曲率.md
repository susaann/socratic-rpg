3.7 曲率

【一、弧微分】

作为曲率的预备知识，我们先来介绍弧微分的概念。

设函数 $f(x)$ 在区间 $(a, b)$ 内具有连续导数。在曲线 $y = f(x)$ 上取固定点 $M_0(x_0, y_0)$ 作为度量弧长的基点（图3-27），并规定依 $x$ 增大的方向作为曲线的正向。对曲线上任一点 $M(x, y)$，规定有向弧段 $\widehat{M_0M}$ 的值 $s$ 如下：$s$ 的绝对值等于这段弧的长度，当有向弧段的方向与曲线的正向一致时 $s > 0$，相反时 $s < 0$。显然，弧 $s$ 与 $x$ 存在函数关系 $s = s(x)$，而且 $s(x)$ 是 $x$ 的单调增加函数。

【此处原有图表，内容描述：曲线y=f(x)上标出基点M0与动点M，弧段标注Δs】

设 $x$、$x + \Delta x$ 为 $(a, b)$ 内两个邻近的点，它们在曲线 $y = f(x)$ 上的对应点为 $M$、$M'$，对应于 $x$ 的增量为 $\Delta x$，弧 $s$ 的增量为 $\Delta s$，即 $\Delta s = \widehat{MM'}$。

经过一系列推导（利用了弦长与弧长之比的极限等于1这一几何性质），最终得到弧微分公式：
$$ds = \sqrt{1 + y'^2} \, dx \quad \text{（即 } \frac{ds}{dx} = \sqrt{1 + y'^2}\text{）}$$

这就是弧微分公式，它告诉我们弧长随 $x$ 的变化率等于 $\sqrt{1 + y'^2}$。

【二、曲率及其计算公式】

我们直觉上就知道：直线不弯曲，半径较小的圆弯曲得比半径较大的圆厉害一些，而其他曲线的不同部分有不同的弯曲程度。比如抛物线 $y = x^2$ 在顶点附近就弯曲得比远离顶点的部分厉害些。

在工程技术中，有时需要定量地研究曲线的弯曲程度。例如，船体结构中的钢梁、机床的转轴等，在荷载作用下要产生弯曲变形，设计时对它们的弯曲必须有一定的限制。为此，我们首先要讨论如何用数量来描述曲线的弯曲程度。

【此处原有图表，内容描述：图3-28 一段较平直的弧段和一段弯曲厉害的弧段对比，切线转角不同；图3-29 两条弧段相同切线转角但长度不同，短弧段弯曲更厉害】

从图3-28可以看出，弧段 $\widehat{M_1M_2}$ 比较平直，当动点沿这段弧移动时切线转过的角度 $\varphi_1$ 不大；而弧段 $\widehat{M_2M_3}$ 弯曲得比较厉害，切线转过的角度 $\varphi_2$ 就比较大。但仅看切线转角还不够——从图3-29可以看出，两段曲线弧 $\widehat{M_1M_2}$ 和 $\widehat{N_1N_2}$ 尽管切线转过的角度都是 $\varphi$，但短弧段比长弧段弯曲得厉害。由此可见，曲线弧的弯曲程度还与弧段的长度有关。

按上面的分析，我们引入描述曲线弯曲程度的曲率概念：

设曲线 $C$ 是光滑的（即曲线上每点都有切线，且切线随切点的移动而连续转动）。在曲线 $C$ 上选定一点 $M_0$ 作为度量弧 $s$ 的基点。设曲线上点 $M$ 对应于弧 $s$，在点 $M$ 处切线的倾角为 $\alpha$，曲线上另外一点 $M'$ 对应于弧 $s + \Delta s$，在点 $M'$ 处切线的倾角为 $\alpha + \Delta\alpha$（图3-30）。弧段 $\widehat{MM'}$ 的长度为 $|\Delta s|$，当动点从 $M$ 移动到 $M'$ 时切线转过的角度为 $|\Delta\alpha|$。

【此处原有图表，内容描述：曲线上点M处切线倾角α与M'处倾角α+Δα】

我们用比值 $\left|\frac{\Delta\alpha}{\Delta s}\right|$，即单位弧段上切线转过的角度的大小，来表达弧段 $\widehat{MM'}$ 的平均弯曲程度，把它叫做弧段 $\widehat{MM'}$ 的平均曲率，记作 $\bar{K}$：
$$\bar{K} = \left|\frac{\Delta\alpha}{\Delta s}\right|$$

类似于从平均速度引进瞬时速度的方法，当 $\Delta s \to 0$ 时（即 $M' \to M$ 时），上述平均曲率的极限叫做曲线 $C$ 在点 $M$ 处的曲率，记作 $K$：
$$K = \lim_{\Delta s \to 0} \left|\frac{\Delta\alpha}{\Delta s}\right|$$

在 $\frac{d\alpha}{ds}$ 存在的条件下，$K$ 也可以表示为：
$$K = \left|\frac{d\alpha}{ds}\right|$$

验证一下这个定义的合理性：

对于直线来说，切线与直线本身重合，当点沿直线移动时，切线的倾角 $\alpha$ 不变（图3-31），$\Delta\alpha = 0$，从而 $K = 0$。这就是说，直线上任意点处的曲率都等于零，这与我们"直线不弯曲"的直觉一致。

对于圆（设半径为 $a$），由图3-32可见，圆在点 $M$、$M'$ 处的切线转过的角 $\Delta\alpha$ 等于圆心角 $\angle MDM'$。设 $\angle MDM' = \Delta\theta$，则 $\Delta s = a \cdot \Delta\theta$，于是
$$\bar{K} = \left|\frac{\Delta\alpha}{\Delta s}\right| = \frac{|\Delta\theta|}{a|\Delta\theta|} = \frac{1}{a}$$
取极限得 $K = \frac{1}{a}$。这说明圆上各点处的曲率都等于半径的倒数，半径越小曲率越大，弯曲得越厉害——完全符合直觉。

【此处原有图表，内容描述：图3-31 直线上两点切线与本身重合；图3-32 圆上两点切线与对应圆心角】

【曲率的计算公式】

一般情况下，我们根据 $K = \left|\frac{d\alpha}{ds}\right|$ 来导出便于实际计算的曲率公式。

设曲线的直角坐标方程是 $y = f(x)$，且 $f(x)$ 具有二阶导数。因为 $\tan\alpha = y'$，所以
$$\sec^2\alpha \cdot \frac{d\alpha}{dx} = y''$$
$$\frac{d\alpha}{dx} = \frac{y''}{1 + \tan^2\alpha} = \frac{y''}{1 + y'^2}$$
于是
$$d\alpha = \frac{y''}{1 + y'^2} \, dx$$

又由弧微分公式 $ds = \sqrt{1 + y'^2} \, dx$，从而得曲率公式：
$$K = \frac{|y''|}{(1 + y'^2)^{3/2}}$$

如果曲线由参数方程 $x = \varphi(t)$，$y = \psi(t)$ 给出，则可利用参数方程求导法求出 $y'$ 和 $y''$，代入上式得：
$$K = \frac{|\varphi'(t)\psi''(t) - \varphi''(t)\psi'(t)|}{[\varphi'^2(t) + \psi'^2(t)]^{3/2}}$$

例1：计算等边双曲线 $xy = 1$ 在点 $(1, 1)$ 处的曲率。

解：由 $y = \frac{1}{x}$，得 $y' = -\frac{1}{x^2}$，$y'' = \frac{2}{x^3}$。因此在点 $(1, 1)$ 处，
$$y'|_{x=1} = -1, \quad y''|_{x=1} = 2$$
代入曲率公式：
$$K = \frac{|2|}{(1 + 1)^{3/2}} = \frac{2}{2\sqrt{2}} = \frac{\sqrt{2}}{2}$$

例2：抛物线 $y = ax^2 + bx + c$ 上哪一点处的曲率最大？

解：$y' = 2ax + b$，$y'' = 2a$，代入曲率公式：
$$K = \frac{|2a|}{[1 + (2ax+b)^2]^{3/2}}$$

因为 $K$ 的分子是常数 $|2a|$，所以只要分母最小，$K$ 就最大。容易看出，当 $2ax + b = 0$，即 $x = -\frac{b}{2a}$ 时，分母最小，$K$ 有最大值 $|2a|$。而 $x = -\frac{b}{2a}$ 对应的点恰好是抛物线的顶点。因此，抛物线在顶点处的曲率最大——抛物线顶点附近弯曲得最厉害！

【曲率的近似公式】

在有些实际问题中，$|y'|$ 同 $1$ 比较起来很小（有的工程技术书上记作 $|y'| \ll 1$），可以忽略不计。这时 $(1 + y'^2)^{3/2} \approx 1$，从而有曲率的近似计算公式：
$$K \approx |y''|$$

经过这样的简化之后，对一些复杂问题的计算和讨论就方便多了。

【三、曲率圆与曲率半径】

设曲线 $y = f(x)$ 在点 $M(x, y)$ 处的曲率为 $K$（$K \neq 0$）。在点 $M$ 处的曲线的法线上，在凹的一侧取一点 $D$，使 $|DM| = \frac{1}{K} = \rho$。以 $D$ 为圆心、$\rho$ 为半径作圆（图3-33），这个圆叫做曲线在点 $M$ 处的曲率圆。曲率圆的圆心 $D$ 叫做曲率中心，曲率圆的半径 $\rho$ 叫做曲线在点 $M$ 处的曲率半径。

【此处原有图表，内容描述：曲线在点M处的曲率圆，圆心D在法线上凹的一侧，半径ρ=1/K】

按上述规定可知，曲率圆与曲线在点 $M$ 有相同的切线和曲率，且在点 $M$ 附近有相同的凹凸方向。因此在实际问题中，常常用曲率圆在点 $M$ 附近的一段圆弧来近似代替曲线弧，以使问题简化。

曲率 $K$（$K \neq 0$）与曲率半径 $\rho$ 有如下关系：
$$\rho = \frac{1}{K}, \quad K = \frac{1}{\rho}$$
也就是说：曲线上一点处的曲率半径与曲率互为倒数。

例3（砂轮磨削问题）：设工件内表面的截线为抛物线 $y = 0.4x^2$（图3-34）。现在要用砂轮磨削其内表面。问用直径多大的砂轮才比较合适？

【此处原有图表，内容描述：抛物线y=0.4x^2表示工件内表面截线】

解：为了在磨削时不使砂轮与工件接触处附近的那部分工件磨去太多，砂轮的半径应不大于抛物线上各点处曲率半径中的最小值。由本节例2知道，抛物线在其顶点处的曲率最大，也就是说曲率半径最小。

因此只需求抛物线 $y = 0.4x^2$ 在顶点 $O(0, 0)$ 处的曲率半径。$y' = 0.8x$，$y'' = 0.8$。在顶点处 $y'|_{x=0} = 0$，$y''|_{x=0} = 0.8$。代入曲率公式得：
$$K = 0.8$$
曲率半径 $\rho = \frac{1}{K} = 1.25$（单位长）。

所以选用砂轮的半径不得超过 $1.25$ 单位长，即直径不得超过 $2.50$ 单位长。

对于用砂轮磨削一般工件内表面时，也有类似结论：选用的砂轮半径不应超过该工件内表面截线上各点处曲率半径的最小值。

【四、曲率中心的计算公式、渐屈线与渐伸线】

设已知曲线的方程是 $y = f(x)$，且其二阶导数 $y''$ 在点 $x$ 不为零，则曲线在对应点 $M(x, y)$ 的曲率中心 $D(\alpha, \beta)$ 的坐标为：
$$\alpha = x - \frac{y'(1 + y'^2)}{y''}, \quad \beta = y + \frac{1 + y'^2}{y''}$$

设 $y = f(x)$ 在点 $M(x, y)$ 的曲率圆方程为 $(\xi - \alpha)^2 + (\eta - \beta)^2 = \rho^2$，其中 $\rho^2 = \frac{(1 + y'^2)^3}{y''^2}$。

推导思路：因为点 $M$ 在曲率圆上，所以 $(x - \alpha)^2 + (y - \beta)^2 = \rho^2$。又因为曲线在点 $M$ 的切线与曲率圆的半径 $DM$ 相垂直（图3-33），所以
$$\frac{y - \beta}{x - \alpha} = -\frac{1}{y'}$$
由两式消去 $x - \alpha$，解出 $\beta$，再代回即可得到上述公式。

当点 $M(x, y)$ 沿曲线 $C$ 移动时，相应的曲率中心 $D$ 的轨迹曲线 $G$ 称为曲线 $C$ 的渐屈线，而曲线 $C$ 称为曲线 $G$ 的渐伸线（图3-35）。所以 $y = f(x)$ 的渐屈线的参数方程为（以 $x$ 为参数）：
$$\alpha = x - \frac{y'(1 + y'^2)}{y''}, \quad \beta = y + \frac{1 + y'^2}{y''}$$

【此处原有图表，内容描述：图3-35 曲线C与其渐屈线G的关系；图3-36 摆线及其渐屈线】

例4：求摆线 $x = a(t - \sin t)$，$y = a(1 - \cos t)$ 的渐屈线方程。

解：$\frac{dx}{dt} = a(1 - \cos t)$，$\frac{dy}{dt} = a\sin t$，所以
$$\frac{dy}{dx} = \frac{\sin t}{1 - \cos t}$$
进一步求得
$$\frac{d^2y}{dx^2} = \frac{\frac{d}{dt}(\frac{dy}{dx})}{\frac{dx}{dt}} = -\frac{1}{a(1 - \cos t)^2}$$

将这些结果代入渐屈线公式并化简，得渐屈线的参数方程为：
$$\alpha = a(t + \sin t), \quad \beta = a(\cos t - 1)$$

令 $t = \tau + \pi$，再作适当的坐标平移，可以证明渐屈线仍为一摆线。摆线的渐屈线还是摆线——这是一个非常优美的几何性质（如图3-36所示）。
