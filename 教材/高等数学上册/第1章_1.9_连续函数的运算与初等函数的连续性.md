知道了连续的定义后，由极限运算法则可以自然地得出连续函数的运算规则。本节的核心结论是：一切初等函数在其定义区间内都是连续的。有了这个结论，求大量函数的极限就变得极其简单——直接代入就行！

一、连续函数的四则运算

定理1：如果 f(x) 和 g(x) 都在点 x₀ 连续，那么它们的和（差）、积、商（分母不为零）也都在 x₀ 连续。

这个定理从极限四则运算法则直接得出。因为 lim(f+g) = lim f + lim g = f(x₀) + g(x₀)。

例1：tan x = sin x / cos x，而 sin x 和 cos x 都在 (-∞, +∞) 内连续，所以 tan x 和 cot x 在它们的定义域内都是连续的。

二、反函数与复合函数的连续性

定理2（反函数的连续性）：如果 y = f(x) 在区间 I 上单调增加（或减少）且连续，那么它的反函数 x = f⁻¹(y) 在对应的值域 { y | y = f(x), x ∈ I } 上也单调增加（或减少）且连续。

例2：y = sin x 在 [-π/2, π/2] 上单调增加且连续，所以反函数 y = arcsin x 在 [-1, 1] 上也是单调增加且连续的。

同理可证：arccos x 在 [-1, 1] 上单调减少且连续；arctan x 在 (-∞, +∞) 内单调增加且连续；arccot x 在 (-∞, +∞) 内单调减少且连续。

总之，所有反三角函数在其定义域内都是连续的。

定理3（复合函数求极限——换元法的加强版）：设 y = f[g(x)] 由 u = g(x) 和 y = f(u) 复合而成，且 f(u) 在 u₀ 连续。如果 lim_{x→x₀} g(x) = u₀，则

lim_{x→x₀} f[g(x)] = lim_{u→u₀} f(u) = f(u₀)

这个定理有一个重要推论：把极限号"穿进"连续函数里面——

lim f[g(x)] = f[lim g(x)]

也就是说，如果外层函数连续，求极限的符号可以和函数符号交换次序。这是一个非常方便的技巧。

例3：求 lim_{x→3} √((x-3)/(x²-9))
解：先求内层极限 lim_{x→3} (x-3)/(x²-9) = lim_{x→3} 1/(x+3) = 1/6，而 √u 在 u = 1/6 处连续，所以原式 = √(1/6)。

定理4（复合函数的连续性）：两个连续函数复合后仍是连续函数。也就是说：如果 u = g(x) 在 x₀ 连续，且 g(x₀) = u₀，而 y = f(u) 在 u₀ 连续，那么复合函数 y = f[g(x)] 在 x₀ 也连续。

例4：讨论 y = sin(1/x) 的连续性。
解：1/x 在 (-∞, 0) 和 (0, +∞) 内连续，sin u 在 (-∞, +∞) 内连续。根据定理4，sin(1/x) 在 (-∞, 0) 和 (0, +∞) 内是连续的。

三、初等函数的连续性

前面已经证明了三角函数和反三角函数在它们的定义域内连续。同样可以证明：

- 指数函数 y = a^x（a > 0, a ≠ 1）：在 (-∞, +∞) 内单调且连续，值域为 (0, +∞)。
- 对数函数 y = log_a x（a > 0, a ≠ 1）：由指数函数的反函数，在 (0, +∞) 内单调且连续。
- 幂函数 y = x^μ：可看作 y = e^(μ·ln x)，是两个连续函数的复合，在 (0, +∞) 内连续。实际上，幂函数在其整个定义域内都是连续的。

综合起来：基本初等函数在它们的定义域内都是连续的。

最后，根据初等函数的定义（由基本初等函数经有限次四则运算和复合而成），再结合定理1和定理4，得到本节最重要的结论：

★ 一切初等函数在其定义区间内都是连续的。

【这个结论怎么用？】

如果 f(x) 是初等函数，且 x₀ 在其定义区间内，那么 lim_{x→x₀} f(x) = f(x₀)——直接代入就行了！

例如：
- 点 x = 0 是初等函数 y = √(1-x²) 的定义区间 [-1, 1] 上的点，所以 lim_{x→0} √(1-x²) = √1 = 1。
- 点 x = π/2 是初等函数 y = ln(sin x) 的定义区间 (0, π) 内的点，所以 lim_{x→π/2} ln(sin x) = ln(sin(π/2)) = ln 1 = 0。

这一结论极大地简化了求极限：遇到初等函数，先看那点是不是定义区间内的点，是的话，直接代入算函数值就行了！

四、补充：幂指函数的极限

对于形如 u(x)^(v(x))（u(x) > 0, u(x) ≠ 1）的幂指函数，如果 lim u(x) = a > 0，lim v(x) = b，那么 lim u(x)^(v(x)) = a^b。这是因为 u(x)^(v(x)) = e^(v(x)·ln u(x))，而指数函数是连续的。

另外，由本节的计算还得到三个重要的等价无穷小关系式（当 x → 0 时）：
- ln(1+x) ~ x
- e^x - 1 ~ x
- (1+x)^a - 1 ~ a·x
