有了导数这个工具，研究函数的图形性质变得非常方便。本节讲两个基本问题：函数在哪里上升/下降？曲线在哪里凹/凸？

一、函数单调性的判定

【判定定理（定理1）】

设函数 y = f(x) 在 [a, b] 上连续，在 (a, b) 内可导。

(1) 如果在 (a, b) 内 f'(x) ≥ 0，且等号仅在有限个点成立，则 f(x) 在 [a, b] 上单调增加；
(2) 如果在 (a, b) 内 f'(x) ≤ 0，且等号仅在有限个点成立，则 f(x) 在 [a, b] 上单调减少。

简单记：导数大于零就上升，导数小于零就下降。导数为零的点可能只是"停顿一下"。

【怎么用——找单调区间】

步骤：
1. 求出导数 f'(x)；
2. 解 f'(x) = 0，找出驻点（及导数不存在的点）；
3. 用这些点把定义域分成若干区间，在每个区间内判断 f'(x) 的正负号。

例：讨论 f(x) = 2x³ - 9x² + 12x - 3 的单调性。
解：f'(x) = 6x² - 18x + 12 = 6(x-1)(x-2)。
驻点 x = 1, 2。
- x ∈ (-∞, 1)：f'(x) > 0，单调增
- x ∈ (1, 2)：f'(x) < 0，单调减
- x ∈ (2, +∞)：f'(x) > 0，单调增

例：讨论 y = ∛x 的单调性。
解：y' = 1/(3·∛x²)，在 x = 0 处导数不存在。x < 0 时 y' > 0，x > 0 时 y' > 0，所以函数在 (-∞, +∞) 上单调增加——虽然 x = 0 处导数不存在，但不影响整体单调性。

【不等式证明中的应用】

例：证明当 x > 1 时，2√x > 3 - 1/x。
证：令 F(x) = 2√x - (3 - 1/x)，则 F'(x) = 1/√x - 1/x² > 0 (x > 1)。所以 F(x) 在 [1, +∞) 上单调增，F(x) > F(1) = 0，即得证。

二、曲线的凹凸性与拐点

单调性告诉我们曲线是上升还是下降。但同样是上升，有的曲线向上弯，有的向下弯——这就是凹凸性。

【定义】

如果曲线弧上任意两点间的弦位于弧段的上方（曲线"兜"着弦），称曲线是凹的（向上凹）；如果弦在弧段的下方，称曲线是凸的（向上凸）。

用二阶导数来判定：

定理2：设 f(x) 在 [a, b] 上连续，在 (a, b) 内具有二阶导数。
(1) 若 f''(x) > 0，则 f(x) 在 [a, b] 上的图形是凹的（开口朝上）；
(2) 若 f''(x) < 0，则 f(x) 在 [a, b] 上的图形是凸的（开口朝下）。

生活类比：二阶导数大于零，就像加速度为正——不仅往前走，而且越走越快，曲线"翘起来"；二阶导数小于零则相反，势头在衰减。

【拐点】

曲线从凹变凸（或凸变凹）的分界点叫拐点。寻找拐点的方法与找单调区间类似：

1. 求 f''(x)；
2. 找出 f''(x) = 0 的点和 f''(x) 不存在的点；
3. 检查这些点两侧 f''(x) 是否变号。变号则是拐点，不变则不是。

例：求 y = 2x³ + 3x² - 12x + 14 的拐点。
解：y' = 6x² + 6x - 12，y'' = 12x + 6 = 12(x + 1/2)。
x < -1/2 时 y'' < 0（凸），x > -1/2 时 y'' > 0（凹）。
所以 (-1/2, 20.5) 是拐点。

例：求 y = ∛x 的拐点。
解：y'' = -2/(9·∛x⁵)，在 x = 0 处不存在。x < 0 时 y'' > 0（凹），x > 0 时 y'' < 0（凸）。所以 (0, 0) 是拐点。

一个重要细节：f''(x₀) = 0 不一定是拐点！比如 y = x⁴，y'' = 12x²，在 x = 0 处 y'' = 0，但左右都大于 0（都凹），所以 (0, 0) 不是拐点。必须检查符号是否变化。
