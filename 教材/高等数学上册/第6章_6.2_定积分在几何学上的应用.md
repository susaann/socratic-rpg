6.2 定积分在几何学上的应用

本节运用元素法来讨论定积分在几何学中的几个重要应用：求平面图形的面积、求体积和求曲线弧长。

【一、平面图形的面积】

1. 直角坐标情形

在第五章中我们已经知道，由曲线 $y = f(x)$（$f(x) \geq 0$）及直线 $x = a$、$x = b$（$a < b$）与 $x$ 轴所围成的曲边梯形的面积为
$$A = \int_a^b f(x)\,dx$$

关键在于如何找到正确的面积元素 $dA$——这需要具体情况具体分析。

例1：计算由两条抛物线 $y^2 = x$ 和 $y = x^2$ 所围成的图形的面积。

解：两抛物线交点为 $(0, 0)$ 和 $(1, 1)$。图形在 $x = 0$ 和 $x = 1$ 之间。
取 $x$ 为积分变量，区间为 $[0, 1]$。在 $[x, x+dx]$ 上，两曲线间窄条的高度近似为 $\sqrt{x} - x^2$，底为 $dx$，所以面积元素为 $dA = (\sqrt{x} - x^2)\,dx$。于是
$$A = \int_0^1 (\sqrt{x} - x^2)\,dx = \left[\frac{2}{3}x^{\frac{3}{2}} - \frac{x^3}{3}\right]_0^1 = \frac{1}{3}$$

例2：计算抛物线 $y^2 = 2x$ 与直线 $y = x - 4$ 所围成的图形的面积。

解：交点 $(2, -2)$ 和 $(8, 4)$。取 $y$ 为积分变量较方便（区间 $[-2, 4]$），窄条面积元素为 $dA = [(y+4) - \frac{y^2}{2}]\,dy$。于是
$$A = \int_{-2}^4 \left(y+4 - \frac{y^2}{2}\right)dy = 18$$

这个例子说明：积分变量的选择会影响计算的简便程度——选择得当，事半功倍。

例3：计算椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 所围成图形的面积。

解：利用对称性，面积 $A = 4A_1$，其中 $A_1$ 是第一象限部分的面积。用参数方程 $x = a\cos t$，$y = b\sin t$ 及换元积分法，得
$$A = 4\int_0^a y\,dx = 4\int_{\frac{\pi}{2}}^0 b\sin t \cdot (-a\sin t)\,dt = 4ab\int_0^{\frac{\pi}{2}} \sin^2 t\,dt = \pi ab$$
当 $a = b$ 时，就得到圆面积公式 $A = \pi a^2$。

2. 极坐标情形

对于由极坐标方程 $\rho = \rho(\theta)$（$\alpha \leq \theta \leq \beta$）给出的曲线与射线 $\theta = \alpha$、$\theta = \beta$ 所围成的曲边扇形，面积元素为
$$dA = \frac{1}{2}\rho^2(\theta)\,d\theta$$
从而面积为 $A = \int_\alpha^\beta \frac{1}{2}\rho^2(\theta)\,d\theta$。

【二、体积】

1. 旋转体的体积

旋转体是由一个平面图形绕某直线旋转一周而成的立体。最常见的旋转体是由曲线 $y = f(x)$ 及 $x = a$、$x = b$、$x$ 轴所围成的曲边梯形绕 $x$ 轴旋转得到的。此时旋转体的体积为
$$V = \int_a^b \pi [f(x)]^2\,dx$$

2. 平行截面面积已知的立体的体积

如果一个立体在 $x$ 处的截面面积 $A(x)$ 已知（$a \leq x \leq b$），那么体积元素为 $dV = A(x)\,dx$，立体体积为
$$V = \int_a^b A(x)\,dx$$

例4（正劈锥体的体积）：以圆为底、以平行且等于底圆直径的线段为顶的正劈锥体，其体积等于同底同高的圆柱体体积的一半，即 $V = \frac{1}{2}\pi R^2 h$。

【三、平面曲线的弧长】

设曲线弧由参数方程 $x = \varphi(t)$，$y = \psi(t)$（$\alpha \leq t \leq \beta$）给出，且 $\varphi'(t)$、$\psi'(t)$ 连续（即曲线是光滑的）。弧长元素（弧微分）为
$$ds = \sqrt{(dx)^2 + (dy)^2} = \sqrt{\varphi'^2(t) + \psi'^2(t)}\,dt$$
从而弧长 $s = \int_\alpha^\beta \sqrt{\varphi'^2(t) + \psi'^2(t)}\,dt$。

若曲线由直角坐标方程 $y = f(x)$（$a \leq x \leq b$）给出，则
$$s = \int_a^b \sqrt{1 + [f'(x)]^2}\,dx$$

若曲线由极坐标方程 $\rho = \rho(\theta)$ 给出，则
$$s = \int_\alpha^\beta \sqrt{\rho^2(\theta) + \rho'^2(\theta)}\,d\theta$$

例5：计算摆线 $x = a(\theta - \sin\theta)$，$y = a(1 - \cos\theta)$ 一拱（$0 \leq \theta \leq 2\pi$）的长度。
解：弧长元素 $ds = a\sqrt{2(1-\cos\theta)}\,d\theta = 2a\sin\frac{\theta}{2}\,d\theta$，于是
$$s = \int_0^{2\pi} 2a\sin\frac{\theta}{2}\,d\theta = 8a$$。
摆线一拱的长度恰好是滚动圆直径的4倍，这个简洁的结果让人印象深刻。

例6：求阿基米德螺线 $\rho = a\theta$（$a > 0$）上从 $\theta = 0$ 到 $\theta = 2\pi$ 的一段弧长。
解：弧长元素 $ds = \sqrt{a^2 + a^2\theta^2}\,d\theta = a\sqrt{1+\theta^2}\,d\theta$，积分得弧长公式（含 $\ln$ 和根式）。
