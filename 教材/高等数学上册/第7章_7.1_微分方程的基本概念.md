7.1 微分方程的基本概念

【什么是微分方程】

函数是客观事物内部联系在数量方面的反映。在很多实际问题中，我们往往不能直接找出所需的函数关系，但根据问题提供的情况，有时可以列出含有未知函数及其导数的关系式——这就是微分方程。建立微分方程后，对它进行研究，找出未知函数来，这就是解微分方程。

一般地，凡表示未知函数、未知函数的导数与自变量之间的关系的方程，叫做微分方程。

微分方程中所出现的未知函数的最高阶导数的阶数，叫做微分方程的阶。例如：
- $y' = 2x$ 是一阶微分方程
- $\frac{d^2s}{dt^2} = -0.4$ 是二阶微分方程
- $xy''' + x^2y'' - 4xy' = 3x^2$ 是三阶微分方程

一般地，$n$ 阶微分方程的形式是
$$F(x, y, y', \cdots, y^{(n)}) = 0$$

【微分方程的解】

找出一个函数，把它代入微分方程后能使该方程成为恒等式，这个函数就叫做该微分方程的解。

如果微分方程的解中含有任意常数，且任意常数的个数与方程的阶数相同（且任意常数相互独立），这样的解叫做微分方程的通解。例如：
- $y = x^2 + C$ 是 $y' = 2x$ 的通解
- $s = -0.2t^2 + C_1 t + C_2$ 是 $\frac{d^2s}{dt^2} = -0.4$ 的通解

确定了通解中的任意常数以后，就得到微分方程的特解。确定任意常数的条件叫做初值条件。

一阶微分方程的初值问题：
$$\begin{cases} y' = f(x, y) \\ y|_{x=x_0} = y_0 \end{cases}$$
其几何意义是：求微分方程的通过点 $(x_0, y_0)$ 的那条积分曲线。

二阶微分方程的初值问题：
$$\begin{cases} y'' = f(x, y, y') \\ y|_{x=x_0} = y_0, \; y'|_{x=x_0} = y'_0 \end{cases}$$
其几何意义是：求通过点 $(x_0, y_0)$ 且在该点处切线斜率为 $y'_0$ 的那条积分曲线。

例：验证 $x = C_1\cos kt + C_2\sin kt$ 是微分方程 $\frac{d^2x}{dt^2} + k^2 x = 0$ 的解。

解：求导得 $\frac{dx}{dt} = -kC_1\sin kt + kC_2\cos kt$，$\frac{d^2x}{dt^2} = -k^2(C_1\cos kt + C_2\sin kt) = -k^2 x$。代入方程得 $-k^2 x + k^2 x = 0$，成为恒等式。因此所给函数是微分方程的解。又因为含有两个独立的任意常数 $C_1$、$C_2$，而方程是二阶的，所以是通解。
