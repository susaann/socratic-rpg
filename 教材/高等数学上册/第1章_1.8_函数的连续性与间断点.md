前面讲了极限，这一节讲一个和极限密切相关的概念——连续性。自然界中很多现象都是连续变化的：气温不会瞬间跳变，河水不会断流，植物慢慢长高。这种"渐变"在数学上的反映，就是函数的连续性。

一、函数的连续性

【增量的概念】

设变量 u 从初值 u₁ 变到终值 u₂，终值与初值的差 u₂ - u₁ 就叫做 u 的增量，记作 Δu = u₂ - u₁。Δ 是一个整体记号，不是 Δ 乘以 u。增量可以是正的（变大了），也可以是负的（变小了）。

对于一个函数 y = f(x)，如果自变量 x 从 x₀ 变到 x₀ + Δx，那么函数值的对应增量就是 Δy = f(x₀ + Δx) - f(x₀)。

【连续的定义】

直观上，连续就是"自变量变化很小时，函数值也变化很小"。用增量语言说就是：如果 Δx → 0 时，Δy → 0，那么函数在 x₀ 处连续。

定义1（增量形式）：设函数 y = f(x) 在点 x₀ 的某一邻域内有定义，如果 lim_{Δx→0} Δy = lim_{Δx→0} [f(x₀+Δx) - f(x₀)] = 0，则称函数在 x₀ 处连续。

令 x = x₀ + Δx，则 Δx → 0 就相当于 x → x₀。又由于 Δy = f(x) - f(x₀)，所以 Δy → 0 等价于 f(x) → f(x₀)。于是有等价的定义：

定义2（极限形式）：设函数 y = f(x) 在点 x₀ 的某一邻域内有定义，如果 lim_{x→x₀} f(x) = f(x₀)，则称函数在 x₀ 处连续。

用 ε-δ 语言：f(x) 在 x₀ 处连续 ⇔ ∀ε > 0, ∃δ > 0, 当 |x - x₀| < δ 时，有 |f(x) - f(x₀)| < ε。

对比极限定义，连续性多了一个要求：极限值必须等于函数值 f(x₀)。也就是说，连续 = 极限存在 + 极限等于函数值。

【左连续与右连续】

- 如果 lim_{x→x₀⁻} f(x) = f(x₀)，称 f(x) 在 x₀ 左连续。
- 如果 lim_{x→x₀⁺} f(x) = f(x₀)，称 f(x) 在 x₀ 右连续。

在区间上每一点都连续的函数，叫做该区间上的连续函数。如果区间包含端点，在右端点要求左连续，在左端点要求右连续。连续函数的图形是一条连续而不间断的曲线。

【哪些函数是连续的】

- 多项式（有理整函数）：对任意实数 x₀，lim_{x→x₀} f(x) = f(x₀)，所以在 (-∞, +∞) 内处处连续。
- 有理分式函数：只要分母不为 0，在其定义域内每一点都连续。
- y = √x 在 (0, +∞) 内连续。
- y = sin x 在 (-∞, +∞) 内连续。（证明思路：|sin(x+Δx) - sin x| ≤ |Δx|，Δx → 0 时增量也趋于 0）
- y = cos x 同样在 (-∞, +∞) 内连续。

二、函数的间断点

函数不连续的点叫间断点。如果函数 f(x) 在 x₀ 的某去心邻域内有定义，且出现以下三种情况之一，x₀ 就是间断点：
(1) f(x) 在 x₀ 没有定义；
(2) 虽然在 x₀ 有定义，但 lim_{x→x₀} f(x) 不存在；
(3) 虽然在 x₀ 有定义且极限存在，但 lim_{x→x₀} f(x) ≠ f(x₀)。

换个说法：连续要求"函数值 = 极限值"，只要三者（左极限、右极限、函数值）中有一个不协调，就断开了。

【间断点的分类】

（1）第一类间断点：左右极限都存在的间断点。又可细分为：
- 可去间断点：左右极限相等。比如 y = (x²-1)/(x-1) 在 x=1 处没有定义，但极限等于 2。如果补充定义 f(1) = 2，就连续了。这个"缺口"可以补上。
- 跳跃间断点：左右极限都存在但不相等。比如 f(x) = { x-1, x<0; 0, x=0; x+1, x>0 }，在 x=0 处左极限 -1，右极限 1，图形在 x=0 处"跳"了一下。

（2）第二类间断点：不属于第一类的都叫第二类。典型的有：
- 无穷间断点：比如 y = tan x 在 x = π/2 处，极限为 ∞。
- 振荡间断点：比如 y = sin(1/x) 在 x=0 处，函数值在 -1 和 1 之间无限振荡。

记住这个分类很简单：先看左右极限在不在——都在就是第一类（相等的可去，不等的跳跃）；至少有一个不存在就是第二类。
