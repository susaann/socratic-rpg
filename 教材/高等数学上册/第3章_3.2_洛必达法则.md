如果两个函数在某点的极限都是 0（或都是 ∞），它们的比值极限会是什么？这种"0/0"或"∞/∞"的不定式没法直接用极限运算法则。洛必达法则就是专门解决这类问题的利器。

一、0/0 型不定式

定理1：设
(1) 当 x → x₀ 时，f(x) → 0 且 F(x) → 0；
(2) 在 x₀ 的某去心邻域内，f'(x) 和 F'(x) 都存在且 F'(x) ≠ 0；
(3) lim f'(x)/F'(x) 存在（或为无穷大），
则 lim f(x)/F(x) = lim f'(x)/F'(x)。

简单说就是：遇到 0/0 型 → 分子分母分别求导 → 再取极限。如果求导后还是 0/0，可以继续用，直到求出结果为止。

例1：求 lim_{x→0} sin(ax)/sin(bx)（b ≠ 0）
解：0/0 型，洛必达：= lim_{x→0} a·cos(ax)/[b·cos(bx)] = a/b。

例2：求 lim_{x→1} (x³ - 3x + 2)/(x³ - x² - x + 1)
解：代入得 0/0。第一次洛必达后得 (3x²-3)/(3x²-2x-1)，还是 0/0。第二次洛必达：6x/(6x-2) → 6/4 = 3/2。

【重要提醒】每次使用前都要确认确实是不定式！如果不是 0/0 或 ∞/∞，就不能用洛必达法则，否则会出错。

二、∞/∞ 型不定式

当 x → x₀ 或 x → ∞ 时，有类似的法则（定理2）。例如：

例3：求 lim_{x→+∞} ln x / x^n（n > 0）
解：∞/∞ 型。洛必达：= lim (1/x)/(n·x^(n-1)) = lim 1/(n·x^n) = 0。

这说明：对数函数增长得比任何正幂次都慢——ln x 虽然也趋向无穷，但"跑不过"x 的任何正次幂。

例4：求 lim_{x→+∞} x^n / e^(λx)（λ > 0）
解：∞/∞ 型，反复用洛必达 n 次后分母不变（多出 λ^n），分子变为 n!，所以极限为 0。

这说明：指数函数增长得比任何幂函数都快。总结三者增长速度：对数 < 幂 < 指数。

三、其他类型的不定式

洛必达法则主要解决 0/0 和 ∞/∞。对于 0·∞、∞-∞、1^∞、0⁰、∞⁰ 等类型，可以通过代数变形化为 0/0 或 ∞/∞。

例5：求 lim_{x→0⁺} x^n·ln x（n > 0）—— 0·(-∞) 型
解：改写为 lim ln x / x^(-n)，变成 ∞/∞ 型。
洛必达：= lim (1/x)/(-n·x^(-n-1)) = lim -x^n/n = 0。

例6：求 lim_{x→π/2} (sec x - tan x)—— ∞-∞ 型
解：sec x - tan x = (1 - sin x)/cos x，当 x → π/2 是 0/0 型。
洛必达：= lim (-cos x)/(-sin x) = 0。

例7：求 lim_{x→0⁺} x^x —— 0⁰ 型
解：设 y = x^x，ln y = x·ln x。
lim ln y = lim x·ln x = 0（由例5）。
所以 lim y = e⁰ = 1。

四、洛必达法则的使用注意事项

1. 能化简先化简——约分、等价无穷小替换优先使用，会让计算简洁很多。
2. 如果 lim f'/F' 不存在（且不为 ∞），不能断言原极限不存在——可能只是洛必达失效，需要换其他方法。
3. 结合重要极限、等价无穷小等技巧使用，效果最好。

例：lim_{x→0} (tan x - x)/(x²·sin x)
先 sin x ~ x，变为 (tan x - x)/x³ → 0/0。洛必达：(sec²x - 1)/3x² = tan²x/3x² → 1/3。比直接对原式洛必达简洁很多。
