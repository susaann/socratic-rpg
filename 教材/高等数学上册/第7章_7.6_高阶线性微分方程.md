7.6 高阶线性微分方程

本节和以下两节讨论在实际问题中应用得非常广泛的高阶线性微分方程，以二阶为主。

【一、二阶线性微分方程举例——振动与电路】

例1（弹簧振子）：弹簧下端挂质量 $m$ 的物体。设弹性恢复力 $f = -cx$（$c$ 为劲度系数），阻尼力 $R = -\mu\frac{dx}{dt}$（与速度成正比）。由牛顿第二定律得：
$$\frac{d^2x}{dt^2} + 2n\frac{dx}{dt} + k^2 x = 0 \quad (其中\; n = \frac{\mu}{2m},\; k^2 = \frac{c}{m})$$

这是有阻尼自由振动的微分方程。如有外加干扰力 $F = H\sin pt$，则方程变为
$$\frac{d^2x}{dt^2} + 2n\frac{dx}{dt} + k^2 x = h\sin pt$$

例2（RLC电路）：电阻 $R$、电感 $L$、电容 $C$ 串联，外加电源 $E = E_m\sin\omega t$。由回路电压定律得：
$$LC\frac{d^2u_c}{dt^2} + RC\frac{du_c}{dt} + u_c = E_m\sin\omega t$$

虽然弹簧振子和 RLC 电路是完全不同的物理系统，但它们满足同样形式的微分方程！这正体现了数学的统一美——不同领域的现象可以用同一类方程来刻画。

一般地，二阶线性微分方程的标准形式为：
$$y'' + P(x)y' + Q(x)y = f(x)$$
$f(x) \equiv 0$ 时称为齐次，$f(x) \not\equiv 0$ 时称为非齐次。

【二、线性微分方程解的结构】

定理1（叠加原理）：如果 $y_1(x)$ 与 $y_2(x)$ 是齐次方程 $y'' + P(x)y' + Q(x)y = 0$ 的两个解，那么
$$y = C_1y_1(x) + C_2y_2(x)$$
也是该方程的解（$C_1$、$C_2$ 为任意常数）。

但这不一定是通解！比如 $y_2 = 2y_1$ 时，$y = (C_1+2C_2)y_1 = Cy_1$，只剩下一个任意常数。

定义（线性相关与线性无关）：如果存在不全为零的常数 $k_1, k_2, \cdots, k_n$ 使得 $k_1y_1 + k_2y_2 + \cdots + k_ny_n \equiv 0$，则称这些函数线性相关；否则称线性无关。

对于两个函数，判断方法是：$\frac{y_2}{y_1} \neq$ 常数 $\iff$ 线性无关。

定理2（齐次方程通解结构）：如果 $y_1(x)$ 与 $y_2(x)$ 是齐次方程的两个线性无关的特解，那么
$$y = C_1y_1(x) + C_2y_2(x)$$
就是齐次方程的通解。

例如，$y'' + y = 0$ 有两个解 $y_1 = \cos x$ 和 $y_2 = \sin x$，且 $\frac{\sin x}{\cos x} = \tan x \neq$ 常数（线性无关），所以通解为 $y = C_1\cos x + C_2\sin x$。

定理3（非齐次方程通解结构）：设 $y^*$ 是非齐次方程 $y'' + P(x)y' + Q(x)y = f(x)$ 的一个特解，$Y$ 是对应齐次方程的通解，则非齐次方程的通解为
$$y = Y + y^*$$
（即：非齐通 = 齐通 + 非奇特）

这个结构定理极其重要：要求非齐次方程的通解，只需求对应齐次方程的通解，再加上非齐次方程的一个特解即可。
