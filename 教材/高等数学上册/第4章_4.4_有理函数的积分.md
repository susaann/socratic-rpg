4.4 有理函数的积分

前面介绍了求不定积分的两个基本方法——换元积分法和分部积分法。本节简要介绍有理函数的积分以及可化为有理函数的积分。

【一、有理函数的积分】

两个多项式的商 $\frac{P(x)}{Q(x)}$ 称为有理函数，又称有理分式。我们总假定分子多项式 $P(x)$ 与分母多项式 $Q(x)$ 之间没有公因式。当分子多项式的次数小于分母多项式的次数时，称为真分式；否则称为假分式。

第一步：如果是假分式，先用多项式除法把它化成一个多项式加一个真分式。例如
$$\frac{2x^4+x^2+3}{x^2+1} = 2x^2 - 1 + \frac{4}{x^2+1}$$
多项式的积分很容易（直接用基本积分表），所以问题集中在真分式的积分。

第二步：把真分式化为部分分式之和。部分分式理论告诉我们，任何一个真分式都可以分解为以下三种最简分式之和：
$$\frac{A}{(x-a)^k}, \qquad \frac{Ax+B}{(x^2+px+q)^k}$$
（这里 $p^2-4q<0$，$A$、$B$ 为待定常数，$k$ 为正整数）

求待定系数的方法常用待定系数法：设出部分分式的形式（含未知系数 $A$、$B$、$C$、$D$…），然后通分，比较分子多项式同次幂的系数，得到一个线性方程组，解出系数。

例1：求 $\int \frac{x+1}{x^2-5x+6}\,dx$。

分母分解：$x^2-5x+6 = (x-2)(x-3)$。设
$$\frac{x+1}{x^2-5x+6} = \frac{A}{x-2} + \frac{B}{x-3}$$
去分母得：$x+1 = A(x-3) + B(x-2) = (A+B)x + (-3A-2B)$。
比较系数：
$$\begin{cases}A + B = 1 \\ -3A - 2B = 1\end{cases}$$
解得 $A = -3$，$B = 4$（或 $A=4, B=-3$，取决于符号选取）。实际解得：$A=4$，$B=-3$。于是
$$\int \frac{x+1}{x^2-5x+6}\,dx = \int \left(\frac{4}{x-3} - \frac{3}{x-2}\right)dx = 4\ln|x-3| - 3\ln|x-2| + C$$

例2：求 $\int \frac{x+2}{(2x+1)(x^2+x+1)}\,dx$。

设
$$\frac{x+2}{(2x+1)(x^2+x+1)} = \frac{A}{2x+1} + \frac{Bx+D}{x^2+x+1}$$
去分母得：$x+2 = A(x^2+x+1) + (Bx+D)(2x+1) = (A+2B)x^2 + (A+B+2D)x + (A+D)$。
比较系数得 $A=2$，$B=-1$，$D=0$。于是
$$\int \frac{x+2}{(2x+1)(x^2+x+1)}\,dx = \ln|2x+1| - \frac{1}{2}\ln(x^2+x+1) + \frac{1}{\sqrt{3}}\arctan\frac{2x+1}{\sqrt{3}} + C$$

例3：求 $\int \frac{x-3}{(x-1)^2(x+1)}\,dx$。

注意：分母中含有重因式 $(x-1)^2$，此时部分分式的形式中须包含 $\frac{Ax+B}{(x-1)^2}$ 和 $\frac{D}{x+1}$ 两项。设
$$\frac{x-3}{(x-1)^2(x+1)} = \frac{Ax+B}{(x-1)^2} + \frac{D}{x+1}$$
展开比较系数得 $A=1$，$B=-2$，$D=-1$。简化后
$$\frac{x-3}{(x-1)^2(x+1)} = \frac{x-2}{(x-1)^2} - \frac{1}{x+1}$$
积分得：
$$\int \frac{x-3}{(x-1)^2(x+1)}\,dx = \ln|x-1| + \frac{1}{x-1} - \ln|x+1| + C$$

【二、可化为有理函数的积分举例】

有些被积函数本身不是有理函数，但通过适当的变量代换，可以化为有理函数来积分。

（1）三角函数有理式的积分——万能的 $\tan\frac{x}{2}$ 代换

由三角学知道，$\sin x$ 和 $\cos x$ 都可以用 $\tan\frac{x}{2}$ 的有理式表示：
$$\sin x = \frac{2\tan\frac{x}{2}}{1+\tan^2\frac{x}{2}}, \quad \cos x = \frac{1-\tan^2\frac{x}{2}}{1+\tan^2\frac{x}{2}}$$
令 $u = \tan\frac{x}{2}$（$-\pi < x < \pi$），则
$$\sin x = \frac{2u}{1+u^2}, \quad \cos x = \frac{1-u^2}{1+u^2}, \quad dx = \frac{2}{1+u^2}\,du$$
这样，任何三角函数有理式的积分都可以化为有理函数的积分。这个代换因而被称为"万能代换"。

例4：求 $\int \frac{1+\sin x}{\sin x(1+\cos x)}\,dx$。

令 $u = \tan\frac{x}{2}$，代入万能代换，得
$$\int \frac{1+\sin x}{\sin x(1+\cos x)}\,dx = \frac{1}{2}\int \left(\frac{1}{u} + 2 + u\right)du = \frac{1}{2}\ln|u| + u + \frac{1}{4}u^2 + C$$
$$= \frac{1}{2}\ln|\tan\frac{x}{2}| + \tan\frac{x}{2} + \frac{1}{4}\tan^2\frac{x}{2} + C$$

（2）简单无理函数的积分——根式代换

当被积函数中含有简单根式时，设根式为 $u$，将整个积分化为有理函数的积分。

例5：$\int \frac{\sqrt{x-1}}{x}\,dx$。
为去掉根号，设 $\sqrt{x-1} = u$，则 $x = u^2 + 1$，$dx = 2u\,du$。
$$\int \frac{\sqrt{x-1}}{x}\,dx = \int \frac{u}{u^2+1} \cdot 2u\,du = 2\int \frac{u^2}{u^2+1}\,du$$
$$= 2\int (1 - \frac{1}{u^2+1})\,du = 2(u - \arctan u) + C$$
$$= 2(\sqrt{x-1} - \arctan\sqrt{x-1}) + C$$

例6：求 $\int \frac{1}{1+\sqrt[3]{x+2}}\,dx$。
为去掉三次根号，设 $\sqrt[3]{x+2} = u$，则 $x = u^3 - 2$，$dx = 3u^2\,du$。
$$\int \frac{1}{1+\sqrt[3]{x+2}}\,dx = 3\int \frac{u^2}{1+u}\,du = 3\int (u-1+\frac{1}{1+u})\,du$$
$$= 3\left(\frac{u^2}{2} - u + \ln|1+u|\right) + C = \frac{3}{2}(x+2)^{\frac{2}{3}} - 3\sqrt[3]{x+2} + 3\ln|1+\sqrt[3]{x+2}| + C$$

例7：$\int \frac{1}{\sqrt{x}+\sqrt[3]{x}}\,dx$。
被积函数中含两个不同的根式 $\sqrt{x}$ 和 $\sqrt[3]{x}$，为了同时消去，令 $x = u^6$（6是2和3的最小公倍数），则 $dx = 6u^5\,du$。
$$\int \frac{1}{\sqrt{x}+\sqrt[3]{x}}\,dx = \int \frac{6u^5}{u^3+u^2}\,du = 6\int \frac{u^3}{u+1}\,du$$
$$= 6\int \left(u^2 - u + 1 - \frac{1}{u+1}\right)du = 2u^3 - 3u^2 + 6u - 6\ln|u+1| + C$$
$$= 2\sqrt{x} - 3\sqrt[3]{x} + 6\sqrt[6]{x} - 6\ln|\sqrt[6]{x}+1| + C$$

例8：$\int \sqrt{\frac{1-x}{1+x}}\,\frac{dx}{x}$。
为去掉根号，设 $\sqrt{\frac{1-x}{1+x}} = u$，经过代数运算可得
$$\int \sqrt{\frac{1-x}{1+x}}\,\frac{dx}{x} = -2\sqrt{\frac{1-x}{1+x}} + 2\ln\left|\frac{1+\sqrt{\frac{1-x}{1+x}}}{1-\sqrt{\frac{1-x}{1+x}}}\right| + C$$

【小结】

以上例子表明：如果被积函数中含有简单根式 $\sqrt[n]{ax+b}$ 或 $\sqrt{\frac{ax+b}{cx+d}}$，可以令这个简单的根式为 $u$。这样，原积分即可化为有理函数的积分。这种方法的核心思路不变：把不会积的变成会积的！
