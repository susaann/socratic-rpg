这一节介绍两个"对头"的概念：无穷小和无穷大。它们是描述函数变化趋势的重要工具。

一、无穷小

【定义】

如果函数 f(x) 当 x → x₀（或 x → ∞）时的极限为 0，就称 f(x) 是当 x → x₀（或 x → ∞）时的无穷小。

特别地，以 0 为极限的数列 { x_n } 称为 n → ∞ 时的无穷小。

例1：因为 lim_{x→1} (x-1) = 0，所以 x-1 是当 x → 1 时的无穷小。又因为 lim_{x→∞} 1/x = 0，所以 1/x 是当 x → ∞ 时的无穷小。

【特别注意】

无穷小 ≠ 很小的数！无穷小是一个函数，它在变化过程中绝对值可以小于任意给定的正数 ε。而"很小的数"（比如百万分之一）是固定值，取 ε 比它更小就"比不过"了。唯一的例外是常数 0——它可以看作无穷小，因为 |0| < ε 永远成立。

【无穷小与函数极限的关系】

定理1（桥梁定理）：在同一个变化过程（x → x₀ 或 x → ∞）中，函数 f(x) 以 A 为极限的充分必要条件是：f(x) = A + α，其中 α 是无穷小。

简单理解就是：f(x) 趋近于 A，就等于说"f(x) = A 加上一个越来越小的尾巴"。这个定理把"极限"和"无穷小"严密地联系在了一起。

二、无穷大

【定义】

如果当 x → x₀（或 x → ∞）时，函数值的绝对值 |f(x)| 可以大于任意预先指定的大正数 M，就称 f(x) 是当 x → x₀（或 x → ∞）时的无穷大。

严格表述：设函数 f(x) 在 x₀ 的某一去心邻域内有定义（或 |x| 大于某个正数时有定义）。如果对于任意给定的正数 M（不论多大），总存在正数 δ（或正数 X），只要 x 满足 0 < |x - x₀| < δ（或 |x| > X），对应的函数值 f(x) 总满足 |f(x)| > M，则称 f(x) 为当 x → x₀（或 x → ∞）时的无穷大。

注意：无穷大的极限按定义是不存在的（因为极限要求是一个确定的数，而∞不是数）。但为了描述方便，我们仍然写 lim f(x) = ∞，或者说"函数的极限是无穷大"。如果进一步区分正负，把 |f(x)| > M 换成 f(x) > M 则记作 lim f(x) = +∞；换成 f(x) < -M 则记作 lim f(x) = -∞。

同样要注意：无穷大 ≠ 很大的数！∞ 不是数，不要和 1 千万、1 亿这样的数混为一谈。

【几何意义】

如果 lim_{x→x₀} f(x) = ∞，那么直线 x = x₀ 是函数 y = f(x) 图形的铅直渐近线（函数图形在这条竖直线附近无限"蹿高"或"跌落"）。

【无穷大与无穷小的关系】

定理2：在自变量的同一变化过程中，如果 f(x) 是无穷大，则 1/f(x) 是无穷小；反之，如果 f(x) 是无穷小且 f(x) ≠ 0，则 1/f(x) 是无穷大。

这个定理非常好理解：一个数如果变得特别大，它的倒数就变得特别小（接近0）；反过来，一个数特别小（接近0但不等于0），它的倒数就特别大。这就像"放大"和"缩小"的互逆关系。

例2：证明 lim_{x→1} 1/(x-1)² = ∞。对于任意 M > 0，要使 1/(x-1)² > M，只需 |x-1| < 1/√M。取 δ = 1/√M 即可。此时直线 x = 1 是 y = 1/(x-1)² 图形的铅直渐近线。
