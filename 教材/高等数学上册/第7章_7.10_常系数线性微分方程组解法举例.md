================================================================================
              7.10  常系数线性微分方程组解法举例
================================================================================

此前各节我们处理的都是"一个方程解一个未知函数"的情形。但在实际问题中——比如多质点振动系统、电路网络、生态种群模型——常常会出现**几个微分方程联立**，共同决定几个未知函数。这些联立的方程称为**微分方程组**。

如果方程组中每一个方程都是常系数线性微分方程，就称为**常系数线性微分方程组**。求解的基本思路是"消元法"——把方程组化归为单个高阶方程，这是我们已经会解的。

【标准的三步走解法】

第一步：从方程组中消去一些未知函数及其各阶导数，得到一个只含一个未知函数的高阶常系数线性微分方程。

第二步：解这个高阶方程，求出那个未知函数。

第三步：把求得的函数代回原方程组，不经过积分就可以求出其余未知函数（因为各函数之间的关系由原方程组确定，不需要额外积分引入新常数）。

──────────────────────────────────────────────────────────────────────

【例 1】解方程组：
    dy/dx = 3y - 2z       ①
    dz/dx = 2y - z        ②

其中 y=y(x)，z=z(x)。

解：设法消去 z。由②式解出 y：
    y = (1/2)(dz/dx + z)    ③

对③求导得 dy/dx = (1/2)(d²z/dx² + dz/dx)    ④

将③、④代入①式，整理得：
    d²z/dx² - 2 dz/dx + z = 0

这是二阶常系数齐次线性方程，特征方程 r²-2r+1=0，二重根 r=1，
通解：z = (C₁ + C₂x) e^x    ⑤

将⑤代入③求得：
    y = ½(2C₁ + C₂ + 2C₂x) e^x    ⑥

⑤⑥联立即为原方程组的通解。

若初值条件为 y(0)=1，z(0)=0，代入⑥和⑤得 C₁=0，C₂=2，
特解为 y=(1+2x)e^x，z=2x e^x。

──────────────────────────────────────────────────────────────────────

【例 2：利用 D 算子简化运算】

D 算子（对 x 求导的记号）在方程组中特别方便——你可以像解代数方程组那样消元：

解方程组：
    (D²-1)x + Dy = e^t        ⑦
    Dx + (D²+1)y = 0          ⑧

为消去 x，做运算 ⑦ - D·⑧：
    (D²-1)x + Dy - D²x - D(D²+1)y = e^t
    -x - D²y = e^t    即  x = -D²y - e^t    ⑨

将⑨代入⑧：D(-D²y-e^t) + (D²+1)y = 0
    -D³y - De^t + D²y + y = 0
    (-D³+D²+1)y = e^t         ⑩

⑩为四阶非齐次线性方程，特征方程 -r⁴+r²+1=0。
【此处OCR模糊，原文可解得特征根，记为 α, -α, βi, -βi】
易求得特解 y*=e^t，于是⑩的通解为：
    y = C₁e^{αt} + C₂e^{-αt} + C₃cos βt + C₄sin βt + e^t

由⑨式 x = -D²y - e^t 可得 x 的表达式。
两个函数联立就是原方程组的通解。

【重要提醒】在求得一个未知函数后求另一个时，不要再次积分！因为积分会引入新的任意常数，而这些常数与原方程之间已经确定了关系，额外积分会导致常数之间出现矛盾。正确做法是直接利用原方程组中的等式关系。

【行列式视角（选修）】

也可用类似解代数方程的"系数行列式"来操作。将⑦⑧写成行列式形式：

    | D²-1   D  | | x |   | e^t |
    | D    D²+1 | | y | = |  0  |

展开得 (D⁴-D²-1)y = -e^t，与⑩一致。系数行列式是 D 的四次多项式，表明该方程组本质上是四阶的，通解中恰好包含四个任意常数。不过再求 x 时，不宜再次用行列式（会引入新常数），而应从原方程组直接推导。
