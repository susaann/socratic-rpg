映射，简单说就是"对应关系"——它规定了一个集合里的每个元素，在另一个集合里能找到哪个唯一的"搭档"。函数是映射的一种特殊情形，也是整个微积分的研究对象。本节就来搞清楚映射和函数到底是什么，以及函数有哪些基本性质。

一、映射

【映射的概念】

假设你有两个非空集合 X 和 Y。如果存在一个规则 f，使得 X 中的每个元素 x，都能在 Y 中找到唯一确定的元素 y 与之对应，那么我们就说 f 是从 X 到 Y 的映射，记作 f: X → Y。其中 y 称为元素 x（在映射 f 下）的像，记作 y = f(x)；而 x 称为 y 的一个原像。集合 X 称为映射 f 的定义域（记作 D_f），X 中所有元素的像组成的集合称为映射 f 的值域（记作 R_f），即 R_f = f(X) = { f(x) | x ∈ X }。

理解映射时要注意两点：

（1）构成一个映射必须具备三个要素：集合 X（定义域 D_f = X）、集合 Y（值域所在的范围，R_f ⊂ Y）、以及对应法则 f（使得每个 x ∈ X 都有唯一确定的 y = f(x) 与之对应）。

（2）对每个 x ∈ X，x 的像 y 是唯一的；但对每个 y ∈ R_f，y 的原像不一定是唯一的。映射的值域 R_f 是 Y 的一个子集，但不一定等于整个 Y。

【三个例子帮你理解】

例1：设 f: R → R，对每个 x ∈ R，f(x) = x²。显然 f 是一个映射，定义域 D_f = R，值域 R_f = { y | y ≥ 0 }，它是 R 的真子集（即比 R 小）。对于 R_f 中的元素 y，除 y = 0 外，它的原像不是唯一的。比如 y = 4 的原像就有 x = 2 和 x = -2 两个。

例2：设 X = { (x, y) | x² + y² = 1 }（单位圆上的点），Y = { (x, 0) | |x| ≤ 1 }（x轴上的线段）。令 f: X → Y，对 (x, y) ∈ X，有唯一确定的 (x, 0) ∈ Y 与之对应。显然 f 是一个映射，定义域 D_f = X，值域 R_f = Y。在几何上，这个映射表示将平面上圆心在原点的单位圆上的点投影到 x 轴的区间 [-1, 1] 上。

例3：设映射 f: [-π/2, π/2] → [-1, 1]，对每个 x ∈ [-π/2, π/2]，f(x) = sin x。这是一个映射，定义域 D_f = [-π/2, π/2]，值域 R_f = [-1, 1]。

【三种特殊映射】

- 满射（到上的映射）：如果 R_f = Y，即 Y 中每一个元素 y 都是 X 中某个元素的像，就称 f 为满射。
- 单射：如果 X 中任意两个不同元素 x₁ ≠ x₂，都有 f(x₁) ≠ f(x₂)，就称 f 为单射。
- 一一映射（双射）：既是单射又是满射的映射。

对照上面的例子：例1的映射既非单射又非满射（因为值域不是整个 R，且正负数能对应同一个像）；例2的映射不是单射（圆上对称的两点投影到同一个点）但是满射；例3的映射既是单射又是满射，因此是一一映射。

映射在不同数学分支中有不同的惯用名称。比如，从非空集 X 到数集 Y 的映射称为 X 上的泛函；从非空集 X 到它自身的映射称为 X 上的变换；从实数集（或其子集）X 到实数集 Y 的映射，就是我们熟悉的概念——函数。

【逆映射与复合映射】

先说逆映射。设 f 是从 X 到 Y 的单射，那么对每个 y ∈ R_f，有唯一的 x ∈ X 满足 f(x) = y。于是我们可以从 R_f 到 X 定义一个新映射 g，对每个 y ∈ R_f，规定 g(y) = x（这个 x 满足 f(x) = y）。这个映射 g 就称为 f 的逆映射，记作 f⁻¹，其定义域为 R_f，值域为 X。

简单理解就是：逆映射把"结果"变回"原因"，它把映射的方向反过来了。但并不是所有映射都有逆映射——只有单射才有。实际上，单射保证了每个像 y 只有一个原像，这才好反过来对应。在例3中，正弦函数 sin x 在 [-π/2, π/2] 上是一一映射，存在逆映射 f⁻¹(x) = arcsin x, x ∈ [-1, 1]，其定义域为 [-1, 1]，值域为 [-π/2, π/2]。

再说复合映射。设有两个映射 g: X → Y₁ 和 f: Y₂ → Z，其中 Y₁ ⊂ Y₂。那么由 g 和 f 可以确定一个从 X 到 Z 的对应法则：对每个 x ∈ X，对应 f[g(x)] ∈ Z。这个映射称为映射 g 和 f 构成的复合映射，记作 f∘g，即 (f∘g)(x) = f[g(x)]，x ∈ X。

构成复合映射的条件是：g 的值域 R_g 必须包含在 f 的定义域 D_f 内（R_g ⊂ D_f），否则不能复合。另外要注意顺序——f∘g 有意义，不一定 g∘f 也有意义。即使两者都有意义，f∘g 和 g∘f 通常也不相同。

例4：设有映射 g: R → [-1, 1]，对每个 x ∈ R，g(x) = sin x；以及映射 f: [-1, 1] → [0, 1]，对每个 u ∈ [-1, 1]，f(u) = √(1-u²)。则 g 和 f 构成的复合映射为 f∘g: R → [0, 1]，对每个 x ∈ R，有 (f∘g)(x) = f[g(x)] = f(sin x) = √(1-sin²x) = |cos x|。

二、函数

【函数的概念】

设 D ⊂ R，则称映射 f: D → R 为定义在 D 上的函数，通常简记为 y = f(x), x ∈ D。其中 x 称为自变量，y 称为因变量，D 称为定义域，记作 D_f。函数定义中，对每个 x ∈ D，按对应法则 f 总有唯一确定的值 y 与之对应，这个值称为函数 f 在 x 处的函数值，记作 f(x)。函数值 f(x) 的全体构成的集合称为函数 f 的值域，记作 R_f = f(D) = { y | y = f(x), x ∈ D }。

注意：记号 f 和 f(x) 的含义有区别。前者表示自变量 x 和因变量 y 之间的对应法则，而后者表示与自变量 x 对应的函数值。但习惯上常用记号"f(x), x ∈ D"或"y = f(x), x ∈ D"来表示定义在 D 上的函数。

函数是从实数集到实数集的映射，因此构成函数的要素是：定义域 D_f 和对应法则 f。如果两个函数的定义域相同、对应法则也相同，那么这两个函数就是相同的，否则就是不同的。

函数的定义域有两种确定方式：一种是有实际背景的函数，根据变量的实际意义确定。例如在自由落体运动中，设物体下落的时间为 t，下落的距离为 s，开始时刻 t = 0，落地时刻 t = T，则 s 与 t 的函数关系是 s = (1/2)gt²，t ∈ [0, T]。另一种是用算式表达的函数，通常约定定义域是使得算式有意义的一切实数组成的集合，这称为函数的自然定义域。比如 y = √(1-x²) 的定义域是闭区间 [-1, 1]，y = 1/√(1-x²) 的定义域是开区间 (-1, 1)。

表示函数有三种主要方法：表格法、图形法、解析法（公式法）。其中图形法基于函数图形的概念，即坐标平面上的点集 { P(x, y) | y = f(x), x ∈ D } 称为函数 y = f(x), x ∈ D 的图形。

【几个典型函数的例子】

例5：常函数 y = 2，定义域 D = (-∞, +∞)，值域 W = {2}。它的图形是一条平行于 x 轴的直线。

例6：绝对值函数 y = |x| = { -x, x < 0; x, x ≥ 0 }，定义域 D = (-∞, +∞)，值域 R_f = [0, +∞)。这称为绝对值函数。

例7：符号函数 y = sgn x = { -1, x < 0; 0, x = 0; 1, x > 0 }。定义域 D = (-∞, +∞)，值域 R_f = {-1, 0, 1}。对于任何实数 x，有性质：x = sgn x · |x|。

例8：取整函数 y = [x]，设 x 为任一实数，不超过 x 的最大整数称为 x 的整数部分，记作 [x]。比如 [0] = 0，[√2] = 1，[π] = 3，[-1] = -1，[-3.5] = -4。取整函数 y = [x] 的定义域 D = (-∞, +∞)，值域为全体整数 Z。它的图形称为阶梯曲线——在 x 为整数值处，图形发生跳跃，跃度为 1。

在例6和例7中我们看到，有时一个函数要用几个式子表示。这种在自变量的不同变化范围中，对应法则用不同式子来表示的函数，称为分段函数。

例9：函数 f(x) = { 2√x, 0 ≤ x ≤ 1; 1+x, x > 1 } 是一个分段函数，定义域 D = [0, +∞)。当 x ∈ [0, 1] 时 f(x) = 2√x；当 x ∈ (1, +∞) 时 f(x) = 1+x。比如 f(1/4) = 2√(1/4) = 1，f(1) = 2，f(3) = 4。

用几个式子来表示一个函数，不仅与函数定义没有矛盾，而且有现实意义。比如在等温过程中，气体压强 p 与体积 V 的函数关系：当 V 不太小时遵循玻意耳定律 p = α/V（α 为常量）；当 V 相当小时就要用范德瓦耳斯方程 p = α/V - β/V²（β 为常量）。

【函数的四种基本特性】

（1）有界性。设函数 f(x) 的定义域为 D，数集 X ⊂ D。如果存在数 K₁，使得 f(x) ≤ K₁ 对任一 x ∈ X 都成立，则称函数 f(x) 在 X 上有上界，K₁ 称为上界。如果存在数 K₂，使得 f(x) ≥ K₂ 对任一 x ∈ X 都成立，则称函数 f(x) 在 X 上有下界，K₂ 称为下界。如果存在正数 M，使得 |f(x)| ≤ M 对任一 x ∈ X 都成立，则称函数 f(x) 在 X 上有界。如果这样的 M 不存在，就说函数 f(x) 在 X 上无界。

简单理解：有界就是函数值不会无限大也不会无限小，被一个"框框"限制住了。

例如，f(x) = sin x 在 (-∞, +∞) 内是有界的，因为 |sin x| ≤ 1 对任意实数 x 都成立。而 f(x) = 1/x 在 (0, 1) 内是无界的（x 越接近 0，函数值越大，没有上限），但在 (1, 2) 内是有界的。

（2）单调性。设函数 f(x) 的定义域为 D，区间 I ⊂ D。如果对于区间 I 上任意两点 x₁ 和 x₂，当 x₁ < x₂ 时恒有 f(x₁) < f(x₂)，则称函数 f(x) 在区间 I 上是单调增加的；如果有 f(x₁) > f(x₂)，则称单调减少。

例如，f(x) = x² 在 [0, +∞) 上是单调增加的，在 (-∞, 0] 上是单调减少的。

（3）奇偶性。设函数 f(x) 的定义域 D 关于原点对称。如果对于任一 x ∈ D，恒有 f(-x) = f(x)，则称 f(x) 为偶函数（图形关于 y 轴对称）；如果 f(-x) = -f(x)，则称 f(x) 为奇函数（图形关于原点对称）。

例如，f(x) = x² 是偶函数（因为 (-x)² = x²），f(x) = x³ 是奇函数（因为 (-x)³ = -x³）。y = sin x 是奇函数，y = cos x 是偶函数。y = sin x + cos x 既不是奇函数也不是偶函数。

（4）周期性。设函数 f(x) 的定义域为 D。如果存在一个正数 l，使得对任一 x ∈ D 有 x ± l ∈ D，且 f(x + l) = f(x) 恒成立，则称 f(x) 为周期函数，l 称为周期。通常所说的周期是指最小正周期。

例如，sin x 和 cos x 的周期是 2π；tan x 的周期是 π。但并非每个周期函数都有最小正周期——狄利克雷（Dirichlet）函数 D(x) = { 1, x 为有理数; 0, x 为无理数 } 就是一个例子，任何正有理数都是它的周期，但不存在最小的正有理数，所以没有最小正周期。

【反函数与复合函数】

作为逆映射的特例，反函数的概念是：设函数 f: D → f(D) 是单射，则存在逆映射 f⁻¹: f(D) → D，称此映射 f⁻¹ 为函数 f 的反函数。也就是说，如果 y = f(x)，那么反过来就有 x = f⁻¹(y)。

例如，函数 y = x³, x ∈ R 的反函数是 y = ∛x（即三次方根）, x ∈ R。

由于习惯上自变量用 x 表示，因变量用 y 表示，所以通常写为 y = f⁻¹(x), x ∈ f(D)。相对于反函数，原来的函数 y = f(x) 称为直接函数。

把直接函数 y = f(x) 和它的反函数 y = f⁻¹(x) 的图形画在同一坐标平面上，这两个图形关于直线 y = x 是对称的。这是因为如果 (a, b) 在 y = f(x) 的图形上（即 b = f(a)），则 (b, a) 在 y = f⁻¹(x) 的图形上（即 a = f⁻¹(b)），而 (a, b) 与 (b, a) 关于 y = x 对称。

复合函数是复合映射的特例。设函数 y = f(u) 的定义域为 D_f，函数 u = g(x) 的定义域为 D_g，且其值域 R_g ⊂ D_f，则称 y = f[g(x)], x ∈ D 为由 u = g(x) 和 y = f(u) 构成的复合函数，记作 f∘g。

一个生动的理解是：复合函数就像"俄罗斯套娃"——外层函数 f 包着内层函数 g，x 先进 g 算出结果，再把这个结果送进 f 算出最终答案。

例如，y = arcsin(sin x), x ∈ R 可构成复合函数（因为 sin x 的值域 [-1, 1] 正好是 arcsin 的定义域）。而 y = √(tan x) 不能直接复合（因为 tan x 可取负值，但根号下不能为负），必须限制 x 的范围，比如取 x ∈ (kπ, (k+1/2)π) 时才有意义。

有时也会遇到两个以上函数构成的复合函数。例如 y = √u, u = cot v, v = x/2 可构成 y = √(cot(x/2))。

【函数的四则运算】

设函数 f(x) 和 g(x) 的定义域分别为 D₁ 和 D₂，且 D = D₁ ∩ D₂ ≠ ∅（两个定义域有交集），则可以定义：
- 和（差）f ± g: (f ± g)(x) = f(x) ± g(x), x ∈ D
- 积 f · g: (f · g)(x) = f(x) · g(x), x ∈ D
- 商 f / g: (f / g)(x) = f(x) / g(x), x ∈ D（要求 g(x) ≠ 0）

例：设函数 f(x) 的定义域关于原点对称，且 f(x) 定义在 (-1, 1) 上。可以证明：f(x) 可以表示为一个偶函数 g(x) 与一个奇函数 h(x) 的和。证法是构造 g(x) = (1/2)[f(x) + f(-x)]，h(x) = (1/2)[f(x) - f(-x)]，容易验证 g 是偶函数、h 是奇函数，且 f(x) = g(x) + h(x)。

【初等函数】

在中学我们已经学过以下五类基本初等函数：
- 幂函数：y = x^μ（μ ∈ R 是常数）
- 指数函数：y = a^x（a > 0 且 a ≠ 1）
- 对数函数：y = log_a x（a > 0 且 a ≠ 1，特别地 a = e 时记为 y = ln x）
- 三角函数：如 y = sin x, y = cos x, y = tan x 等
- 反三角函数：如 y = arcsin x, y = arccos x, y = arctan x 等

由常数和基本初等函数经过有限次的四则运算和有限次的函数复合步骤所构成、并可用一个式子表示的函数，称为初等函数。例如 y = √(1-x²)、y = sin²x、y = √(cot(x/2)) 等都是初等函数。在本课程中讨论的函数绝大多数都是初等函数。

【双曲函数与反双曲函数】

以 e 为底的指数函数 y = e^x 和 y = e^(-x) 可以组合出三个实用的函数：

- 双曲正弦：sh x = (e^x - e^(-x)) / 2
- 双曲余弦：ch x = (e^x + e^(-x)) / 2
- 双曲正切：th x = sh x / ch x = (e^x - e^(-x)) / (e^x + e^(-x))

这三个双曲函数的性质如下：

双曲正弦 sh x：定义域 (-∞, +∞)，奇函数，图形过原点且关于原点对称，在 (-∞, +∞) 内单调增加。当 x 的绝对值很大时，图形在第一象限内接近于曲线 y = (1/2)e^x，在第三象限内接近于 y = -(1/2)e^(-x)。

双曲余弦 ch x：定义域 (-∞, +∞)，偶函数，图形过 (0, 1) 且关于 y 轴对称。在 (-∞, 0) 内单调减少，在 (0, +∞) 内单调增加。ch 0 = 1 是最小值。当 |x| 很大时，图形接近于 y = (1/2)e^x。

双曲正切 th x：定义域 (-∞, +∞)，奇函数，图形过原点且关于原点对称，在 (-∞, +∞) 内单调增加。图形夹在水平直线 y = 1 和 y = -1 之间。

双曲函数有与三角函数类似的关系式：
- sh(x ± y) = sh x ch y ± ch x sh y
- ch(x ± y) = ch x ch y ± sh x sh y
- ch²x - sh²x = 1
- sh 2x = 2 sh x ch x
- ch 2x = ch²x + sh²x

（对比三角恒等式：cos²x + sin²x = 1，sin(x+y) = sin x cos y + cos x sin y 等，帮助记忆）

反双曲函数是双曲函数的反函数，都可用自然对数表示：

- 反双曲正弦：y = arsh x = ln(x + √(x²+1))，定义域 (-∞, +∞)，奇函数，在 (-∞, +∞) 内单调增加。

- 反双曲余弦（主值）：y = arch x = ln(x + √(x²-1))，定义域 [1, +∞)，在 [1, +∞) 上单调增加。

- 反双曲正切：y = arth x = (1/2)ln((1+x)/(1-x))，定义域 (-1, 1)，在 (-1, 1) 内是单调增加的奇函数，图形关于原点对称。
