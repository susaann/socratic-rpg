================================================================================
                7.8  常系数非齐次线性微分方程（第二类型与共振）
================================================================================

上一节我们处理了非齐次项 f(x) 是多项式乘以指数函数的情形。本节讨论更一般的第二类型：

    f(x) = e^{λx} [ P_l(x) cos ωx + Q_n(x) sin ωx ]

其中 P_l(x)、Q_n(x) 分别是 l 次、n 次多项式。这类非齐次项在物理中非常常见——比如带有阻尼和周期性外力的振动系统，其方程就属于这一类型。

【核心结论】
对于二阶常系数非齐次线性方程 y'' + py' + qy = f(x)，当 f(x) 是上述形式时，特解可设为：

    y* = x^k e^{λx} [ R_m^{(1)}(x) cos ωx + R_m^{(2)}(x) sin ωx ]

其中：
  - m = max{l, n}，即取两个多项式次数的较大值；
  - R_m^{(1)}(x)、R_m^{(2)}(x) 是两个待定的 m 次多项式（系数不同，各自独立待定）；
  - k 按 λ+ωi（或 λ-ωi）是否为特征方程的根来取值：
      · 如果不是特征根 → k = 0；
      · 如果是特征单根 → k = 1。

这个结论可以推广到 n 阶方程，只需注意 k 是特征方程中含根 λ+ωi（或 λ-ωi）的"重数"。

【为什么这样设？——思路简述】
利用欧拉公式 cos θ = (e^{iθ}+e^{-iθ})/2，sin θ = (e^{iθ}-e^{-iθ})/(2i)，可以把 f(x) 拆成两个"多项式乘纯指数"的叠加：

    f(x) = P̅(x) e^{(λ+ωi)x} + P̂(x) e^{(λ-ωi)x}

然后分别套用上一节的待定系数法，最后再由叠加原理合并。运算过程中会产生复系数多项式，但最终结果可以整理回实函数形式，就是上面给出的公式。

──────────────────────────────────────────────────────────────────────

【例 3】求 y'' + y = x cos 2x 的一个特解。

解：这是二阶常系数非齐次线性方程，f(x) 属于 e^{λx}[P_l cos ωx + Q_n sin ωx] 型，
其中 λ=0，ω=2，P_l(x)=x（l=1），Q_n(x)=0（n=0），故 m=1。

对应齐次方程 y''+y=0 的特征方程为 r²+1=0，根为 r=±i。

由于 λ+ωi = 2i 不是特征根，所以 k=0，应设特解为：

    y* = (ax+b) cos 2x + (cx+d) sin 2x

代入原方程，整理后比较 cos 2x 和 sin 2x 的系数，得到方程组：

    -3a = 1,   -3b+4c = 0,   -3c = 0,   -3d-4a = 0

解得 a=-1/3，b=0，c=0，d=4/9。于是：

    y* = -⅓x cos 2x + 4/9 sin 2x

【例 4】求 y'' - y = e^x cos 2x 的一个特解。

解：这里 λ=1，ω=2，P_l(x)=1（l=0），Q_n(x)=0（n=0），m=0。

特征方程 r²-1=0，根 r=±1。λ+ωi = 1+2i 不是特征根，k=0，应设：

    y* = e^x (a cos 2x + b sin 2x)

求一阶和二阶导数后代入原方程，整理得：

    4e^x [(-a+b) cos 2x - (a+b) sin 2x] = e^x cos 2x

比较系数得 -a+b=1/4，a+b=0，解得 a=-1/8，b=1/8。

    y* = ⅛ e^x (sin 2x - cos 2x)

【例 5：无阻尼强迫振动与共振】回到第六节例 1，物体受周期性外力作用，运动方程为：

    d²x/dt² + k²x = h sin pt    （无阻尼强迫振动）

对应齐次方程 d²x/dt² + k²x = 0 的特征根为 r=±ki，齐次通解为：

    X = A sin(kt+φ)

右端 f(t)=h sin pt 属于前述类型（λ=0，ω=p）。

分两种情况：
（1）若 p ≠ k（外力频率不等于固有频率）：此时 ±pi 不是特征根，设
    x* = a cos pt + b sin pt，代入求得 a=0，b=h/(k²-p²)，
    通解为 x = A sin(kt+φ) + [h/(k²-p²)] sin pt。
    运动由自由振动和强迫振动叠加而成。当 p 接近 k 时，强迫振动振幅 |h/(k²-p²)| 会很大。

（2）若 p = k（外力频率等于固有频率）：此时 ±ki 是特征单根，应设
    x* = t(a cos kt + b sin kt)，
    代入求得 a=-h/(2k)，b=0，
    通解为 x = A sin(kt+φ) - (h/2k) t cos kt。

观察这个解：第二项的振幅随着时间 t 的增大而无限增大——这就是**共振现象**。

【重要物理启示】为了避免共振（比如桥梁、机械结构），应使外力频率远离系统的固有频率；反过来，如果要利用共振（比如乐器共鸣箱），则应让两者尽量接近。有阻尼的强迫振动可作类似讨论，这里从略。
