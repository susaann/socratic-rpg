## 4.2 Why Nonlinearity Is Necessary: A Formal Proof

### LLL Linear Layers = 1 Linear Layer

This is one of the most important theoretical results in neural network architecture.

**Theorem** : A network of LLL layers where every activation function is linear (or the identity) is equivalent to a single linear layer, regardless of LLL.

**Proof** : Consider an LLL-layer network with identity activations. The output of layer lll is:

h(l)=W(l)h(l−1)+b(l)\mathbf{h}^{(l)} = \mathbf{W}^{(l)} \mathbf{h}^{(l-1)} + \mathbf{b}^{(l)}h(l)=W(l)h(l−1)+b(l)

Substituting recursively:

h(1)=W(1)x+b(1)\mathbf{h}^{(1)} = \mathbf{W}^{(1)} \mathbf{x} + \mathbf{b}^{(1)}h(1)=W(1)x+b(1) h(2)=W(2)h(1)+b(2)=W(2)W(1)x+W(2)b(1)+b(2)\mathbf{h}^{(2)} = \mathbf{W}^{(2)} \mathbf{h}^{(1)} + \mathbf{b}^{(2)} = \mathbf{W}^{(2)}\mathbf{W}^{(1)} \mathbf{x} + \mathbf{W}^{(2)}\mathbf{b}^{(1)} + \mathbf{b}^{(2)}h(2)=W(2)h(1)+b(2)=W(2)W(1)x+W(2)b(1)+b(2)

At layer LLL:

h(L)=(∏l=L1W(l))⏟W′x+constant vector⏟b′=W′x+b′\mathbf{h}^{(L)} = \underbrace{\left(\prod_{l=L}^{1} \mathbf{W}^{(l)}\right)}_{\mathbf{W}'} \mathbf{x} + \underbrace{\text{constant vector}}_{\mathbf{b}'} = \mathbf{W}'\mathbf{x} + \mathbf{b}'h(L)=W′(l=L∏1​W(l))​​x+b′constant vector​​=W′x+b′

This is a single affine transformation. No matter how many layers, the product of matrices is a matrix; the composition of linear maps is a linear map. **Depth without nonlinearity adds zero expressive power.** □\square□

For a general linear activation σ(z)=az+c\sigma(z) = az + cσ(z)=az+c, the argument is analogous — the composed function is still h(L)=Ax+c\mathbf{h}^{(L)} = \mathbf{A}\mathbf{x} + \mathbf{c}h(L)=Ax+c for some matrix A\mathbf{A}A and vector c\mathbf{c}c.

**Corollary** : nonlinear activation functions are not a design choice — they are a necessity. Without them, depth provides no benefit.
