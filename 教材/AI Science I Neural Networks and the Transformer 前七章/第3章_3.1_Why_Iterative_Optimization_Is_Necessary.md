## 3.1 Why Iterative Optimization Is Necessary

Some learning problems admit closed-form solutions. For a linear model f(x)=w⊤x+bf(\mathbf{x}) = \mathbf{w}^\top \mathbf{x} + bf(x)=w⊤x+b minimizing MSE, the optimal weights satisfy the normal equations:

w^=(X⊤X)−1X⊤y\hat{\mathbf{w}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}w^=(X⊤X)−1X⊤y

This is a single matrix inversion — no iteration required. Two reasons prevent this approach from generalizing to deep networks.

### Non-convexity

The MSE loss of a linear model is **convex** in w\mathbf{w}w — the loss surface is a bowl-shaped paraboloid with a single global minimum, which the normal equations find directly.

As soon as the model contains a nonlinearity, convexity is lost. A two-layer network:

y^=w2⊤σ(W1x+b1)+b2\hat{y} = \mathbf{w}_2^\top \sigma(\mathbf{W}_1 \mathbf{x} + \mathbf{b}_1) + b_2y^​=w2⊤​σ(W1​x+b1​)+b2​

has a loss surface that is highly non-convex with respect to the parameters (W1,b1,w2,b2)(\mathbf{W}_1, \mathbf{b}_1, \mathbf{w}_2, b_2)(W1​,b1​,w2​,b2​). Such surfaces contain:

  * **Local minima** : points where the loss is locally minimal but not globally so
  * **Saddle points** : points where the gradient is zero but the Hessian has both positive and negative eigenvalues — a minimum in some directions and a maximum in others
  * **Plateaus** : extended regions where the gradient is near zero but the function is not at a minimum



No closed-form expression exists for the global minimum of a non-convex function.

### Scale

Even if the loss were convex, the cost of the normal equations grows as O(np2)O(np^2)O(np2) to form X⊤X\mathbf{X}^\top \mathbf{X}X⊤X and O(p3)O(p^3)O(p3) to invert it. For a model with p=109p = 10^9p=109 parameters (the scale of modern language models), O(p3)=O(1027)O(p^3) = O(10^{27})O(p3)=O(1027) operations. At a petaflop (101510^{15}1015 operations per second), this would require approximately 101210^{12}1012 seconds — roughly 32,000 years.

Gradient descent scales as O(np)O(np)O(np) per pass through the data (or O(Bp)O(Bp)O(Bp) with mini-batches of size BBB). It is the only approach that is computationally viable at this scale.
