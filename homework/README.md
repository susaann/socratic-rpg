# 作业系统说明

## 如何使用

### 1. 准备作业文件

在 `homework/待批改/` 文件夹中创建一个 Markdown 文件，格式如下：

```markdown
# 第X课作业：章节名

## 题目 1
题目内容（可包含 LaTeX 公式）

## 题目 2
题目内容

## 题目 3
...
```

> **注意**：文件只需要包含题目，不需要预先写解答。解答在对话中逐题进行。

### 2. 发起批改

在对话中对 AI 说：

- **"请批改作业 第X课作业_章节名.md"** — 批改指定文件
- **"开始批改作业"** — 列出所有待批改文件，让你选择

### 3. 批改过程

- 教师逐题展示题目
- 你在对话中给出解答
- 教师判断正误并引导（苏格拉底式，不给答案）
- 可以随时说"这题不会"跳过

### 4. 批改结果

批改完成后：
- 教师给出口头总结（正确/部分正确/未掌握 + 建议）
- 批改结果写入作业文件末尾
- 文件从 `homework/待批改/` 移动到 `homework/` 根目录

## 文件结构

```
homework/
├── README.md                 ← 你在这里
├── 待批改/                    ← 放入待批改的作业文件
└── 第X课作业_章节名.md        ← 批改完成后的作业文件（含批改结果）
```
